## <a name="youll-be-asked-for-the-following-information-when-you-sign-up"></a>Du kommer att uppmanas att ange följande information när du registrerar dig

- **Adress och kontaktinformation för din prenumeration:**

  - **Land** där tjänsterna kommer att användas. Du kan **inte** ändra landet senare, även under registrerings processen. Du måste starta om registrerings guiden.

  - **E-postadress** och **telefonnummer** så att vi kan kontakta dig angående prenumerationen om så behövs. Om du till exempel glömt ditt lösenord använder vi informationen för att skicka dig ett tillfälligt lösenord. Vi skickar också faktureringsinformation till den e-postadress du anger.

    > [!NOTE]
    > E-postadressen som du anger här skiljer sig från din Microsoft 365-e-postadress (ditt inloggnings namn nedan). Eftersom det är hit vi skickar din faktureringsinformation rekommenderar vi att du använder en e-postadress som är lämplig för att ta emot affärs-e-post.
  
- **Ett inloggnings namn (användar-ID):** Detta användar-ID blir din initiala e-postadress för Microsoft 365, så att du snabbt kommer igång.

    Detta användar-ID är den e-postadress som du använder för att logga in. Om namnet på företaget är till exempel Fourth Coffee kan du välja rob@fourthcoffee.onmicrosoft.com som användar-ID.

    De flesta lägger till sin egen anpassade domän strax efter att de har anmält sig så att de kan börja få e-post. Om du till exempel har en egen domän med namnet fourthcoffee.com kan du konfigurera din e-postadress som rob@fourthcoffee.com.

- **Betalningsinformation:**

    Du kan betala för ditt abonnemang med ett kredit kort. Om kostnaden når ett visst belopp kan du också få möjligheten att betala med faktura.

    > [!IMPORTANT]
    >  När du registrerar dig bör du välja det bästa betalnings alternativet för din organisation. Om du vill byta betalningsalternativ senare måste du ringa betalningssupporten.
