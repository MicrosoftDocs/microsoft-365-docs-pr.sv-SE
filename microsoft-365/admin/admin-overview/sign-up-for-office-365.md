::: moniker range="o365-worldwide"

Registrera dig för Microsoft 365 för företag så att ditt team kan börja använda de senaste versionerna av Word, Excel, PowerPoint och andra Office-program i ditt företag.
  
::: moniker-end

::: moniker range="o365-21vianet"

Office 365 som drivs av 21Vianet är utformat för att uppfylla behoven av säkra, tillförlitliga och skalbara molntjänster i Kina. Den här tjänsten drivs av teknik som Microsoft har licensierat till 21Vianet. Microsoft använder inte själva tjänsten. 21Vianet driver, tillhandahåller och hanterar leverans av tjänsten. 21Vianet är den största carrier-neutrala Internet datacentertjänster leverantör i Kina, som tillhandahåller hosting, hanterade nätverkstjänster och cloud computing infrastrukturtjänster. Genom att licensiera Microsoft-teknik driver 21Vianet lokala Microsoft-datacenter för att ge dig möjlighet att använda Microsoft-tjänster samtidigt som dina data finns i Kina. 21Vianet tillhandahåller även dina prenumerations- och faktureringstjänster samt support.
  
> [!NOTE]
> Dessa tjänster omfattas av kinesiska lagar.
  
När du registrerar dig för Office 365 som drivs av 21Vianet kan ditt team börja använda de senaste versionerna av Word, Excel, PowerPoint och andra Office-program i ditt företag.
  
Är du redo att registrera dig? [Välj en plan](https://products.office.com/zh-cn/business/compare-office-365-for-business-plans).

## <a name="youll-be-asked-for-the-following-information-when-you-sign-up"></a>Du blir tillfrågad om följande information när du registrerar dig

- **Adress och kontaktinformation för din prenumeration:**

  - **Land** där tjänsterna kommer att användas. Du **kan inte** ändra detta senare, inte ens under registreringen, utan du måste i så fall starta om registreringsguiden.

  - **E-postadress** och **telefonnummer** så att vi kan kontakta dig angående prenumerationen om så behövs. Om du till exempel glömt ditt lösenord använder vi informationen för att skicka dig ett tillfälligt lösenord. Vi skickar också faktureringsinformation till den e-postadress du anger.

    > [!NOTE]
    > Den e-postadress du anger här skiljer sig från din Microsoft 365-e-postadress (ditt inloggningsnamn nedan). Eftersom det är hit vi skickar din faktureringsinformation rekommenderar vi att du använder en e-postadress som är lämplig för att ta emot affärs-e-post.
  
- **Ett inloggningsnamn (användar-ID):** Detta blir din första Microsoft 365-e-postadress, bara för att komma igång snabbt.

    Det här är den e-postadress du använder för att logga in. Om namnet på företaget är till exempel Fourth Coffee kan du välja rob@fourthcoffee.onmicrosoft.com som användar-ID.

    De flesta lägger till sin egen anpassade domän strax efter att de registrerat sig så att de kan börja få e-post till den. Om du till exempel har en egen domän med namnet fourthcoffee.com kan du konfigurera din e-postadress som rob@fourthcoffee.com.

- **Betalningsinformation:**

    Du kan betala för din prenumeration med ett kreditkort. Om kostnaden når ett visst belopp kan du också få möjligheten att betala med faktura.

    > [!IMPORTANT]
    >  När du registrerar dig måste du välja det bästa betalningsalternativet för din organisation. Om du vill byta betalningsalternativ senare måste du ringa betalningssupporten.
