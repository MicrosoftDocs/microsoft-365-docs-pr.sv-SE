# <a name="admin-center-overview"></a>Översikt över administrationscentret 

  
När du köper ett [microsoft 365-program för företag](https://go.microsoft.com/fwlink/?linkid=856886) eller ett [Microsoft 365 för företag-abonnemang](https://www.microsoft.com/microsoft-365/business)ger vi dig ett speciellt konto som har administratörs behörigheter. Det här kontot kallas ett administratörskonto.
  
Med det här kontot får du till gång till administrations centret för Microsoft 365 för att hantera företaget: lägga till användare, hantera prenumeration, återställa lösen ord med mera. Prova!
  
::: moniker range="o365-worldwide"

 **Logga in med ditt administratörs konto på <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> .**

::: moniker-end

::: moniker range="o365-germany"

**Logga in med ditt administratörs konto i Office 365 Germany på <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">https://portal.office.de</a> .**

::: moniker-end

::: moniker range="o365-21vianet"

**Logga in med ditt administratörs konto till Office 365 som drivs av 21Vianet på <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">https://portal.partner.microsoftonline.cn</a> .**

::: moniker-end

Om du får meddelandet "**du har inte behörighet att komma åt sidan eller utför åtgärden**" är du inte administratör.

## <a name="who-has-admin-permissions-in-my-business"></a>Vem har administratörsbehörigheter i min verksamhet?
<a name="bkmk_admin"> </a>

När du söker efter en administratör som kan återställa ditt lösenord, ta bort ett konto eller utföra andra uppgifter har du följande alternativ:
  
- **Universitet och skolor**: Kontakta ditt team för teknisk support. Vanligtvis hittar du en länk på universitetets webbplats. I mindre skolor är det möjligt att det bara finns några personer som har administratörs behörighet. 
    
- **Stora företag**: Kontakta den interna supportavdelningen/teknisk support. 
    
- **Småföretag**: Kontakta företagsägaren/delägaren. Ofta ger de administratörsbehörighet till IT-konsulten som utför allt datorunderhåll i verksamheten. 
    
Den person som registrerar sig för och köper ett Microsoft 365 för företag-abonnemang får som standard administratörs behörigheter. Den personen kan tilldela administratörs behörigheter till andra personer för att under lätta hanteringen av Microsoft 365 för sin organisation.
  
Om du inte vet vem du ska kontakta på jobbet eller skolan kan du be personen som gav dig ditt användar konto och lösen ord.
