# <a name="admin-center-overview"></a>Översikt över administrationscentret 

  
När du köper ett [Microsoft 365 Apps for business-abonnemang](https://go.microsoft.com/fwlink/?linkid=856886) eller ett [Microsoft 365 for Business-abonnemang](https://www.microsoft.com/microsoft-365/business)ger vi dig ett särskilt konto som har administratörsbehörighet. Det här kontot kallas ett administratörskonto.

## <a name="who-has-admin-permissions-in-my-business"></a>Vem har administratörsbehörigheter i min verksamhet?
<a name="bkmk_admin"> </a>

När du söker efter en administratör som kan återställa ditt lösenord, ta bort ett konto eller utföra andra uppgifter har du följande alternativ:
  
- **Universitet och skolor**: Kontakta ditt team för teknisk support. Vanligtvis hittar du en länk på universitetets webbplats. På mindre skolor kan det bara finnas några få personer som har administratörsbehörighet. 
    
- **Stora företag**: Kontakta den interna supportavdelningen/teknisk support. 
    
- **Småföretag**: Kontakta företagsägaren/delägaren. Ofta ger de administratörsbehörighet till IT-konsulten som utför allt datorunderhåll i verksamheten. 
    
Som standard får den person som registrerar sig för och köper en Microsoft 365 för företag-prenumeration administratörsbehörighet. Den personen kan tilldela administratörsbehörighet till andra personer för att hjälpa dem att hantera Microsoft 365 för sin organisation.
  
Om du inte har någon aning om vem du ska kontakta på jobbet eller skolan för att få hjälp kan du prova att fråga personen som gav dig ditt användarkonto och lösenord.
