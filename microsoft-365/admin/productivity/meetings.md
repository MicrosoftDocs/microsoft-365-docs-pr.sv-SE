# <a name="meetings--people-experiences"></a>Möten – upplevelser för människor

Produktivitetsresultatet ger insikter om organisationens digitala omvandlingsresa genom dess användning av Microsoft 365 och de teknikupplevelser som stöder den. Organisationens resultat återspeglar mätvärden för personer och teknikupplevelse och kan jämföras med riktmärken från organisationer som påminner om din. Möteskategorin är en del av åtgärderna för personer. Mer information finns i översikten [över Productivity Score](productivity-score.md) och [Microsofts sekretesspolicy.](https://privacy.microsoft.com/privacystatement)

## <a name="prerequisites"></a>Förutsättningar

För att komma igång med mötesinsikter måste personer i organisationen vara licensierade för:

- Microsoft Teams

Mer information finns i [tilldela licenser till användare.](../manage/assign-licenses-to-users.md)

När personer har varit aktiva i Teams minst en gång under de senaste 28 dagarna börjar du se insikterna.

## <a name="why-your-organizations-meetings-score-matters"></a>Därför är organisationens mötesresultat viktiga

Möten, där människor utforskar idéer, planerar, löser problem och fatta beslut, är en grundläggande fråga för organisationens produktivitet. Efterforskningar anger att när personer använder verktyg för onlinemöte effektivt brukar de spara upp till 104 minuter per vecka.

## <a name="how-we-calculate-the-meetings-score"></a>Så här räknar vi ut mötesresultatet

Vi ger en primär insyn i upplevelsen som innehåller nyckeltal för den här kategorin. Sedan används ett poängramverk, som anges nedan, för dessa mått för att beräkna din organisations poäng.

### <a name="primary-insight"></a>Primär insyn

Microsoft Teams kan integreras Outlook kalendern och har en mängd funktioner som gör dina möten mer engagerande och effektiva. Den primära insikten tittar på Microsoft Teams möten som har anordnats i din organisation. Den primära insikten spårar också hur många möten som följer minst en av metodtipsen för Teams möten.

:::image type="content" source="../../media/primaryinsights-meetings.png" alt-text="PRimary insikter för möten med metodtips.":::

1. **Sidhuvud:** Visar procentandelen onlinemöten på Microsoft Teams under de senaste 28 dagarna som haft video- eller skärmdelning under mötet.
2. **Brödtext:** Innehåller mer information om hur följande metodtips för engagemang under ett möte, till exempel användning av video eller skärmdelning, kan göra möten mer effektiva.
3. **Visualisering (aktuell status):**

      - I det här vågräta liggande stapeldiagrammet representerar den blå delen (färgade) procentandelen som visas i rubriken
      - Bråktalet (täljare/nämnare) används för att beräkna det procenttal som visas i rubriken
         - Täljare: Antalet onlinemöten Microsoft Teams bland annat personer från organisationen som har använt video- eller skärmdelning.
         - Nämnare: Antalet onlinemöten Microsoft Teams inklusive personer från organisationen som anordnats de senaste 28 dagarna.
      - Peer-prestandavärdet för det viktiga mätvärdet visas också som procent.
1. **Länk till resurser:** Välj den här länken om du vill visa hjälpinnehåll.

### <a name="scoring-framework"></a>Poängramverk

Mötesresultatet för organisationen mäter i vilken grad onlinemöten Microsoft Teams organisationen följt metodtips de senaste 28 dagarna. Det är viktat utifrån antalet personer i din organisation som deltar i mötena och mötenas varaktighet.

## <a name="explore-more-about-meetings-in-your-organization"></a>Utforska mer om möten i din organisation

Vi ger dig även stödinformation så att du förstår hur personer i organisationen kan genomföra möten. Måtten bidrar inte direkt till ditt Produktivitetsresultat, men kan hjälpa dig att skapa en handlingsplan som en del av din digitala omvandling.
