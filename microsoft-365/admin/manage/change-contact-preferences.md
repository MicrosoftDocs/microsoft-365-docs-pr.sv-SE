# <a name="change-your-contact-preferences-for-communications-from-microsoft"></a>Ändra dina kontaktinställningar för kommunikation från Microsoft

Använd sidan **Kontaktinställningar** om du vill uppdatera organisationens kontaktinformation och välj den Microsoft 365 produktrelaterad kommunikation som du vill ta emot.
  
Läs [Uppdatera administratörstelefonnumret och -e-postadressen Microsoft 365](update-phone-number-and-email-address.md) information om hur du ändrar administratörens mobiltelefonnummer och e-postadress.
  
Information om hur du ändrar företagets profilinformation, till exempel organisationens namn och adress, företagstelefonnummer och teknisk kontaktinformation finns i [Ändra adress, e-postadress och annan information för organisationen](change-address-contact-and-more.md).
  
## <a name="to-update-your-information"></a>Så här uppdaterar du din information
  
1. I administrationscentret väljer du **Inställningar** ikonen längst upp till höger i sidhuvudet.

2. Välj **Kontaktinställningar.**

3. Välj de typer av produktrelaterad kommunikation som du vill ta emot.

4. Ange det företagstelefonnummer, företagsmobilnummer eller den företagsadress du vill bli kontaktad på för produktrelaterad kommunikation.
    Du kan använda valfri e-postadress, inklusive din e-postadress från Microsoft, för att ta emot dessa meddelanden. Om du inte lägger till någon e-postadress för kontaktinställningar används din alternativa e-postadress (på sidan **Personlig information**).

5. Välj **Spara**.
  
När du Office 365 Enterprise, Microsoft 365 Business Standard eller Microsoft 365 Business Basic kan dina användare, beroende på landet eller regionen för användarna i organisationen, ha registrerats automatiskt för att få produktrelaterad kommunikation när du registrerade dig som administratör för Microsoft 365.
