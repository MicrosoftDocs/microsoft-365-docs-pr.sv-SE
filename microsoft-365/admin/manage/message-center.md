# <a name="message-center"></a>Meddelandecenter


Om du vill hålla reda på kommande ändringar, inklusive nya och ändrade funktioner, planerat underhåll eller andra viktiga meddelanden, går du till <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">Message center</a>. 
  
Så här öppnar du Meddelandecenter:

::: moniker range="o365-worldwide"

- Gå till **Health** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">hälsomeddelandecentret</a>i administrationscentret .

::: moniker-end

::: moniker range="o365-germany"

- Gå till **Hälsomeddelandecentret** i <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">administrationscentret</a> > **Message center**.

::: moniker-end

::: moniker range="o365-21vianet"
 
- Gå till **Hälsomeddelandecentret** i <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">administrationscentret</a> > **Message center**.

::: moniker-end

Du kan också använda [Microsoft 365 Admin-appen](https://go.microsoft.com/fwlink/p/?linkid=627216) på din mobila enhet för att visa Message Center, vilket är ett bra sätt att hålla dig uppdaterad med push-meddelanden.

Du kan välja en flik för att se **Alla aktiva meddelanden,** Meddelanden **av stor betydelse,** **Olästa meddelanden**och Avvisade **meddelanden**. Och du kan använda den **nedrullningsvänliga** menyn Filter för att se alla meddelanden i en viss meddelandekategori.

Här är en snabb översikt över den information du ser i varje kolumn. Markera någon av kolumnrubrikerna om du vill ändra sorteringsordningen.
  
|**Kolumn**|**Beskrivning**|
|:-----|:-----|

### <a name="major-updates"></a>Större uppdateringar

Större uppdateringar kan granskas genom att välja fliken **Hög prioritet** högst upp i meddelandecentret.

Större uppdateringar meddelas minst 30 dagar i förväg när en åtgärd krävs och kan omfatta:
  
- Ändringar av den dagliga produktiviteten, till exempel inkorg, möten, delegationer, delning och åtkomst

- Ändringar av teman, webbdelar och andra komponenter som kan påverka anpassade funktioner

- Ökar eller minskar till synlig kapacitet som lagring, antal regler, objekt eller varaktigheter

- Ändringar av produktmärkning som kan:

  - Orsaka förvirring för slutanvändare,

  - Resultera i ändringar i helpdeskprocesser och referensmaterial, eller

  - Ändra en URL

- En ny tjänst eller ett nytt program

- Ändringar som kräver en administratörsåtgärd (exklusive problem med att förhindra eller åtgärda)

- Ändringar i var dina data lagras
  
### <a name="preferences"></a>Inställningar

Om administrationen distribueras över hela organisationen kanske du inte vill eller behöver se inlägg om alla Microsoft 365-tjänster. Varje administratör kan:

- Ange inställningar som styr vilka meddelanden som visas i Meddelandecenter.
- Filtrera meddelanden
- Ange e-postinställningar för att ta emot en veckovis sammanfattning av alla meddelanden, e-postmeddelanden endast för större uppdateringar och e-postmeddelanden för datasekretessmeddelanden.  

::: moniker range="o365-worldwide"

1. Välj **Redigera inställningar** högst upp i Meddelandecenter.

2. Kontrollera att kryssrutan är markerad för varje tjänst som du vill övervaka. Avmarkera kryssrutorna för de tjänster som du vill filtrera bort från meddelandecentervyn.

3. Sammanfatta e-postmeddelanden är aktiverade som standard och skickas till din primära e-postadress. Om du vill sluta ta emot veckosammandraget avmarkerar du kryssrutan **Skicka en veckosammanfattning av mina meddelanden.** <br/><br/>E-postmeddelande för större uppdateringar är en separat kontroll. Om du vill få e-postmeddelanden om större uppdateringar kontrollerar du att **Skicka e-postmeddelanden till mig för större uppdateringar** har valts. Avmarkera kryssrutan om du vill sluta få e-post om större uppdateringar. <br><br/>Markera kryssrutan **bredvid Skicka e-postmeddelanden till mig för datasekretessmeddelanden för** att ta emot separata e-postmeddelanden om datasekretessmeddelanden (datasekretessmeddelanden ingår inte i veckosammandraget). <br><br/>Du kan välja eller rensa din primära e-postadress, men du kan inte ändra den. Om du vill ange andra e-postadresser som e-postmeddelandet med datasekretessen skickas till markerar du kryssrutan **Andra e-postadresser** och anger de andra e-postadresserna där du vill att e-postmeddelandet ska skickas. Ange e-postadressen för en Microsoft 365-grupp eller en distributionslista om fler än två personer ska få e-post för datasekretess.

4. Välj **Spara** om du vill behålla ändringarna.
  
::: moniker-end
