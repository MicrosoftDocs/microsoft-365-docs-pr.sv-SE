# <a name="message-center"></a>Meddelandecenter

Gå till Meddelandecenter om du vill hålla reda på kommande ändringar, till exempel nya och ändrade funktioner, planerat underhåll eller andra <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">viktiga meddelanden.</a>
  
Så här öppnar du Meddelandecenter:

::: moniker range="o365-worldwide"

- Gå till Hälsomeddelandecenter  > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">i administrationscentret.</a>

::: moniker-end

::: moniker range="o365-germany"

- Gå till <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">Hälsomeddelandecenter</a>  > **i administrationscentret.**

::: moniker-end

::: moniker range="o365-21vianet"

- Gå till <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">Hälsomeddelandecenter</a>  > **i administrationscentret.**

::: moniker-end

Du kan också använda [Microsoft 365 Admin-appen](https://go.microsoft.com/fwlink/p/?linkid=627216) på din mobila enhet för att visa Meddelandecenter, vilket är ett bra sätt att hålla dig aktuell med push-meddelanden.

### <a name="messages"></a>Meddelanden

I Meddelandecenter visas en vy över alla aktiva meddelanden i ett tabellformat. Som standard visas det senaste meddelandet högst upp i listan. Du kan välja **Tjänst för** att visa meddelanden för olika tjänster, till exempel Microsoft 365-appar, SharePoint Online osv.   Under **Tagga** kan du **välja påverkan på** administratörer, **datasekretess,** funktionsuppdatering, **huvuduppdatering,** ny funktion **eller meddelanden som påverkar** användare.   Under **Meddelandetillstånd** kan du välja **Favoriter,** **Olästa** eller **Uppdaterade** meddelanden.

På fliken Arkiv visas de meddelanden som du har arkiverat. Om du vill arkivera ett meddelande väljer du Arkivera i **meddelandefönstret.**

### <a name="major-updates"></a>Större uppdateringar

Större uppdateringar kan granskas genom att välja **huvuduppdateringen** i **listrutan** Taggar.

Större uppdateringar meddelas minst 30 dagar i förväg när en åtgärd krävs och kan omfatta:
  
- Ändringar av daglig produktivitet, till exempel inkorg, möten, möten, delning och åtkomst

- Ändringar av teman, webbdelar och andra komponenter som kan påverka anpassade funktioner

- Ökar eller minskar synlig kapacitet, till exempel lagring, antal regler, objekt eller varaktighet

- Ändringar av produktmärke som kan:

  - Orsaka förvirring för slutanvändarna

  - Resultera i ändringar av supportprocesser och referensmaterial, eller

  - Ändra en URL

- En ny tjänst eller ett nytt program

- Ändringar som kräver en administratörsåtgärd (endast för att förhindra eller åtgärda problem)

- Ändringar av var dina data lagras
  
### <a name="preferences"></a>Inställningar

Om administrationen har fördelats i organisationen kan det hända att du inte vill eller behöver se inlägg om alla Microsoft 365-tjänster. Varje administratör kan:

- Ange inställningar som styr vilka meddelanden som visas i Meddelandecenter.
- Filtrera meddelanden
- Ange e-postinställningar för att få en veckovis sammanfattning av alla meddelanden, e-postmeddelanden för endast större uppdateringar och e-postmeddelanden om datasekretessmeddelanden.  

::: moniker range="o365-worldwide"

1. Välj **Inställningar** längst upp i Meddelandecenter.

2. På fliken **Anpassad** vy kontrollerar du att kryssrutan är markerad för varje tjänst som du vill övervaka. Avmarkera kryssrutorna för de tjänster som du vill filtrera bort från vyn i Meddelandecenter.

3. Sammanfattningsmeddelanden är aktiverat som standard och skickas till din primära e-postadress. Om du vill sluta ta emot vecko digest avmarkerar du kryssrutan Skicka e-postaviseringar från **Meddelandecenter** på fliken **E-post.** 

   Du kan också ange upp till två e-postadresser avgränsade med semikolon.

   Du kan också välja vilka e-postmeddelanden du vill få samt en veckovis sammanfattning av de tjänster du väljer.

4. Välj **Spara** om du vill behålla ändringarna.
  
::: moniker-end

::: moniker range="o365-germany"

1. Välj **Inställningar för Redigera meddelandecenter** högst upp i Meddelandecenter.

2. Se till att knappen är inställd på **På** för varje tjänst du vill övervaka. Använd växlingsknappen för att ändra inställningen till **Av** för de tjänster som du vill filtrera bort från vyn i Meddelandecenter.

3. Sammanfattningsmeddelanden är aktiverat som standard och skickas till din primära e-postadress. Om du vill sluta ta emot vecko digest ändrar du inställningen Skicka en veckovis sammanfattning av **mina** meddelanden till **Av.** 

   E-postaviseringar för större uppdateringar är en separat kontroll. Om du vill få e-postmeddelanden om större uppdateringar kontrollerar du att Skicka **e-post för större uppdateringar** är **på.** Ändra inställningen till Av om **du inte** vill få e-post om större uppdateringar. 

   Om du vill få e-postmeddelanden om datasekretessmeddelanden kontrollerar du att Skicka **e-postmeddelanden om datasekretessmeddelanden** är **på.** Om du inte vill få de här meddelandena ändrar du inställningen till **Av.** (Datasekretessmeddelanden ingår inte i vecko digesten.)

   Du kan markera eller ta bort din primära e-postadress, men du kan inte ändra den. Om du vill ange andra e-postadresser som den veckovisa e-postsammanfattningen skickas till kontrollerar du att Skicka en veckovis sammanfattning **av mina meddelanden** är **På.** Ange e-postadressen för en Microsoft 365-grupp eller en distributionslista om du vill att sammanfattningen ska skickas till fler än två personer.

4. Välj **Spara** om du vill behålla ändringarna.<br/>

::: moniker-end

::: moniker range="o365-21vianet"

1. Välj **Inställningar** längst upp i Meddelandecenter.

2. På fliken **Anpassad** vy kontrollerar du att kryssrutan är markerad för varje tjänst som du vill övervaka. Avmarkera kryssrutorna för de tjänster som du vill filtrera bort från vyn i Meddelandecenter.

3. Sammanfattningsmeddelanden är aktiverat som standard och skickas till din primära e-postadress. Om du vill sluta ta emot vecko digest avmarkerar du kryssrutan Skicka e-postaviseringar från **Meddelandecenter** på fliken **E-post.** 

   Du kan också ange upp till två e-postadresser avgränsade med semikolon. <br><br/>Du kan också välja vilka e-postmeddelanden du vill få samt en veckovis sammanfattning av de tjänster du väljer.

4. Välj **Spara** om du vill behålla ändringarna.<br/>

::: moniker-end

## <a name="unsubscribe-from-message-center-emails"></a>Avbryta prenumerationen från e-postmeddelanden i Meddelandecenter

1. Sammanfattningsmeddelanden är aktiverat som standard och skickas till din primära e-postadress. Om du vill sluta ta emot vecko digest väljer du **Inställningar** och sedan **E-post.**

    - Avmarkera kryssrutan **Skicka en veckovis sammanfattning av** mina meddelanden.
    - E-postaviseringar för större uppdateringar är en separat kontroll. Om du inte vill få e-postmeddelanden om större uppdateringar kontrollerar du att Skicka e-post för större **uppdateringar** inte är markerad.
    -   Om du vill sluta ta emot e-postmeddelanden om datasekretessmeddelanden kontrollerar du att kryssrutan Skicka e-postmeddelanden för datasekretess inte är markerad.   (Datasekretessmeddelanden ingår inte i vecko digesten.)

2. Välj **Spara** om du vill behålla ändringarna.<br/>
