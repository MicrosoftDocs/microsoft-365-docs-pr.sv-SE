# <a name="message-center-in-office-365"></a>Meddelandecenter i Office 365


Om du vill hålla reda på kommande ändringar, inklusive nya och ändrade funktioner, planerat underhåll eller andra viktiga meddelanden, går du till <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">Message center</a>. Det är där vi lägger upp officiella meddelanden som låter dig ta ett proaktivt förhållningssätt till förändringshantering. Varje inlägg ger dig en översikt på hög nivå över en planerad ändring och hur den kan påverka användarna och länkar ut till mer detaljerad information som hjälper dig att förbereda dig.
  
Om du vill öppna Meddelandecenter kontrollerar du först att du har valt **växlingsknappen Prova det nya administrationscentret** på startsidan för administrationscentret och sedan:

::: moniker range="o365-worldwide"

- Gå till > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">hälsomeddelandecentret</a>i administrationscentret . **Health**

::: moniker-end

::: moniker range="o365-germany"

- Gå till > **hälsomeddelandecentret**i <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">administrationscentret</a>. **Health**

::: moniker-end

::: moniker range="o365-21vianet"
 
- Gå till > **hälsomeddelandecentret**i <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">administrationscentret</a>. **Health**

::: moniker-end

Du kan också använda [Microsoft 365 Admin-appen](https://go.microsoft.com/fwlink/p/?linkid=627216) på din mobila enhet för att visa Message Center, vilket är ett bra sätt att hålla dig uppdaterad med push-meddelanden.

### <a name="major-updates"></a>Större uppdateringar

Viktiga uppdateringar kan granskas genom att välja fliken **Hög prioritet** högst upp i meddelandecentret.

Större uppdateringar meddelas minst 30 dagar i förväg när en åtgärd krävs.
  
Större uppdateringar definieras som:
  
- Ändringar av den dagliga produktiviteten, till exempel inkorg, möten, delegationer, delning och åtkomst

- Ändringar av teman, webbdelar och andra komponenter som kan påverka anpassade funktioner

- Ökar eller minskar till synlig kapacitet som lagring, antal regler, objekt eller varaktigheter

- Ändringar av produktmärkning som kan:

  - Orsaka förvirring för slutanvändare,

  - Resultera i ändringar i helpdeskprocesser och referensmaterial, eller

  - Ändra en URL

- En ny tjänst eller ett nytt program

- Ändringar som kräver en administratörsåtgärd (exklusive problem med att förhindra eller åtgärda)

- Ändringar i var dina data lagras

När du öppnar ett större uppdateringsinlägg för att läsa det visas rubriken **Huvudstor uppdatering**, vem den **gäller för**, samt den aktuella utrullningsstatusen och en beskrivning av hur den påverkar dig.
  
### <a name="preferences"></a>Inställningar

Om administrationsarbetet har fördelats i organisationen, kan det hända att du inte vill eller behöver se inlägg om alla Office 365-tjänster. Varje administratör kan:

- Ange inställningar som styr vilka meddelanden som visas i Meddelandecenter.
- Filtrera meddelanden
- Ange e-postinställningar för att ta emot en veckovis sammanfattning av alla meddelanden, e-postmeddelanden endast för större uppdateringar och e-postmeddelanden för datasekretessmeddelanden.

2. Kontrollera att kryssrutan är markerad för varje tjänst som du vill övervaka. Avmarkera kryssrutorna för de tjänster som du vill filtrera bort från meddelandecentervyn.

3. Sammanfatta e-postmeddelanden är aktiverade som standard och skickas till din primära e-postadress.
