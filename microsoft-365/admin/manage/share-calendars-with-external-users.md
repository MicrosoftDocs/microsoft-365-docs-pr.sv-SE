# <a name="share-calendars-with-external-users"></a>Dela kalendrar med externa användare

Det är ibland nödvändigt för användarna att schemalägga möten med personer utanför organisationen. För att förenkla processen att hitta vanliga Mötes tider kan du använda Microsoft 365 för att göra kalendrar tillgängliga för dessa personer. Det här är personer som måste se lediga och upptagna tider för användare i organisationen, men har inte användar konton för Microsoft 365-organisationen.

Du kan aktivera kalender delning för alla användare i organisationen i administrations centret för Microsoft 365. När delning är aktiverat kan användarna använda Outlook Web App för att dela sina kalendrar med alla i eller utanför organisationen. Personer i organisationen kan visa den delade kalendern tillsammans med sin egen kalender. Personer utanför organisationen får ett e-postmeddelande med en URL som de kan använda för att visa kalendern. Användare i organisationen bestämmer när de ska dela och hur många som ska delas.

> [!NOTE]
> Om du vill dela kalendrar med en organisation som använder Exchange Server 2013 (en lösning på platsen), Exchange-administratören måste du konfigurera en autentisering relation med molnet. Detta kallas för Federation och måste uppfylla minst program varu krav. [Mer information finns i ](https://technet.microsoft.com/library/dd638083%28v=exchg.150%29.aspx) Starta.
  
## <a name="enable-calendar-sharing-using-the-microsoft-365-admin-center"></a>Aktivera kalender delning med administrations centret för Microsoft 365

1. Gå till **Inställningar** \> **organisations inställningar** i administrations centret.

2. På fliken **tjänster** väljer du **kalender**.
  
3. På sidan **kalender** väljer du om du vill låta användarna dela sina kalendrar med personer utanför organisationen som har Microsoft 365 eller Exchange. Välj om du vill tillåta anonyma användare (användare utan autentiseringsuppgifter) för att komma åt kalendrar via en e-postinbjudan.

4. Välj vilken typ av kalender information som ska vara tillgänglig för användarna. Du kan tillåta all information eller begränsa den till endast tid, ämne och plats.

## <a name="invite-people-to-access-calendars"></a>Bjuda in personer att få åtkomst till kalendrar

När delning har Aktiver ATS kan kalender ägare utöka inbjudningar till specifika användare. Se [Dela kalendern i Outlook Web App](https://support.microsoft.com/office/7ecef8ae-139c-40d9-bae2-a23977ee58d5) för anvisningar.
