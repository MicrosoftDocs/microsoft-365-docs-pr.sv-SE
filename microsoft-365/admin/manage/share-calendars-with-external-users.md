# <a name="share-calendars-with-external-users"></a>Dela kalendrar med externa användare

Ibland är det nödvändigt för användarna att schemalägga möten med personer utanför organisationen. För att förenkla processen med att hitta vanliga mötestider Microsoft 365 du göra kalendrar tillgängliga för dessa personer. Det här är personer som behöver se lediga och upptagna tider för användare i organisationen, men som inte har användarkonton för Microsoft 365 organisation.

Du kan aktivera kalenderdelning för alla användare i organisationen Microsoft 365 administrationscentret. När delning har aktiverats kan användarna använda Outlook Web App och dela sina kalendrar med personer i eller utanför organisationen. Personer i organisationen kan visa den delade kalendern tillsammans med sin egen kalender. Personer utanför organisationen får ett e-postmeddelande med en URL som de kan använda för att visa kalendern. Användarna i organisationen bestämmer när de ska dela och hur mycket de vill dela.

> [!NOTE]
> Om du vill dela kalendrar med en organisation som använder Exchange Server 2013 (en lösning på platsen), Exchange-administratören måste du konfigurera en autentisering relation med molnet. Detta kallas federation och måste uppfylla minimikraven för programvara. [Mer information finns i ](/exchange/sharing-exchange-2013-help) Starta.
  
## <a name="enable-calendar-sharing-using-the-microsoft-365-admin-center"></a>Aktivera kalenderdelning med hjälp Microsoft 365 administrationscentret

1. I administrationscentret går du till sidan **Inställningar** \> **organisation Inställningar**.

2. Välj **Kalender** på fliken **Tjänster.**
  
3. På sidan **Kalender** väljer du om du vill låta användare dela sina kalendrar med personer utanför din organisation som har Microsoft 365 eller Exchange. Välj om du vill tillåta anonyma användare (användare utan autentiseringsuppgifter) att få åtkomst till kalendrar via en e-postinbjudan.

4. Välj vilken typ av kalenderinformation som användarna ska få tillgång till. Du kan tillåta all information eller begränsa den till endast tid eller tid, ämne och plats.

## <a name="invite-people-to-access-calendars"></a>Bjuda in personer att få åtkomst till kalendrar

När delning har aktiverats kan kalenderägare utöka inbjudningar till specifika användare. Se [Dela kalendern i Outlook Web App](https://support.microsoft.com/office/7ecef8ae-139c-40d9-bae2-a23977ee58d5) för anvisningar.
