# <a name="get-started-with-the-roles-page"></a>Komma igång med sidan roller

På sidan Roller kan du ge användarna behörighet att utföra uppgifter i administrationscentren. Detta hjälper din organisation att sprida uppgifter till rätt personer och hjälper till att skydda dina data.

:::image type="content" source="../../media/roles-main-page.png" alt-text="En figur som visar administratörsroller":::

> [!TIP]
> Letar du efter de detaljerade rollbeskrivningarna? Kontrollera [administratörsrollbehörigheter i Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#available-roles) och Om [administratörsroller](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles).

## <a name="about-the-admin-roles-page"></a>Om sidan administratörsroller

Du kan exportera administratörslistan samt söka och filtrera efter roll.

:::image type="content" source="../../media/admin-role-page-options.png" alt-text="Filtrera eller importera administratörsroller":::

|||
|:-----|:-----|
|  <br/> |Använd **exportadministratörslista** för att få en fullständig lista över alla administratörsanvändare i organisationen. Listan lagras i en Csv-excel-fil.   <br/> |

## <a name="get-the-most-out-of-the-roles"></a>Få ut mesta av rollerna

Läs innehållet nedan om du vill veta mer om de olika administratörsrollerna och vilka uppgifter rollerna kan utföra i organisationen.

> [!NOTE]
Detta är inte en uttömmande lista över alla behörigheter som dessa roller har. Välj länken **Läs mer** om du vill ha mer information om varje roll.

### <a name="exchange-admin"></a>Exchange-administratör

Tilldela rollen Exchange-administratör till användare som behöver visa och hantera användarens e-postpostlådor, Microsoft 365-grupper och Exchange Online. De kan också öppna och hantera tjänstförfrågningar till Microsoft-support. [Läs mer](https://docs.microsoft.com/office365/admin/add-users/about-exchange-online-admin-role)

### <a name="global-admin"></a>Global administratör

Tilldela den globala administratörsrollen till användare som behöver global åtkomst till de flesta hanteringsfunktioner och data i Microsofts onlinetjänster. Att ge för många användare global åtkomst är en säkerhetsrisk och vi rekommenderar att du har mellan två och fyra globala administratörer. Endast globala administratörer kan återställa lösenord för alla användare och lägga till och hantera domäner. De kan också öppna och hantera tjänstförfrågningar till Microsoft-support . Den person som registrerade sig för Microsofts onlinetjänster blir automatiskt global administratör. [Läs mer](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)

### <a name="global-reader"></a>Global läsare

Tilldela den globala administratörsrollen för läsare till användare som behöver visa administratörsfunktioner och inställningar i alla administrationscenter som den globala administratören kan visa. Den globala roll för läsarens administratör kan inte redigera några inställningar. [Läs mer](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)

### <a name="helpdesk-admin"></a>Supportadministratör

Tilldela administratörsrollen Helpdesk till användare som vill återställa lösenord, tvinga användare att logga ut för eventuella säkerhetsproblem. De kan också öppna och hantera tjänstförfrågningar till Microsoft-support. Helpdesk-administratören kan bara hjälpa användare och användare som inte har administratörsbe tilldelats dessa roller: katalogläsare, gäst inbjudna, Helpdesk-administratör, Message center-läsare och Rapportläsare. [Läs mer](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)

### <a name="service-admin"></a>Tjänstadministratör

Tilldela rollen tjänstadministratör till användare som behöver skapa tjänstbegäranden för Azure-, Microsoft 365- och Office 365-tjänster. [Läs mer](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)

### <a name="sharepoint-admin"></a>SharePoint-administratör

När du köper en Microsoft 365-prenumeration skapas en gruppwebbplats automatiskt och den globala administratören anges som den primära webbplatssamlingsadministratören. Tilldela SharePoint-administratörsrollen till användare som du vill komma åt i Administrationscentret för SharePoint. Användare med sharepoint-administratörsrollen kan skapa och hantera webbplatssamlingar, utse administratörer för webbplatssamlingar och hantera användarprofiler. Användare med sharepoint-administratörsrollen kan också hantera Microsoft 365-grupper och öppna tjänstförfrågningar via Microsoft-support. [Läs mer](https://docs.microsoft.com/sharepoint/sharepoint-admin-role)

### <a name="teams-service-admin"></a>Teams-tjänstadministratör

Tilldela administratörsrollen Teams till användare som du vill komma åt och hantera teams & Administrationscenter för Skype. Användare med administratörsrollen Teams kan också hantera Microsoft 365-grupper och öppna tjänstförfrågningar via Microsoft-support. [Läs mer](https://docs.microsoft.com/MicrosoftTeams/using-admin-roles)

### <a name="user-admin"></a>Användaradministratör

Tilldela rollen användaradministratör till användare som du vill komma åt och hantera återställning av användarlösenord och hantera användare och grupper. De kan också öppna och hantera tjänstförfrågningar till Microsoft-support. [Läs mer](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)
