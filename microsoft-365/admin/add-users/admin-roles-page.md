# <a name="get-started-with-the-roles-page"></a>Kom igång med rollsidan

På sidan Roller kan du ge användarna behörighet att utföra uppgifter i administrationscenter. Detta hjälper din organisation att sprida uppgifter till lämpliga personer och hjälper till att skydda dina data.

:::image type="content" source="../../media/roles-main-page.png" alt-text="En figur som visar administratörsroller":::

> [!TIP]
> Letar du efter de detaljerade rollbeskrivningarna? Kolla [in administratörsrollbehörigheter i Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#available-roles) och Om [administratörsroller](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles).

## <a name="about-the-admin-roles-page"></a>Om sidan administratörsroller

Du kan exportera administratörslistan samt söka och filtrera efter roll.

:::image type="content" source="../../media/admin-role-page-options.png" alt-text="Filtrera eller importera administratörsroller":::

|||
|:-----|:-----|
|  <br/> |Använd **Exportera administratörslista** för att få en fullständig lista över alla administratörsanvändare i organisationen. Listan lagras i en CSV-fil i Excel.   <br/> |
|  <br/> |Använd **Sök** för att söka efter en administratörsroll och se dina användare som har tilldelats den rollen.   <br/> |
|  <br/> |Använd **Filter** om du vill ändra vyn för visade administratörsroller.   <br/> |

## <a name="get-the-most-out-of-the-roles"></a>Få ut det mesta av rollerna

Läs innehållet nedan om du vill veta mer om de olika administratörsrollerna och vilka uppgifter rollerna kan utföra i organisationen.

> [!NOTE]
Detta är inte en uttömmande lista över alla behörigheter som dessa roller har. Välj länken **Läs mer** om du vill ha mer information om varje roll.

### <a name="exchange-admin"></a>Exchange-administratör

Tilldela rollen som Exchange-administratör till användare som behöver visa och hantera postlådor, Office 365-grupper och Exchange Online som tillhör din användare. De kan också öppna och hantera serviceförfrågningar till Microsoft-supporten. [Mer information](https://docs.microsoft.com/office365/admin/add-users/about-exchange-online-admin-role)

### <a name="global-admin"></a>Global administratör

Tilldela den globala administratörsrollen till användare som behöver global åtkomst till de flesta hanteringsfunktioner och data i Microsofts onlinetjänster. Att ge för många användare global åtkomst är en säkerhetsrisk och vi rekommenderar att du har mellan två och fyra globala administratörer. Endast globala administratörer kan återställa lösenord för alla användare och lägga till och hantera domäner. De kan också öppna och hantera serviceförfrågningar till Microsoft-supporten . Personen som registrerade sig för Microsofts onlinetjänster blir automatiskt en global administratör. [Läs mer](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)

### <a name="global-reader"></a>Global läsare

Tilldela den globala administratörsrollen för läsare till användare som behöver visa administratörsfunktioner och inställningar i alla administratörscenter som den globala administratören kan visa. Rollen för den globala administratörsrollen för läsare kan inte redigera några inställningar. [Mer information](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)

### <a name="helpdesk-admin"></a>Supportadministratör

Tilldela rollen Helpdesk-administratör till användare som vill återställa lösenord, tvinga användare att logga ut för eventuella säkerhetsproblem. De kan också öppna och hantera serviceförfrågningar till Microsoft-supporten. Helpdesk-administratören kan bara hjälpa användare och användare som tilldelats dessa roller som inte administreras: Katalogläsare, Gästinbjudare, Helpdesk-administratör, meddelandecenterläsare och rapportläsare. [Mer information](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)

### <a name="service-admin"></a>Tjänstadministratör

Tilldela tjänstadministratörsrollen till användare som behöver skapa tjänstbegäranden för Azure-, Microsoft 365- och Office 365-tjänster. [Mer information](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)

### <a name="sharepoint-admin"></a>SharePoint-administratör

När du köper en Microsoft 365- eller Office 365-prenumeration skapas en gruppwebbplats automatiskt och den globala administratören anges som den primära webbplatssamlingsadministratören. Tilldela SharePoint-administratörsrollen till användare som du vill komma åt administrationscentret för SharePoint. Användare med rollen SharePoint-administratör kan skapa och hantera webbplatssamlingar, utse administratörer för webbplatssamlingar och hantera användarprofiler. Användare med sharepoint-administratörsrollen kan också hantera Office 365-grupper och öppna tjänstförfrågningar via Microsoft-support. [Mer information](https://docs.microsoft.com/sharepoint/sharepoint-admin-role)

### <a name="teams-service-admin"></a>Teams-tjänstadministratör

Tilldela rollen Teams-administratör till användare som du vill komma åt och hantera teamen & Skype-administrationscenter. Användare med rollen Teams-administratör kan också hantera Office 365-grupper och öppna tjänstförfrågningar via Microsoft-support. [Mer information](https://docs.microsoft.com/MicrosoftTeams/using-admin-roles)

### <a name="user-admin"></a>Användaradministratör

Tilldela användaradministratörsrollen till användare som du vill komma åt och hantera återställningar av användarlösenord och hantera Office 365-användare och grupper. De kan också öppna och hantera serviceförfrågningar till Microsoft-supporten. [Mer information](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles#roles-available-in-the-microsoft-365-admin-center)
