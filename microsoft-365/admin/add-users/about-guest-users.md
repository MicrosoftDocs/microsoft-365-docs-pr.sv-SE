# <a name="guest-users-in-microsoft-365-admin-center"></a>Gäst användare i administrations Center för Microsoft 365

Alla gäster som du lägger till i Microsoft Teams, SharePoint eller Azure Active Directory läggs också till i **gäst användarens** lista i administrations centret för Microsoft. Gäster kan delta i möten, Visa dokument och chatt i team som de är inbjudna till.
När en användare visas i listan **gäst användare** kan du ta bort deras åtkomst där.

Om du vill visa gäst användare, expandera **användare** i administrations centret för Microsoft 365 och välj **gäst användare**.

## <a name="add-guests-in-azure-active-directory"></a>Lägga till gäster i Azure Active Directory

Information om hur du lägger till gäster i Azure Active Directory finns i [lägga till gäst användare](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal).

När du har lagt till en användare kan du också tilldela dem till en grupp eller ge dem till gång till ett program i organisationen. När du har lagt till en användare i Azure AD-portalen kommer den användaren också att visas på sidan **gäst användare** i administrations centret för Microsoft 365.
När en användare har lagts till i listan **gäst användare** kan de [läggas till i grupper](../create-groups/manage-guest-access-in-groups.md#add-guests-to-a-microsoft-365-group-from-the-admin-center) i administrations centret för Microsoft 365.

Se [lägga till gäster i bulk](https://docs.microsoft.com/azure/active-directory/b2b/tutorial-bulk-invite) för att bjuda in flera gäster att samar beta med din organisation.


## <a name="remove-a-guest"></a>Ta bort en gäst

1. Expandera **användare** i administrations centret för Microsoft 365 och välj **gäst användare**.
1. På sidan **gäst användare** väljer du den användare du vill ta bort och väljer sedan **ta bort en användare**. 

Om du vill ta bort användare i Azure AD-portalen läser du [ta bort en gäst användare och resurser](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal#clean-up-resources).
