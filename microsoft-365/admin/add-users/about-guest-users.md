# <a name="guest-users-in-microsoft-365-admin-center"></a>Gästanvändare i administrationscentret för Microsoft 365

Alla gäster som du lägger till i Microsoft Teams, SharePoint eller Azure Active Directory läggs också till i listan **Gästanvändare** i administrationscentret för Microsoft. Gäster kan delta i möten, visa dokument och chatta i Teams som de har bjudits in till.
När en användare visas i listan **Gästanvändare** kan du ta bort deras åtkomst där.

Om du vill visa gästanvändare expanderar du Användare i det vänstra navigeringsfältet i administrationscentret för Microsoft 365 och väljer **sedan Gästanvändare.**

## <a name="add-guests-in-azure-active-directory"></a>Lägga till gäster i Azure Active Directory

Information om hur du lägger till gäster i Azure Active Directory finns i [Lägga till gästanvändare.](/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal)

När du har lagt till en användare kan du också tilldela dem till en grupp eller ge dem åtkomst till ett program i din organisation. När du har lagt till en användare i Azure AD-portalen visas den användaren även på sidan Gästanvändare i administrationscentret för Microsoft 365. 
När en användare har lagts till i **listan Gästanvändare** kan de läggas till [i Grupper](../create-groups/manage-guest-access-in-groups.md#add-guests-to-a-microsoft-365-group-from-the-admin-center) i administrationscentret för Microsoft 365.

Se [lägga till gäster i grupp](/azure/active-directory/b2b/tutorial-bulk-invite) för att bjuda in flera gäster för att samarbeta med organisationen.


## <a name="remove-a-guest"></a>Ta bort en gäst

1. I administrationscentret för Microsoft 365 expanderar **du Användare** och väljer sedan **Gästanvändare.**
1. På sidan **Gästanvändare** väljer du den användare du vill ta bort och väljer sedan Ta **bort en användare.** 

Information om hur du tar bort användare i Azure AD-portalen finns [i Ta bort en gästanvändare och resurser.](/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal#clean-up-resources)
