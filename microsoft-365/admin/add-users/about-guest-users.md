# <a name="guest-users-in-microsoft-365-admin-center"></a>Gästanvändare i Microsoft 365 administrationscenter

När du lägger till gäster i dina Microsoft Teams eller SharePoint läggs dessa användare till i listan Gästanvändare i administrationscentret. Du kan också lägga till gästanvändare i Azure Active Directory så att de kan samarbeta med organisationens användare. Gästanvändare kan delta i möten, visa dokument och chatta i Teams som de är inbjudna till.
När en användare har lagts till i listan Gästanvändare kan du ta bort deras åtkomst där.

Om du vill visa gästanvändare expanderar du **Användare**i administrationscentret för Microsoft 365 i microsoft 365 i den vänstra navigeringscentralen och väljer sedan **Gästanvändare**.

## <a name="add-guest-users-in-azure-active-directory"></a>Lägga till gästanvändare i Azure Active Directory

Information om hur du lägger till gästanvändare i Azure Active Directory finns [i lägga till gästanvändare](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal).

När du lägger till en användare kan du också tilldela dem till en grupp eller ge dem åtkomst till en app i organisationen. När du har lagt till en uer i Azure AD-portalen visas även den användaren på sidan Gästanvändare i administrationscentret för Microsoft 365.
När en användare har lagts till i listan Gästanvändare kan de [läggas till](../create-groups/manage-guest-access-in-groups.md#add-guests-to-a-microsoft-365-group-from-the-admin-center) i grupper i microsoft 365-administrationscentret.

Se [Lägga till gästanvändare i grupp](https://docs.microsoft.com/azure/active-directory/b2b/tutorial-bulk-invite) om du vill bjuda in flera gäster att samarbeta med din organisation.


## <a name="remove-a-guest-user"></a>Ta bort en gästanvändare

Så här tar du bort en gästanvändare:

1. Expandera **Användare** i administrationscentret för Microsoft 365 och välj sedan **Gästanvändare**.
1. På sidan **Gästanvändare** väljer du den användare som du vill ta bort och väljer sedan **Ta bort en användare**. 

Information om hur du tar bort användare i Azure AD-portalen finns i [ta bort en gästanvändare och resurser](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal#clean-up-resources).
