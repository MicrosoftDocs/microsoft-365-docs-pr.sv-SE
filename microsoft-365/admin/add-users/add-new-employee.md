# <a name="add-a-new-employee-to-microsoft-365"></a>Lägga till en ny anställd i Microsoft 365

Den här artikeln hjälper dig att få en ny anställd att komma igång med Microsoft 365 för företag. Vi utgår från att du är administratör och redan har installerat [Microsoft 365](../setup/setup.md)och att nu har någon ny börjat på ditt företag.
  
Du är på rätt plats om den nya medarbetaren behöver Microsoft 365 och du använder ett [Microsoft 365-abonnemang](https://products.office.com/business/compare-office-365-for-business-plans) som gör att du kan installera Office-appar som Word och Excel på en dator. 
  
 **Är du inte administratör?** [Lär dig hantera Microsoft 365 hjälper](https://support.microsoft.com/office/396b8d9e-e118-42d0-8a0d-87d1f2f055fb) företag och hemanvändare att konfigurera. 
  
 **Saknar du Office-appar i ditt abonnemang?** Följ stegen nedan, men hoppa över avsnitten för att installera appar. Använd [onlineversionerna av Office i](https://support.microsoft.com/office/91a4ec74-67fe-4a84-a268-f6bdf3da1804) stället. 
  
Här är en kortfattad översikt: 
  
|**Steg**|**Varför?**|
|:-----|:-----|
|[Steg 1: Skapa ett Microsoft 365-konto för den anställda](#step-1-create-a-microsoft-365-account-for-the-employee) <br/> |Varje gång en ny anställd ansluter till ditt företag skapar du ett konto för dem så att de kan börja använda Microsoft 365.  <br/> |
|[Steg 2: Ge den anställda deras användar-ID och lösenord](#step-2-give-the-employee-their-user-id-and-password) <br/> |När du skapar ett konto får du ett ID och lösenord som du kan överföra till den anställda så att han eller hon kan logga in.  <br/> |
|[Steg 3: Förklara var du loggar in](#step-3-explain-where-to-sign-in) <br/> |Inloggningsplatsen är [https://www.office.com](https://www.office.com) <br/> |
|[Steg 4: Hjälp den anställda att komma igång](#step-4-help-your-employee-get-started) <br/> |Låt den anställda veta hur de ska använda OneDrive eller gruppwebbplatser i organisationen.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Steg 1: Skapa ett Microsoft 365-konto för den anställda


Instruktioner finns i [Lägga till användare och tilldela licenser samtidigt.](add-users.md) När du konfigurerade den nya medarbetaren kan du välja att skicka inloggningsuppgifter till den anställdas personliga konto. På så sätt får de ett e-postmeddelande från Microsoft Online Service-teamet som beskriver hur de loggar in på Microsoft 365.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Steg 2: Ge den anställda deras användar-ID och lösenord


Såvida du inte har skickat det till den anställdas personliga e-postadress skriver du ut den anställdas inloggningsnamn och lösenord och ger dem till dem. Eller så kan du tala om det för honom eller henne via telefon.
  
Eftersom de ännu inte har tillgång till sin e-post ska du inte skicka informationen till den e-postadressen.
  
## <a name="step-3-explain-where-to-sign-in"></a>Steg 3: Förklara var du loggar in 


Precis som med Facebook, Amazon och Gmail loggar den anställda in för att använda Microsoft 365. Ge dem följande inloggningsinformation:
  
- Logga in på [https://www.office.com](https://www.office.com) .
    
- Välj **Logga in** och ange sedan användar-ID och lösenord.
