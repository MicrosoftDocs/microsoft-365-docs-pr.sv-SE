# <a name="add-a-new-employee-to-microsoft-365"></a>Lägga till en ny anställd i Microsoft 365

Den här artikeln hjälper dig att gå ombord på en ny anställd till Microsoft 365 för företag. Vi förutsätter att du är administratör och att du redan har [konfigurerat Microsoft 365](../setup/setup.md)och nu har du någon ny ansluter till företaget.
  
Du är på rätt plats om den nya medarbetaren behöver Microsoft 365 och du använder ett [Microsoft 365-abonnemang](https://products.office.com/business/compare-office-365-for-business-plans) som gör att du kan installera Office-appar som Word och Excel på en dator. 
  
 **Är du inte administratör?** [Lär dig igenom Microsoft 365](https://support.office.com/article/office-basics-video-training-396b8d9e-e118-42d0-8a0d-87d1f2f055fb) hjälper företags- och hemanvändare med konfigurerat. 
  
 **Inga Office-appar i ditt abonnemang?** Följ stegen nedan, men hoppa över avsnitten för att installera appar. Använd [onlineversionerna av Office](https://support.office.com/article/91a4ec74-67fe-4a84-a268-f6bdf3da1804.aspx) i stället. 
  
Här är en snabb översikt: 
  
|**Steg**|**Varför göra så här?**|
|:-----|:-----|
|[Steg 1: Skapa ett Microsoft 365-konto för medarbetaren](#step-1-create-a-microsoft-365-account-for-the-employee) <br/> |Varje gång en ny anställd ansluter till ditt företag skapar du ett konto åt dem så att de kan börja använda Microsoft 365.  <br/> |
|[Steg 2: Ge den anställde deras användar-ID och lösenord](#step-2-give-the-employee-their-user-id-and-password) <br/> |När du skapar ett konto får du ett ID och lösenord som du kan skicka till din medarbetare så att de kan logga in.  <br/> |
|[Steg 3: Förklara var du ska logga in](#step-3-explain-where-to-sign-in) <br/> |Inloggningsplatsen är[https://www.office.com](https://www.office.com) <br/> |
|[Steg 4: Hjälp din medarbetare att komma igång](#step-4-help-your-employee-get-started) <br/> |Meddela den anställde hur du använder OneDrive eller några gruppwebbplatser i organisationen.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Steg 1: Skapa ett Microsoft 365-konto för medarbetaren


Instruktioner finns [i Lägga till användare individuellt eller i grupp i Microsoft 365 - Hjälp om administratörer](add-users.md). När du konfigurerar den nya medarbetaren kan du välja att skicka inloggningsuppgifter till medarbetarens personliga konto. På så sätt får de ett e-postmeddelande från Microsoft Online Service Team som talar om för dem hur de ska logga in på Microsoft 365.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Steg 2: Ge den anställde deras användar-ID och lösenord


Om du inte har skickat den till deras personliga e-postadress skriver du ut den anställdes inloggningsnamn och lösenord och lämnar det till dem. Eller berätta informationen via telefon.
  
Eftersom de ännu inte har tillgång till sin e-post ska du inte skicka informationen till den e-postadressen.
  
## <a name="step-3-explain-where-to-sign-in"></a>Steg 3: Förklara var du ska logga in 


Precis som Facebook, Amazon eller Gmail loggar din medarbetare in för att använda Microsoft 365. Ge dem följande inloggningsinformation:
  
- Logga in [https://www.office.com](https://www.office.com)på .
    
- Välj **Logga in**och skriv sedan användar-ID och lösenord.
