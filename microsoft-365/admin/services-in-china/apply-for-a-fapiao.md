# <a name="apply-for-a-fapiao-for-office-365-operated-by-21vianet"></a>Ansök om en fapiao för Office 365 som drivs av 21Vianet

> [!NOTE]
> Administrationscentret förändras. Om dina erfarenheter inte överensstämmer med uppgifterna som visas här kan du läsa mer i [Om det nya administrationscentret för Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet).

Du kan skicka din fapiao-begäran till 21Vianet fapiao Management System cirka tre dagar efter att du har betalat. När du har skickat din fapiao-begäran kommer den att behandlas om två dagar.
  
![Flödes schema som visar fapiao program process.](../../media/bf14884a-53f9-4c53-971c-b9b8ad6ec8d3.png)
  
## <a name="step-1-apply-for-a-fapiao"></a>Steg 1: Ansök om en fapiao

Det finns två sätt att ansöka om ett fapiao:
  
1. När du har gjort en betalning på webbplatsen Office 365 som drivs av 21Vianet får du ett e-postmeddelande med beställnings nummer och en länk till fapiao hanterings system. Du kan använda länken i e-postmeddelandet för att skapa ett konto i <a href="https://go.microsoft.com/fwlink/p/?linkid=837466" target="_blank">21Vianet-fapiao informations hanterings system</a>.

    Eller

2. Du kan lägga till en fapiao från <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">administrations centret</a>.

    ![Sidan räkningar i administrations centret för Microsoft 365.](../../media/a6e3b953-abd4-46aa-a910-08c517915a21.png)
  
## <a name="step-2-register-with-the-21vianet-fapiao-management-system"></a>Steg 2: registrera dig med det fapiao hanterings systemet 21Vianet

> [!NOTE]
> Du kan använda samma e-postnamn och lösen ord som ditt Office 365-konto, eller så kan du använda något annat.
  
1. Gå till <a href="https://go.microsoft.com/fwlink/p/?linkid=837466" target="_blank">21Vianet fapiao informations hanterings system</a>.

2. Ange beställnings nummer, e-postadress och lösen ord i registrerings formuläret och välj sedan **registrering**.

    ![Fönstret 21Vianet Fapioa Management System registrerings sida.](../../media/60d39184-95b2-4ea4-a8a2-3e11763bec87.png)
  
3. När registreringen är färdig skickar systemet ett aktiverings meddelande till din e-postadress. Öppna e-postmeddelandet och välj länken för att aktivera ditt konto.

## <a name="step-3-submit-your-bill-for-a-fapiao"></a>Steg 3: skicka in din räkning för en fapiao

1. Logga in på <a href="https://go.microsoft.com/fwlink/p/?linkid=837465" target="_blank">fapiao Management System</a>.

2. Välj fakturerings post och välj sedan **Använd faktura**.

    > [!NOTE]
    > Betalnings systemet finns på en plattform från tredje part och tar tre dagar att synkronisera ordern och betalnings posten.
  
    ![Fapiao-sidan som visar de senaste beställningarna.](../../media/b319767d-1d10-4cb4-b270-c5fbcee1368e.png)
  
3. Välj typ av fapiao, ange den information som krävs och välj sedan **Nästa**.

    ![Fapiao där du väljer fapiao-typen.](../../media/56fe3db1-c20f-4082-a39d-02d7ac41fec8.png)
  
    > [!NOTE]
    > - För en vanlig moms fapiao behöver du bara ange köparens namn.
    > - Vid behov kan du använda en fapiao med en annan rubrik. Men du kan bara använda en fapiao-rubrik för en faktura i systemet. Om du vill dela upp fapiao i olika mängder eller rubriker ber vi <a href="https://portal.partner.microsoftonline.cn/Support/SupportOverview.aspx" target="_blank">dig skicka din förfrågan i administrations centret</a>.
    > - Nästa gång du ansöker om en fapiao visas automatiskt den föregående fapiao-informationen.
    > - Om du behöver ett inköps intyg eller en åter betalning måste betalar namn och fapiao titel stämma överens.

4. Välj en leverans metod och ange Utskicks informationen. Du kan välja Yunda eller sa (frakt hämt). Du kan också gå till 21Vianet-Shanghai gren för att få fapiao. Välj **Nästa**.

    ![Formuläret för fapiao visar e-postinformation.](../../media/bba500b4-a51d-477b-81a7-9113b08d39f1.png)

    ![Bekräftelse sidan i fapiao-formuläret.](../../media/18706d9d-defc-4285-8fd3-990448b44a18.png)
  
## <a name="step-4-check-application-progress"></a>Steg 4: kontrol lera programmets förlopp

Programmet överförs automatiskt till 21Vianet-tjänst Center och kommer att utföras inom två arbets dagar.
  
När du har skickat programmet kan du kontrol lera förloppet när som helst. Vi kommer att uppdatera statusen för din ansökan med kommentarer som **emitterat** eller **skickat**.
  
![Fönstret 21Vianet fapiao Management system som visar status för begäran.](../../media/6cd696ec-d630-4fce-9f27-935a0d5f0ebe.png)
  
När fapiao är slut visar systemet ett meddelande och visar den förväntade tiden för att skicka in fapiao.
  
![Fel meddelande som visas när 21Vianet fapiao-hanterings systemet inte används.](../../media/effe0796-83aa-4a91-a488-15d6f58c01dc.png)
  
## <a name="faqs"></a>Vanliga frågor och svar

### <a name="what-services-can-i-get-from-online-support"></a>Vilka tjänster kan jag få från online-support?

Du kan kontrol lera förloppet för din fapiao-begäran och ta reda på varför du inte har fått fapiao.
  
Om du vill ändra fapiao-namnet ber vi dig skicka fapiao tillbaka till oss så återskapas fapiao. Du kan skicka begäran via <a href="https://portal.partner.microsoftonline.cn/Support/SupportOverview.aspx" target="_blank">administrations centret</a>.
  
![Administrations centrets support formulär som används för att skicka en fapiao tillbaka för att begära en rubrik ändring.](../../media/2a413e9e-f30b-4f26-adbf-6287cc217a0f.png)
  
### <a name="how-do-i-change-my-login-email-address-and-password"></a>Hur ändrar jag min inloggnings-e-postadress och mitt lösen ord?

1. Logga in på <a href="https://go.microsoft.com/fwlink/p/?linkid=837465" target="_blank">fapiao</a>och välj sedan **ändra e-postadress**.

    ![Den nedrullningsbara menyn efter att en användare har loggat in på 21Vianet fapiao Management system.](../../media/ee6de24b-6be2-41e6-8aec-e0c3cb0ea35e.png)
  
2. Om du har glömt ditt inloggnings lösen ord kan du återställa lösen ordet med din inloggnings-e-postadress. Fapiao skickar ett nytt lösen ord till din e-postadress. Du kan använda det nya lösen ordet för att logga in.

    ![Sekventiella bilder som visar hur du återställer ditt lösen ord i 21Vianet-fapiao.](../../media/2edb0a47-1286-4792-804d-7e84534c8370.png)
  
3. Om du har glömt din inloggnings-e-postadress kontaktar du 21Vianet kund tjänst på (86) 400-089-0365.

### <a name="how-do-i-find-my-order-id"></a>Hur hittar jag mitt order-ID?

1. I [administrations centret](https://go.microsoft.com/fwlink/p/?linkid=850627)går du till sidan **betalnings** \> **räkningar &-betalningar** .

2. Leta reda på den faktura du vill använda, Välj att visa eller Välj att ladda ned PDF-filen.

### <a name="what-if-i-enter-the-wrong-email-address-when-i-register"></a>Vad händer om jag anger fel e-postadress när jag registrerar mig?

Om du anger fel e-postadress när du registrerar får du inte e-postmeddelandet. Registrerings länken i e-postmeddelandet upphör automatiskt efter 24 timmar. Du kan återgå till <a href="https://go.microsoft.com/fwlink/p/?linkid=837466" target="_blank">registrerings sidan</a> och registrera dig igen med rätt e-postadress.
  
### <a name="what-if-i-dont-receive-an-activation-email"></a>Vad gör jag om jag inte får ett e-postmeddelande?

Om du inte får ett konto för att aktivera e-post inom 24 timmar efter att du har registrerat dig går du till <a href="https://go.microsoft.com/fwlink/p/?linkid=837466" target="_blank">21Vianet fapiao informations hanterings system</a>, anger din e-postadress och väljer sedan **skicka e-postmeddelandet**. Systemet skickar om e-postadressen för konto aktivering till din registrerade e-postadress.
  
![Fönstret 21Vianet Fapioa Management System registrerings sida.](../../media/60d39184-95b2-4ea4-a8a2-3e11763bec87.png)
  
Om du fortfarande inte får ett e-postmeddelande för aktivering, kontakta 21Vianet kund tjänst på (86) 400-089-0365.
