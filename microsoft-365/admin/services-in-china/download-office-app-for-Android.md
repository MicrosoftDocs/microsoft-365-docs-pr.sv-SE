# <a name="office-app-for-android-for-office-365-operated-by-21vianet"></a>Office-appen för Android för Office 365 som drivs av 21Vianet

Microsoft Office-appen för Android kombinerar program för Word, Excel och PowerPoint till en enda app som kan laddas ner för Android-telefoner. Med Office-appen för Android kan du ansluta till Office 365 precis som med mobil programmen Word, Excel och PowerPoint. Office-appen för Android-nedladdning påverkar inte befintliga installationer av Word, Excel och PowerPoint.

Några Office-appar för Android-funktioner är inte tillgängliga för Office 365 som drivs av 21Vianet-kunder:

- Bild till text och bild till tabell 
- Konvertera foton till Word-dokument från lins 
- Åtgärden Överför filer 
- Anteckningarna förblir lokala och synkroniseras inte till servern
- Länk förhands granskning i QR
- Live persona kort i avsnittet mig
- Klassificering, märkning och skydd (CLP)

## <a name="security-considerations"></a>Säkerhets aspekter

Om din organisation skickar appar till mobila enheter i personalen föreslår vi att du ersätter programmen Word, Excel och PowerPoint med Office-appen för Android.
