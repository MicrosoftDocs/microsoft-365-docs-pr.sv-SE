# <a name="office-app-for-ios-for-office-365-operated-by-21vianet"></a>Office-appen för iOS för Office 365 som drivs av 21Vianet

Microsoft Office-appen för iOS kombinerar program för Word, Excel och PowerPoint till en enda app som kan laddas ner för iOS-telefoner. Med Office-appen för iOS kan du ansluta till Office 365 precis som med mobil programmen Word, Excel och PowerPoint. Office-appen för iOS-nedladdning påverkar inte befintliga installationer av Word, Excel och PowerPoint.

Några Office-appar för iOS-funktioner är inte tillgängliga för Office 365 som drivs av 21Vianet-kunder:

- Bild till text och bild till tabell 
- Konvertera foton till Word-dokument från lins 
- Åtgärden Överför filer 
- Anteckningarna förblir lokala och synkroniseras inte till servern
- Länk förhands granskning i QR
- Live persona kort i avsnittet mig
- Klassificering, märkning och skydd (CLP)


## <a name="download-the-office-app-for-ios"></a>Ladda ned Office-appen för iOS

- Ladda ner Office-appen för iPhone från [App Store](https://products.office.com/mobile/office?rtc=2). 

> [!NOTE]
> Office-appen för iOS är för närvarande endast tillgänglig för iPhone. Stöd för iPad läggs till senare. 

## <a name="security-considerations"></a>Säkerhets aspekter

Om din organisation skickar appar till mobila enheter i personalen föreslår vi att du ersätter programmen Word, Excel och PowerPoint med Office-appen för iOS.
