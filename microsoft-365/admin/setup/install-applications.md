# <a name="install-office-applications"></a>Installera Office-program

Nu när du har konfigurerat Microsoft 365 kan du konfigurera Office-programmen på din Mac, PC eller mobila enheter.
  
Följ dessa länkar för information om hur du:
  
- Installera Office-program: [Installera Office på din PC eller Mac](https://support.microsoft.com/office/4414eaaf-0478-48be-9c42-23adc4716658)

- Konfigurera mobila enheter: [Microsoft 365 mobilinstallation – hjälp](https://support.microsoft.com/office/7dabb6cb-0046-40b6-81fe-767e0b1f014f)

- Konfigurera e-post i Outlook: [Windows](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b) eller [Mac](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b#PickTab=Outlook_for_Mac)
 
- [Uppgradera användare till de senaste apparna](upgrade-users-to-latest-office-client.md) 

::: moniker range="o365-21vianet"

Om du har köpt Azure Active Directory Premium (AADP) Plan 1 eller Plan 2 är du berättigad till Microsoft Identity Manager (MIM). Om du vill hämta MIM går du till [Download Center](https://www.microsoft.com/zh-cn/download/details.aspx?id=58498).

::: moniker-end

Har du problem? De här resurserna för felsökning kan vara till hjälp:
  
- [Felsöka installation av Office och Microsoft 365](https://support.microsoft.com/office/35ff2def-e0b2-4dac-9784-4cf212c1f6c2)
