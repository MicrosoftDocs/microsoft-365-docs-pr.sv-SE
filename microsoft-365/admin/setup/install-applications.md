# <a name="install-office-applications"></a>Installera Office-program

Nu när du har konfigurerat Office 365 kan du konfigurera Office-programmen på din Mac, PC eller mobila enheter.
  
Följ dessa länkar för information om hur du:
  
- Installera Office-program: [Installera Office på din PC eller Mac](https://support.office.com/article/4414eaaf-0478-48be-9c42-23adc4716658.aspx)

- Konfigurera mobila enheter: [Konfigurera Office 365 - Hjälp](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f.aspx)

- Konfigurera e-post i Outlook: [Windows](https://support.office.com/article/6e27792a-9267-4aa4-8bb6-c84ef146101b.aspx) eller [Mac](https://support.office.com/article/6e27792a-9267-4aa4-8bb6-c84ef146101b.aspx#PickTab=Outlook_for_Mac)

::: moniker range="o365-21vianet"

Om du har köpt Azure Active Directory Premium (AADP) Plan 1 eller Plan 2 är du berättigad till Microsoft Identity Manager (MIM). Om du vill hämta MIM går du till [Download Center](https://www.microsoft.com/zh-cn/download/details.aspx?id=58498).

::: moniker-end

Har du problem? De här resurserna för felsökning kan vara till hjälp:
  
- [Felsöka installationen av Office och Office 365](https://support.office.com/article/35ff2def-e0b2-4dac-9784-4cf212c1f6c2.aspx)
