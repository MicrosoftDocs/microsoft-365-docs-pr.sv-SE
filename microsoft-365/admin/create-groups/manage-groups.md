# <a name="manage-an-office-365-group-in-the-microsoft-365-admin-center"></a>Hantera en Office 365-grupp i administrationscentret för Microsoft 365

När du har [skapat en Office 365-grupp](create-groups.md) och lagt till gruppmedlemmar kan du konfigurera gruppen. Du kan redigera gruppens namn eller beskrivning, hantera ägare eller medlemmar och ange om externa avsändare kan skicka e-post till gruppen och om du vill skicka kopior av gruppkonversationer till medlemmar. 

## <a name="edit-the-group-name-or-description"></a>Redigera gruppens namn eller beskrivning

::: moniker range="o365-worldwide"

1. Expandera **Grupper**i administrationscentret och klicka sedan på **Grupper**.

2. Markera den grupp som du vill redigera och klicka sedan på **Redigera namn och beskrivning**.

3. Uppdatera namnet och beskrivningen och välj sedan **Spara**.

::: moniker-end

::: moniker range="o365-germany"

1. Expandera **Grupper**i administrationscentret och klicka sedan på **Grupper**.

2. Markera den grupp som du vill redigera och klicka sedan på **Redigera namn och beskrivning**.

3. Uppdatera namnet och beskrivningen och välj sedan **Spara**.

::: moniker-end

::: moniker range="o365-21vianet"

1. Expandera **Grupper**i administrationscentret och klicka sedan på **Grupper**.

2. Markera den grupp som du vill redigera och klicka sedan på **Redigera namn och beskrivning**.

3. Uppdatera namnet och beskrivningen och välj sedan **Spara**.

::: moniker-end

## <a name="manage-group-owners-and-members"></a>Hantera gruppägare och medlemmar

::: moniker range="o365-worldwide"

1. Expandera **Grupper**i administrationscentret och klicka sedan på **Grupper**.

2. Klicka på namnet på den grupp som du vill lyckas öppna inställningsfönstret.

3. På fliken **Medlemmar** väljer du om du vill hantera ägare eller medlemmar.

4. Välj **Lägg till** om du vill lägga till någon eller klicka på **X** om du vill ta bort någon.

5. Klicka på **Stäng**.

::: moniker-end

## <a name="send-copies-of-conversations-to-group-members-inboxes"></a>Skicka kopior av konversationer till gruppmedlemmarnas inkorgar
  
När du använder administrationscentret för att skapa en grupp får användarna som standard inte kopior av gruppmeddelanden och mötesinbjudningar som skickas till deras inkorgar. De måste gå till gruppen för att se konversationer och möten. Du kan ändra den här inställningen i administrationscentret.

När du aktiverar den här inställningen får gruppmedlemmarna en kopia av gruppmeddelanden och mötesinbjudningar som skickas till deras Outlook-inkorg. De kan läsa och ta bort kopian utan att någon annan påverkas. En kopia av e-postmeddelandet finns kvar i gruppens inkorg.

Gruppmedlemmar kan välja bort att ta emot dessa e-postmeddelanden genom att välja att sluta följa gruppen i Outlook.

::: moniker range="o365-worldwide"

1. Expandera **Grupper**i administrationscentret och klicka sedan på **Grupper**.

2. Klicka på namnet på den grupp som du vill lyckas öppna inställningsfönstret.

3. På fliken **Inställningar** väljer du **Skicka kopior av gruppkonversationer och händelser till gruppmedlemmar** om du vill att medlemmar ska ta emot kopior av gruppmeddelanden och kalenderobjekt i sin egen inkorg.

4. Välj **Spara**.

::: moniker-end

::: moniker range="o365-germany"

1. Gå till sidan \> **Gruppergrupper** i administrationscentret och välj sedan den grupp som du vill ändra. **Groups**

2. Välj Redigera **bredvid Namn** **.**

3. Aktivera **Skicka kopior av gruppkonversationer och händelser till gruppmedlemmarnas inkorgar** till **På** om du vill att medlemmar ska ta emot kopior av gruppmeddelanden och kalenderobjekt i sin egen inkorg.

4. Välj **Spara**.

::: moniker-end

::: moniker range="o365-21vianet"

1. Gå till sidan \> **Gruppergrupper** i administrationscentret och välj sedan den grupp som du vill ändra. **Groups**

2. Välj Redigera **bredvid Namn** **.**

3. Aktivera **Skicka kopior av gruppkonversationer och händelser till gruppmedlemmarnas inkorgar** till **På** om du vill att medlemmar ska ta emot kopior av gruppmeddelanden och kalenderobjekt i sin egen inkorg.

4. Välj **Spara**.

::: moniker-end

## <a name="let-people-outside-the-organization-email-the-group"></a>Låta personer utanför organisationen skicka e-post till gruppen

Det här alternativet är bra om du vill ha ett företags e-postadress som info@contoso.com.
 
::: moniker range="o365-worldwide"

1. Expandera **Grupper**i administrationscentret och klicka sedan på **Grupper**.

2. Klicka på namnet på den grupp som du vill lyckas öppna inställningsfönstret.

3. I listan Över administrationscentergrupper väljer du namnet på den grupp som du vill ändra och väljer sedan **Tillåt externa avsändare att skicka den här gruppen till e-post**på fliken **Inställningar.**
    
4. Välj **Spara**.

::: moniker-end

::: moniker range="o365-germany"

1. Uppdatera administratörscentret så att den nya gruppen visas.
    
2. Markera den grupp som du vill ändra i listan Administrationscentergrupper och välj sedan **Redigera**bredvid **Namn**. 
    
3. Ställ in alternativet **Tillåt personer utanför organisationen att skicka e-post till gruppen** på **På**.
    
4. Välj **Spara**.

::: moniker-end

::: moniker range="o365-21vianet"

1. Uppdatera administratörscentret så att den nya gruppen visas.
    
2. Markera den grupp som du vill ändra i listan Administrationscentergrupper och välj sedan **Redigera**bredvid **Namn**. 
    
3. Ställ in alternativet **Tillåt personer utanför organisationen att skicka e-post till gruppen** på **På**.
    
4. Välj **Spara**.

::: moniker-end
