# <a name="options-for-accessing-email-from-your-mobile-device"></a>Alternativ för att komma åt e-post från din mobila enhet

Din organisation har konfigurerat inställningar som styr vilka mobilappar du kan använda för att komma åt ditt arbetskonto för e-post, kalender och kontakter.
  
Outlook för Android och Outlook för iOS-mobilappar är utformade för ditt arbete eller din skole-e-post, och är de enda apparna som tillhandahåller funktioner som [Fokuserad inkorg](https://support.office.com/article/f445ad7f-02f4-4294-a82e-71d8964e3978.aspx), som håller viktiga e-postmeddelanden i centrum, och [Schemaläggningsassistenten](https://go.microsoft.com/fwlink/?linkid=873406), vilket hjälper dig att hitta mötestider som arbetar med kollegor och helt enkelt hittar tillgängliga konferensrum.
  
## <a name="end-user-experience"></a>Slutanvändarupplevelse

 **Endast Outlook**
  
Din organisation kräver att du använder mobilapparna Outlook för Android eller Outlook för iOS för att komma åt företagets e-post, kalender och kontakter. Dina data börjar synkroniseras när du har hämtat och installerat Outlook för Android eller Outlook för iOS.
  
![Exempel på e-post för att använda Outlook för att synkronisera e-post](../../media/798d942a-4181-4dcb-8039-cd9f2edd9723.png)
  
Kolla [in Optimera Outlook-mobilappen för din iOS- eller Android-telefon](https://support.office.com/article/de075b19-b73c-4d8a-841b-459982c7e890.aspx) för fler Outlook-funktioner. Och gå till [Outlook för iOS och Android Hjälpcenter](https://support.office.com/article/cd84214e-a5ac-4e95-9ea3-e07f78d0cde6.aspx) om du stöter på några problem. 
  
 **Alla e-postappar**
  
Din organisation rekommenderar att du använder mobilappen Outlook för Android eller iOS för att komma åt ditt Office 365-arbets- eller skolkonto för e-post, kalender och kontakter. Om du inte vidtar några åtgärder inom de närmaste timmarna börjar din e-post automatiskt synkroniseras. Om du väljer att använda mobilappen Outlook för Android eller iOS synkroniseras dina data bara med mobilappen Outlook för Android eller Outlook för iOS. Om du väljer att fortsätta använda appen från tredje part börjar dina data synkroniseras direkt.
