# <a name="set-up-your-domain-host-specific-instructions"></a>Konfigurera din domän (tjänstspecifika instruktioner)

Om du vill börja använda en egen domän (contoso.com) med Microsoft 365 måste du verifiera domänen och konfigurera domänens DNS-poster. 
  
Du kan lägga till och hantera DNS-poster med de administrativa verktygen hos domänvärden, eller ge Microsoft kontroll över dina domänposter så konfigureras de åt dig.
  
Välj din domänvärd nedan för de exakta stegen. Om du inte är säker på vem din värd är kan du gå [till Hitta din domänregistrator.](find-your-domain-registrar.md)
