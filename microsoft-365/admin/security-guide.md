# <a name="security-info-graphic-for-small-businesses"></a>Säkerhetsinformationsgrafik för småföretag

**Sammanfattning:** Lär dig de grundläggande villkoren för de hot ditt företag står inför och vad du kan göra för att skydda dina data, konton och enheter. Som administratör för Office 365 för företag eller för Microsoft 365 Business vidtar du åtgärder för att göra ditt företag säkrare och hjälper alla anställda att lära sig hur du skyddar dina företagsdata och enheter..

Länkarna för PowerPoint och PDF nedan kan laddas ned och skrivas ut i tabloidformat (kallas även redovisning, 11 x 17 eller A3).
