# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Hämta den senaste versionen av Microsoft 365-användningsanalys

Mallappen kan uppdateras med nya data eller nya visualiseringar flera gånger per år. Din befintliga instans fortsätter att fungera, men om du vill hämta den senaste versionen måste en ny instans skapas och alla anpassningar måste tillämpas på den nya instansen. Se [Aktivera användningsanalys för Microsoft 365](enable-usage-analytics.md).
