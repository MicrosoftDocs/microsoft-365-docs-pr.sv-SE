# <a name="cortana-in-office-365"></a>Cortana i Office 365

Cortana är en molnbaserad digital assistent och Microsoft 365- och Office 365-tjänst som fungerar på dina enheter och Microsoft-tjänster. Cortana kan använda information som lagras i Microsoft 365 och Office 365 för att hjälpa personer i organisationen att hålla sig uppdaterade och få insikter, föreslagna uppgifter och hjälp med sina möten, dokument och kontakter.
  
## <a name="info-for-admins"></a>Information för administratörer

Efterlevnad är ett åtagande som Microsoft gör gentemot företagskunder. [Läs mer om Microsofts ramverk för efterlevnad.](https://go.microsoft.com/fwlink/p/?LinkId=2109173)

I enlighet med andra Microsoft 365- och Office 365-tjänster skyddas och skyddas kompatibla Cortana-funktioner med förbehåll för villkoren för onlinetjänster som innehåller en uppsättning löften som innebär skydd av användardata mot oavsiktlig förlust, ändring, obehörigt röjande eller åtkomst, eller olaglig förstörelse. Alla andra Cortana-funktioner (dvs. Cortana-anslutna tjänster) omfattas av [Microsoft Services-avtalet](https://go.microsoft.com/fwlink/p/?LinkId=2109174) och [Microsofts sekretesspolicy.](https://go.microsoft.com/fwlink/p/?LinkId=2109175)

Cortana-tjänster är uppdelade i två datakategorier, **kompatibla** och **valfria anslutna tjänster**, som du kan styra.

## <a name="turn-off-cortana-optional-connected-services"></a>Inaktivera Cortana-anslutna tjänster som tillval

Cortana-valfria anslutna tjänster kan stängas av för anställda i organisationen. Detta inaktiverar Cortanas valfria anslutna tjänster i Cortana i Windows och Cortana-mobilappen. Detta inkluderar Cortana-påminnelser, listor, uppgifter och andra funktioner. Tjänster i den kompatibla kategorin (d.v.s. Cortana OST-upplevelser), till exempel Cortanas Briefing-e-post och Spela upp mina e-postmeddelanden i Outlook mobile, förblir aktiva.

1. Välj **Inställningar inställningar** > **i** administrationscentret för Microsoft 365 och välj **Cortana**.

4. Markera kryssrutan för **Tillåt Cortana-valfria anslutna upplevelser att använda dina organisationers Microsoft-värddata** för att aktivera eller inaktivera Cortana-anslutna upplevelser.

5. Välj **Spara ändringar**.

## <a name="turn-off-cortana-ost-experiences"></a>Inaktivera Cortana OST-upplevelser

Organisationens anställda kan välja bort Cortana OST-upplevelser individuellt genom att följa stegen nedan.

1. Öppna Outlook Mobile.

2. Gå till **Inställningar**.
  
3. Välj **Spela upp min e-post**.

4. Flytta växlingen till av på de konton som du vill inaktivera.

### <a name="briefing-email"></a>Briefing e-post

Om du inaktiverar Cortana i Aktivitetsfältet inaktiveras inte Cortanas briefing-e-post. Anställda måste avsluta prenumerationen individuellt. Personer från din organisation kan välja bort Cortanas Briefing-e-post genom att välja **Avsluta prenumerationen** i meddelandets sidfot.
