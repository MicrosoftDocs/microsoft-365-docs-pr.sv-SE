# <a name="cortana-in-microsoft-365"></a>Cortana i Microsoft 365

Cortana, din personliga produktivitetsassistent, erbjuder AI-baserade upplevelser för att spara tid och fokusera på det som är viktigast. Cortana hjälper dina användare att öka sin personliga produktivitet under hela dagen, både på arbetet och i livet. När användare är inloggade med giltiga arbets- eller skolkonton kan de få molnbaserade hjälptjänster med Cortana i Microsoft 365-upplevelser som uppfyller Sekretess, säkerhet och efterlevnad på företagsnivå i Office 365 ("Cortana-företagstjänster"). 

Som personlig produktivitetsassistent har Cortana utformats för att tillhandahålla funktioner som på ett säkert och säkert sätt bearbetar och använder Data i Office 365, till exempel e-postmeddelanden, filer, chattar, för att spara tid, öka effektiviteten och förbättra användarnas produktivitet.

I framtiden fokuserar vi Cortana på produktivitet inom företaget.

- I enlighet med andra Office 365-tjänster uppfyller Cortanas företagstjänster samma sekretess,säkerhet och efterlevnad på företagsnivå som i villkoren för [onlinetjänster (OST).](https://www.microsoft.com/licensing/product-licensing/products)

- Nya Microsoft 365-upplevelser, till exempel genomgången i e-postmeddelandet och Spela upp mina e-postmeddelanden, kommer att aktiveras med hjälp av Cortanas företagstjänster och helt och hållet följa de här löftena. De här funktionerna är för närvarande tillgängliga i hela världen (standard för flera innehavare). Mer information om hur du hittar användningsplatsen finns i [Visa ytterligare egenskapsvärden för konton.](../../enterprise/view-user-accounts-with-microsoft-365-powershell.md?view=o365-worldwide#view-additional-property-values-for-accounts)

- Användare kan ansluta till Cortanas företagstjänster som beskrivs här via Cortana i Windows 10 (version 2004 och senare), samt klientprogram, som Outlook för iOS och Android, med separata licensvillkor. 

- Befintliga konsumentupplevelser, till exempel Cortana i Windows 10 (version 1909 och tidigare) och Cortana-appen på iOS och Android, styrs av [Microsofts](https://www.microsoft.com/licensing/product-licensing/products) tjänstavtal och [Microsofts](https://privacy.microsoft.com/privacystatement) sekretesspolicy (se avsnittet Befintliga tjänster för konsumenter nedan). De här villkoren reglerar även Cortanas företagstjänster som tillhandahålls till användaren när du loggar in med deras konsumentinloggningsuppgifter.

## <a name="what-data-is-processed-by-cortana-enterprise-services"></a>Vilka data bearbetas av Cortanas företagstjänster? 

Cortanas företagstjänster bearbetar frågor från användaren, Office-data som behövs för att uppfylla användarens begäran och andra telemetri som genereras av Microsoft-system för att köra tjänsten. I de data som samlas in av Cortanas företagstjänster ingår textrepresentationen av användarens talade frågor (t.ex. transkribering från taligenkänning). Den här texten är kunddata och hanteras i enlighet med villkoren för [onlinetjänster.](https://www.microsoft.com/licensing/product-licensing/products) Det används endast för att utveckla och förbättra maskininlärningsmodeller som är konsekventa med villkoren för onlinetjänster.

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-enterprise-services"></a>Vilken är styrningsmodellen för kunddata i Cortanas företagstjänster?

I enlighet med andra Office 365-tjänster skyddas Cortanas företagstjänster och omfattas av [villkoren för onlinetjänster.](https://www.microsoft.com/licensing/product-licensing/products) Detta omfattar en uppsättning utfästelser för skydd av kunddata mot oavsiktlig förlust, ändring, obehörig avslöjande eller åtkomst eller olagliga intrång. Kunddata har också strikta åtkomstbegränsningar. Microsoft använder bara kunddata för att tillhandahålla de tjänster som överenskommits och för ändamål som är kompatibla med dessa tjänster. Se tabellen nedan för mer information.

## <a name="cortana-enterprise-services-in-microsoft-365-experiences"></a>Cortanas företagstjänster i Microsoft 365-upplevelser

### <a name="cortana-in-windows-10-version-2004-and-later"></a>Cortana i Windows 10 (version 2004 och senare)

Med Cortana-appen i Windows 10 kan användarna snabbt få information i Microsoft 365, använda typer och talade frågor för att kommunicera med andra, kontrollera kalendrar, ställa in påminnelser och mycket mer.

Cortana i Windows kan hjälpa användarna att hantera sina scheman och uppgifter på ett bättre sätt. De kan lägga till listor i Microsoft To Do, slå upp lokal information, få definitioner och hålla reda på de senaste nyheterna, vädret och ekonomiinformationen med Bing-sökning.

Cortana i Windows 10, version 2004 och senare uppfyller samma sekretess-, säkerhets- och efterlevnadsvillkor på företagsnivå för Cortanas företagstjänster, som återspeglas i villkoren för [onlinetjänster (OST).](https://www.microsoft.com/licensing/product-licensing/products)

### <a name="how-to-opt-out-of-cortana-in-windows-10"></a>Så här avanmäler du dig från Cortana i Windows 10

Administratörer kan konfigurera Cortana i Windows 10 för organisationen med hjälp av Experience\AllowCortana MDM-principen eller via Grupprincip: Datorkonfiguration\Administrativa mallar\Windows-komponenter\Sök\Tillåt Cortana.

Från och med Windows 10, version 2004, är Cortana en UWP-app (Universell Windows-plattform) förinstallerad med Windows och uppdateras regelbundet via Microsoft Store. Om du vill ta emot de senaste uppdateringarna av Cortana måste [du aktivera uppdateringar via Microsoft Store.](/windows/configuration/stop-employees-from-using-microsoft-store)

[Läs mer om Cortana i Windows 10](/windows/configuration/cortana-at-work/cortana-at-work-overview)

### <a name="cortana-voice-assistance-in-teams"></a>Cortana rösthjälp i Teams

Cortana-rösthjälp i Teams-mobilappen och på microsoft Teams visningsenheter gör att Microsoft 365 Enterprise-användare kan effektivisera kommunikation, samarbete och mötesrelaterade uppgifter med talat naturligt språk. Användare kan prata med Cortana genom att välja mikrofonknappen längst upp till höger i Teams-mobilappen eller genom att säga &#8220;Cortana&#8221; i Microsoft Teams-skärmen. För att snabbt komma i kontakt med sitt team handsfree och när de är på språng kan användare säga frågor som &#8220;-samtal till Abby&#8221; eller &#8220;skicka ett meddelande till mitt nästa&#8221;. Användare kan också ansluta till möten genom att säga &#8220;ansluta till mitt nästa&#8221; och använda rösthjälp för att dela filer, kontrollera deras kalender med mera. De här rösthjälpsupplevelserna levereras med Cortana-tjänster i företagsklass som helt uppfyller Sekretess, säkerhet och efterlevnad för Office 365 och som fortfarande återspeglas i villkoren för [onlinetjänster (OST).](https://www.microsoft.com/licensing/product-licensing/products)

**Administratörskontroll**

Cortana-rösthjälp aktiveras som standard för klientorganisationen. Administratörer kan styra vem i klientorganisationen som kan använda Cortanas rösthjälp i Teams via en policy (TeamsCortanaPolicy). Den här principen kan ställas in på antingen användarkontonivå eller klientnivå. Administratörer kan också använda fältet CortanaVoiceInvocationMode i den här principkontrollen för att avgöra om Cortana är inaktiverad, aktiverad med endast push-knappanrop eller aktiverat med aktivera ordanrop (gäller även enheter som stöder det, som Microsoft Teams-skärmen). Observera att i tiden för den första versionen av Microsoft 365 Enterprise-användare i USA på engelska kommer Teams-mobilappen inte att stödja aktivering av Word, men kommer att stödjas i framtiden.

**Användarkontroll**

Enskilda användare kan prova Cortana rösthjälp i Teams-mobilappen genom att klicka på mikrofonknappen. De kan prova Cortana rösthjälp på visningsenheter i Microsoft Teams genom att säga &#8220;Cortana.&#8221; De kan också styra om Cortana i Teams är aktiverat för deras enhet via en inställning i Teams-mobilappen eller på Microsoft Teams-skärmen: 

1. Öppna Teams mobilapp eller gå till den omgivande (hem)skärmen i Microsoft Teams-skärmen.

2. Gå till Inställningar i **Teams-mobilappen.** På skärmen Microsoft Teams väljer du användarens avatar och sedan Inställningar. Om Cortana till exempel är &#8220;Cortana går du till Inställningar.&#8221;

3. Välj **Cortana.**

4. Flytta reglaget till **På** **eller Av** beroende på om du vill ha Cortana rösthjälp på enheten.

[Läs mer om Cortana rösthjälp i Teams](/microsoftteams/cortana-in-teams)

### <a name="conversational-ai-in-outlook-for-ios-with-cortana"></a>Konversations-AI i Outlook för iOS med Cortana

I Outlook för iOS gör Cortanas röstdrivna ai-upplevelse för konversationer att användare kan be sin produktivitetsassistent att schemalägga möten, skriva e-postmeddelanden, hantera sin kalender och inkorg och hitta all slags information.

Med hjälp av naturligt språk, taligenkänning, maskininlärning och lingvistiska talspråk baserade på Microsoft AI-teknik lär sig Cortana ditt sammanhang för att hjälpa dig att hålla ordning, hålla kontakten med personer och saker som är viktiga för dig och ta kontroll över din dag.

Använd bara mikrofonknappen för att be Cortana att lägga till mötesmottagare eller ändra datum, tider eller platser. Du kan också be Cortana att hitta specifika händelser. Slutligen kan du be Cortana att skriva snabba e-postmeddelanden, vidarebefordra meddelanden eller svara på trådar. Cortanas mikrofon kan också hjälpa dig att starta Spela upp mina e-postmeddelanden i Outlook för iOS, så att du kan lyssna på din inkorg handsfree.

Inledningsvis är den här nya konversations ai-funktionen med Cortana tillgänglig på engelska för kunder i USA som använder Outlook för iOS med ett Microsoft 365-arbetskonto. Mer information finns i Starta [en konversation med din personliga produktivitetsassistent i Outlook med Cortana.](https://techcommunity.microsoft.com/t5/outlook-blog/start-a-conversation-with-your-personal-productivity-assistant/ba-p/2071416#:~:text=Conversational%20AI%20allows%20you%20to,time%2C%20all%20with%20your%20voice)

### <a name="play-my-emails"></a>Spela upp mina e-postmeddelanden

Spela upp mina e-postmeddelanden (som du ansluter till via Outlook Mobile) är en röststyrd, handsfree-upplevelse där användare kan lyssna på nya meddelanden i den fokuserade inkorgen och ändra till sin dag via högtalarna på sin telefon, hörlurar eller anslutna ljudenhet. Användare kan be Cortana att läsa upp sina senaste e-postmeddelanden och be Cortana att vidta åtgärder som flagga, arkivera, ta bort och hoppa över meddelanden. Den här funktionen är särskilt användbar om du behöver komma i god tid till din e-post när du kör, multitasking eller när du är på resa. När användaren pratar med Cortana i Spela upp mina e-postmeddelanden skickas röstsamtalsförfrågan direkt till Cortanas företagstjänster. En text till tal-läsning av användarens e-post bearbetas i Office 365-molnet. Under den här processen bearbetas inga Office 365-data på användarens mobila enhet och inga e-postdata sparas. En avskrift av talade kommandon (t.ex. "markera som läst", "nästa", "flagga" osv.) kan behållas i enlighet med dataskyddsvillkoren i Villkoren för Microsoft [Online Services.](https://www.microsoft.com/licensing/product-licensing/products)

Cortana kommer att anropa när ett e-postmeddelande är skyddat och pausa kort innan du läser meddelandet för att ge användarna tillräckligt med tid för att pausa uppspelningen eller hoppa till nästa meddelande. I likhet med ett privat telefonsamtal bör användarna vara försiktig när de initierar uppspelningen på platser där konfidentiell information eventuellt kan repeteras. I dessa fall rekommenderar vi att organisationens anställda har hörlurar i lämpliga miljöer när de använder Spela upp mina e-postmeddelanden i Outlook Mobile.


### <a name="opt-out-of-play-my-emails"></a>Avanmäl mig från Spela upp mina e-postmeddelanden

Enskilda användare kan avanmäla sig från Spela upp mina e-postmeddelanden genom att följa anvisningarna nedan.

1. Öppna Outlook Mobile.

2. Gå till **Inställningar.**
  
3. Välj **Spela upp mina e-postmeddelanden**.

4. Flytta reglaget till inaktivera för de konton som du vill inaktivera.

[Läs mer om Spela upp mina e-postmeddelanden](https://support.microsoft.com/help/4558256)

### <a name="briefing-email"></a>Genomgång av e-post

Cortana skickar en personlig genomgång via e-post med uppgifter och  åtaganden som du har gjort med ett bekvämt sätt att markera dem som klara eller schemalägga fokustid för att få dem gjorda. Den innehåller även en sammanfattning av dina möten och relevanta dokument för dagen. Cortana extraherar information från en användares e-postmeddelanden och lagrar den i sin Exchange Online-postlåda tills den konsolideras i genomgångsmeddelandet. På nolltid är personlig information tillgänglig utanför din Exchange Online-postlåda.

### <a name="how-to-opt-out-of-briefing-email"></a>Så här avanmäler du dig från genomgångsmeddelande

Administratörer kan konfigurera genomgång för organisationen med hjälp [av PowerShell](/briefing/be-admin) i Exchange Online. Enskilda personer kan avanmäla sig från Cortanas e-post för genomgång **genom att välja Avbryt** prenumeration i sidfoten i meddelandet.
