# <a name="cortana-in-microsoft-365"></a>Cortana i Microsoft 365

Cortana, din personliga produktivitetsassistent, erbjuder AI-drivna upplevelser för att spara tid och fokusera uppmärksamheten på det som är viktigast. Cortana hjälper användarna att öka sin personliga produktivitet under hela dagen i både arbete och liv. När användarna är inloggade med giltiga arbets- eller skolkonton kan de få molnbaserade Cortana-hjälptjänster inom Microsoft 365-produktivitetsverktygen. Cortana i Microsoft 365 uppfyller sekretess-, säkerhets- och efterlevnadslöften på Microsoft 365-nivå.

Som en personlig produktivitetsassistent är Cortana utformad för att leverera funktioner som på ett säkert och säkert sätt bearbetar och resonerar över Microsoft 365-data som e-post, filer, chattar etc., för att spara tid, öka effektiviteten och öka användarnas produktivitet.

Framöver fokuserar vi Cortana på företagets produktivitet.

- I enlighet med andra Microsoft 365-tjänster uppfyller Cortana i Microsoft 365 samma sekretess-, säkerhets- och efterlevnadslöften på företagsnivå som återspeglas i [villkoren för onlinetjänster (OST).](https://www.microsoft.com/en-us/licensing/product-licensing/products)

- Nya molnbaserade Cortana-hjälptjänster, till exempel informationsmeddelandet och Spela upp mina e-postmeddelanden, aktiveras med Cortana i Microsoft 365 och uppfyller dessa löften fullt ut. Dessa funktioner är för närvarande tillgängliga på den engelska (USA) marknaden för användare med brevlådor i den amerikanska geo, och för anställda i multinationella företag med USA-baserade brevlådor.

- Användare kan ansluta till Cortana i Microsoft 365-tjänster som beskrivs här via klientprogram, till exempel Outlook Mobile, med förbehåll för separata licensvillkor.

- Befintliga konsumentupplevelser, inklusive Cortana i Windows 10 (version 1909 och tidigare) och Cortana-appen på iOS och Android, styrs av [Microsoft Services Agreement](https://www.microsoft.com/en-us/licensing/product-licensing/products) och [Microsofts sekretesspolicy](https://privacy.microsoft.com/en-us/privacystatement). Dessa villkor kommer också att styra Cortana i Microsoft 365-tjänster som tillhandahålls användaren när de är inloggade med sina konsumentuppgifter.

## <a name="what-data-is-processed-by-cortana-in-microsoft-365"></a>Vilka data bearbetas av Cortana i Microsoft 365?

Cortana i Microsoft 365 bearbetar frågor från användaren, Office-data som behövs för att uppfylla användarens begäran och annan telemetri som genereras av Microsoft-system för att köra tjänsten. De data som samlas in av Cortana i Microsoft 365 innehåller textrepresentationen av användarens talade frågor (dvs. transkriptioner från taligenkänning). Dessa textdata är kunddata och hanteras i enlighet med villkoren för [onlinetjänster](https://www.microsoft.com/en-us/licensing/product-licensing/products). Det används endast för att utveckla och förbättra maskininlärningsmodeller som överensstämmer med villkoren för [onlinetjänster](https://www.microsoft.com/en-us/licensing/product-licensing/products).

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-in-microsoft-365"></a>Vad är styrningsmodellen för kunddata i Cortana i Microsoft 365?

I enlighet med andra Microsoft 365- och Microsoft 365-tjänster är Cortana i Microsoft 365 skyddad och omfattas av villkoren för [onlinetjänster](https://www.microsoft.com/en-us/licensing/product-licensing/products). Detta inkluderar en uppsättning löften om skydd av kunddata mot oavsiktlig förlust, ändring, obehörigt avslöjande eller åtkomst, eller olaglig förstörelse. Kunddata är också föremål för strikta åtkomstbegränsningar. Microsoft använder endast kunddata för att tillhandahålla de överenskomna tjänsterna och för ändamål som är kompatibla med dessa tjänster. Se tabellen nedan för mer information.

## <a name="cortana-in-microsoft-365-features"></a>Cortana i Microsoft 365-funktioner

### <a name="play-my-emails"></a>Spela upp mina e-postmeddelanden

Spela min e-post (som ansluten till via Outlook mobile) är en röstdriven, handsfree-upplevelse för användare att lyssna på sin inkorg via högtalarna på sin telefon, hörlurar eller ansluten ljudenhet. Användare kan be Cortana att läsa sina senaste e-postmeddelanden högt och be Cortana att vidta åtgärder som flagga, arkivera, ta bort och hoppa över meddelanden. Den här funktionen är särskilt användbar för att komma ikapp din e-post när du pendlar eller är på språng. När användaren pratar med Cortana i Spela upp mina e-postmeddelanden går begäran om talljud direkt till Cortana i Microsoft 365. En text till talavläsning av användarens e-post bearbetas i Microsoft 365-molnet. Under den här processen bearbetas inga Microsoft 365-data på användarens mobila enhet och inga nya e-postdata sparas. En utskrift av talade kommandon (d.v.s. "markera som läst", "nästa", "flagga" etc.) kan behållas i enlighet med dataskyddsvillkoren i [villkoren för onlinetjänster](https://www.microsoft.com/en-us/licensing/product-licensing/products).

Cortana ropar när ett e-postmeddelande är skyddat och pausar kort innan du läser meddelandet för att ge användarna tillräckligt med tid för att pausa uppspelningen eller hoppa till nästa meddelande. I likhet med ett privat telefonsamtal bör användarna vara försiktiga när de startar uppspelning på platser där konfidentiell information eventuellt kan höras. I dessa fall rekommenderas att anställda i organisationen använder hörlurar i lämpliga miljöer när de använder Spela upp mina e-postmeddelanden i Outlook-mobilen.

### <a name="opt-out-of-play-my-emails"></a>Välj bort spela upp mina e-postmeddelanden

Enskilda personer kan välja bort Spela upp mina e-postmeddelanden med hjälp av följande steg.

1. Öppna Outlook-mobilen.

2. Gå till **Inställningar**.
  
3. Välj **Spela upp mina e-postmeddelanden**.

4. Flytta växlingsknappen till av på de konton som du vill inaktivera.

### <a name="briefing-email"></a>Informationsmeddelande

Cortana skickar ett personligt informationsmeddelande med uppgifter och åtaganden som du gjort med ett bekvämt sätt att markera dem som **klara** eller schemalägga fokustiden för att få dem gjorda. Den innehåller också en sammanfattning av dina möten och relevanta dokument för din dag. Cortana extraherar information från en användares Microsoft 365-meddelanden och lagrar den i sin Microsoft 365-postlåda tills den konsolideras i Briefing-e-post. Personuppgifter är inte vid något tillfälle tillgängliga utanför din Microsoft 365-postlåda.

### <a name="how-to-opt-out-of-briefing-email"></a>Så här väljer du bort informationsmeddelande

Administratörer kan konfigurera Briefing för sin organisation med hjälp av en [växlingsknapp](https://docs.microsoft.com/briefing/be-admin) i administrationscentret för Microsoft 365. Enskilda personer kan välja bort Cortanas briefing-e-post genom att välja Avsluta prenumeration i sidfoten på meddelandet.

## <a name="how-is-cortana-in-microsoft-365-different-from-other-cortana-features-i-may-have-previously-experienced"></a>Hur skiljer sig Cortana i Microsoft 365 från andra Cortana-funktioner som jag tidigare har upplevt?

Här är de två sätten att tänka på hur Cortana fungerar i ditt företag:

**Nya upplevelser för organisationer med Cortana i Microsoft 365**: Cortana i Microsoft 365-tjänster är utformade för att uppfylla säkerhets- och efterlevnadsbehoven för organisationer.

1. Detta är en ny tjänst och diskuteras här i detta dokument.

2. För tjänster som omfattas av villkoren för onlinetjänster är Microsoft en databehandlare: Microsoft samlar in och använder kunddata från kunder endast för att tillhandahålla de onlinetjänster som våra kunder begär och för de ändamål som våra kunder instruerar. Enligt EU:s allmänna dataskyddsförordning (GDPR) är kunden registeransvarig för sina uppgifter. Se villkoren för [onlinetjänster](https://www.microsoft.com/en-us/licensing/product-licensing/products) och [införa mer sekretesstransparens för våra kommersiella molnkunder.](https://blogs.microsoft.com/eupolicy/2019/11/18/introducing-privacy-transparency-commercial-cloud-customers/)

3. Som ett exempel är Spela upp mina e-postmeddelanden en Cortana-tjänst som användarna kan ansluta till via Outlook för iOS och använder Cortana i Microsoft 365.

4. Administratörer har alltid kontroller för valfria anslutna upplevelser för Cortana, liknande valfria anslutna upplevelser när du använder Office ProPlus-program.

**Befintliga tjänster för konsumenter**: Cortana-valfria anslutna tjänster är främst utformade för konsumentupplevelser och levereras för närvarande i Windows 10 (version 1909 och tidigare) och Cortana-appen på iOS och Android.

1. De här upplevelserna möjliggör funktioner som väder, nyheter och trafik.

2. Klientadministratörer kan styra om Cortana i Windows 10 (version 1909 och tidigare) och Cortana-appen på iOS och Android kan tillåta Cortana att ansluta till Microsoft 365-klientdata.

Inaktivera Cortana-åtkomst till organisationens Microsoft-värddata

1. I administrationscentret för Microsoft 365 väljer du **Inställningar** > **tjänster** och väljer **Cortana**.

2. Avmarkera kryssrutan Tillåt **att dessa Cortana-valfria anslutna upplevelser använder organisationens Microsoft-värddata** för att inaktivera Cortana-anslutna upplevelser.

3. Välj **Spara ändringar**.

För tjänster som regleras av [Microsoft Services Agreement](https://go.microsoft.com/fwlink/p/?LinkId=2109174) och [Microsofts sekretesspolicy](https://privacy.microsoft.com/en-us/privacystatement)är Microsoft personuppgiftsansvarig. Som personuppgiftsansvarig använder Microsoft data för att förbättra produkter och tjänster i enlighet med [Microsofts sekretesspolicy](https://privacy.microsoft.com/en-us/privacystatement).
