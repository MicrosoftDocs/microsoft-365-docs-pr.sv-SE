# <a name="cortana-in-microsoft-365"></a>Cortana i Microsoft 365

Cortana, din personliga produktivitetsassistent, erbjuder AI-baserade upplevelser för att spara tid och fokusera på det som är viktigast. Cortana har utformats för att ge funktioner som på ett säkert sätt bearbetar och skäl över Office 365-data som e-post, filer, chattar osv. för att spara tid, öka effektiviteten och förbättra användarnas produktivitet.

När användare är inloggade med giltiga arbets- eller skolkonton kan de få molnbaserade hjälptjänster med Cortana i Microsoft 365-upplevelser som uppfyller Office 365:s sekretess, säkerhet och efterlevnad på företagsnivå **("Cortana företagstjänster")** på företagsnivå. 


- **Cortana företagstjänster** omfattar Cortana i Windows 10 (version 2004 och senare), Outlook för iOS och Android, Microsoft Teams-mobilappar för iOS och Android [Microsoft Teams visas.](/microsoftteams/devices/teams-displays)

- Dessa olika funktioner omfattas av separata licensvillkor och har separata steg för avanmälning som beskrivs nedan.

- I enlighet med andra Office 365 tjänster uppfyller Cortana företagstjänster samma sekretess,säkerhet och efterlevnad på företagsnivå som i villkoren för [onlinetjänster (OST).](https://www.microsoft.com/licensing/product-licensing/products)

- Nya Microsoft 365 funktioner, till exempel genomgången i e-postmeddelanden och Spela upp mina e-postmeddelanden, kommer att aktiveras med Cortana företagstjänster och helt följa de här löftena. De här funktionerna är för närvarande tillgängliga i hela världen (standard för flera innehavare). Mer information om hur du hittar användningsplatsen finns i [Visa ytterligare egenskapsvärden för konton.](../../enterprise/view-user-accounts-with-microsoft-365-powershell.md#view-additional-property-values-for-accounts)

- Befintliga konsumentupplevelser, t.ex. Cortana i Windows 10 (version 1909 och tidigare), regleras av [Microsofts](https://www.microsoft.com/licensing/product-licensing/products) tjänstavtal och [Microsofts](https://privacy.microsoft.com/privacystatement) sekretesspolicy (se avsnittet Befintliga tjänster för konsumenter nedan). Dessa villkor gäller även för Cortana företagstjänster som tillhandahålls till användaren när de är inloggade med sina konsumentinloggningsuppgifter.

## <a name="what-data-is-processed-by-cortana-enterprise-services"></a>Vilka data bearbetas av Cortana företagstjänster? 

Cortana företagstjänster bearbeta frågor från användaren Office data som behövs för att uppfylla användarens begäran och andra telemetri som skapats av Microsoft-system för att köra tjänsten. I de data Cortana företagstjänster ingår textrepresentation av användarens talade frågor (t.ex. transkribering från taligenkänning). Den här texten är kunddata och hanteras i enlighet med villkoren för [onlinetjänster.](https://www.microsoft.com/licensing/product-licensing/products) Det används endast för att utveckla och förbättra maskininlärningsmodeller som är konsekventa med villkoren för onlinetjänster.

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-enterprise-services"></a>Vilken är styrningsmodellen för kunddata i Cortana företagstjänster?

I enlighet med Office 365 tjänster skyddas Cortana företagstjänster och omfattas av [villkoren för onlinetjänster.](https://www.microsoft.com/licensing/product-licensing/products) Detta omfattar en uppsättning utfästelser för skydd av kunddata mot oavsiktlig förlust, ändring, obehörig avslöjande eller åtkomst eller olagliga intrång. Kunddata har också strikta åtkomstbegränsningar. Microsoft använder bara kunddata för att tillhandahålla de tjänster som överenskommits och för ändamål som är kompatibla med dessa tjänster. Se tabellen nedan för mer information.

## <a name="cortana-enterprise-services-in-microsoft-365-experiences"></a>Cortana företagstjänster i Microsoft 365 upplevelse

### <a name="cortana-in-windows-10-version-2004-and-later"></a>Cortana i Windows 10 (version 2004 och senare)

Med Cortana-appen Windows 10 kan användarna snabbt få information i Microsoft 365, använda typer och talade frågor för att kommunicera med andra, kontrollera kalendrar, ställa in påminnelser och mycket mer.

Cortana i Windows hjälpa användarna att hantera sina scheman och aktiviteter bättre. De kan lägga till i sina listor i Microsoft To Do, slå upp lokal information, få definitioner och hålla reda på de senaste nyheterna, väder- och ekonomiinformationen med hjälp av Bing sökning.

Cortana i Windows 10, version 2004 och senare, uppfyller samma sekretess-, säkerhets- och efterlevnadsvillkor på företagsnivå för Cortana-företagstjänster, som återspeglas i villkoren för [onlinetjänster (OST).](https://www.microsoft.com/licensing/product-licensing/products)

### <a name="how-to-opt-out-of-cortana-in-windows-10"></a>Hur du avanmäler Cortana i Windows 10

Att avanmäla Cortana i Windows 10 avanmäler inte andra funktioner som styrs separat. Administratörer kan konfigurera Cortana i Windows 10 för organisationen med hjälp av principen Experience\AllowCortana MDM eller via Grupprincip: Datorkonfiguration\Administrativa mallar\Windows Components\Search\Tillåt Cortana.

Från och med Windows 10, version 2004, är Cortana en UWP-app (Universell Windows-plattform) förinstallerad med Windows och uppdateras regelbundet Microsoft Store. För att få de senaste Cortana måste du [aktivera uppdateringar via Microsoft Store](/windows/configuration/stop-employees-from-using-microsoft-store).

[Läs mer om Cortana i Windows 10](/windows/configuration/cortana-at-work/cortana-at-work-overview)

För att snabbt komma i kontakt med sitt team handsfree och när de är på språng kan användare säga frågor som &#8220;-samtal till Abby&#8221; eller &#8220;skicka ett meddelande till mitt nästa&#8221;. Användare kan också ansluta till möten genom att säga &#8220;ansluta till mitt nästa&#8221; och använda rösthjälp för att dela filer, kontrollera deras kalender med mera. De här rösthjälpsupplevelserna levereras med Cortana företagstjänster som helt uppfyller Office 365:s sekretess, säkerhet och efterlevnad vad som utspelas i [OST (Online Services Terms).](https://www.microsoft.com/licensing/product-licensing/products)

### <a name="conversational-ai-in-outlook-for-ios-with-cortana"></a>Konversations-AI i Outlook för iOS med Cortana

I Outlook för iOS gör den röstdrivna konversations-AI-upplevelsen i Cortana att användare kan be sin produktivitetsassistent att schemalägga möten, skriva e-postmeddelanden, hantera sin kalender och inkorg och hitta all slags information.

Med hjälp av naturligt språk, taligenkänning, maskininlärning och lingvistiska talspråk baserade på Microsoft AI-teknik lär Cortana känna ditt sammanhang för att hjälpa dig att hålla ordning, hålla kontakten med personer och saker som är viktiga för dig och ta kontroll över din dag.

Använd bara mikrofonknappen för att be Cortana att lägga till mötesmottagare eller ändra datum, tider eller platser. Du kan också be Cortana att hitta specifika händelser. Slutligen kan du be Cortana att skriva snabba e-postmeddelanden, vidarebefordra meddelanden eller svara på trådar. Cortana mikrofon kan också hjälpa dig att starta Spela upp mina e-postmeddelanden i Outlook för iOS, så att du kan lyssna på din inkorg handsfree.

Till en början är den här nya konversationsära AI-funktionen med Cortana tillgänglig på engelska för kunder i USA som använder Outlook för iOS med ett Microsoft 365 arbetskonto. Mer information finns i Starta [en konversation med din personliga produktivitetsassistent i Outlook med Cortana](https://techcommunity.microsoft.com/t5/outlook-blog/start-a-conversation-with-your-personal-productivity-assistant/ba-p/2071416#:~:text=Conversational%20AI%20allows%20you%20to,time%2C%20all%20with%20your%20voice).

### <a name="play-my-emails"></a>Spela upp mina e-postmeddelanden

Spela upp mina e-postmeddelanden (som anslutna till via Outlook mobile) är en röststyrd, handsfree-upplevelse där användare kan lyssna på nya meddelanden i sin fokuserade inkorg och ändra sina dagar via högtalarna på sin telefon, hörlurar eller anslutna ljudenhet. Användare kan be Cortana att läsa upp sina senaste e-postmeddelanden och Cortana att vidta åtgärder som flagga, arkivera, ta bort och hoppa över meddelanden. Den här funktionen är särskilt användbar om du behöver komma i god tid till din e-post när du kör, multitasking eller när du är på resa. När användaren pratar för Cortana i Spela upp mina e-postmeddelanden, skickas talljudbegäran direkt till Cortana företagstjänster. En text till tal-läsning av användarens e-post bearbetas i Office 365 molnet. Under den här processen behandlas Office 365 data på användarens mobila enhet och inga e-postdata sparas. En avskrift av talade kommandon (t.ex. "markera som läst", "nästa", "flagga" osv.) kan behållas i enlighet med dataskyddsvillkoren i Villkoren för Microsoft [Online Services.](https://www.microsoft.com/licensing/product-licensing/products)

Cortana visas när ett e-postmeddelande är skyddat och tillfälligt pausas innan meddelandet läss, så att användarna får tillräckligt med tid att pausa uppspelningen eller hoppa till nästa meddelande. I likhet med ett privat telefonsamtal bör användarna vara försiktig när de initierar uppspelningen på platser där konfidentiell information eventuellt kan repeteras. I de här fallen rekommenderar vi att organisationens anställda har hörlurar i lämpliga miljöer när de använder Spela upp mina e-postmeddelanden i Outlook mobil.


### <a name="how-to-opt-out-of-play-my-emails"></a>Så här avanmäler du dig från Spela upp mina e-postmeddelanden

Enskilda användare kan avanmäla sig från Spela upp mina e-postmeddelanden genom att följa anvisningarna nedan.

1. Öppna Outlook mobil.

2. Gå till **Inställningar**.
  
3. Välj **Spela upp mina e-postmeddelanden**.

4. Flytta reglaget till inaktivera för de konton som du vill inaktivera.

[Läs mer om Spela upp mina e-postmeddelanden](https://support.microsoft.com/help/4558256)

### <a name="briefing-email"></a>Genomgång av e-post

Cortana skickar en personlig genomgång via e-post med uppgifter och åtaganden  som du har gjort med ett bekvämt sätt att markera dem som klara eller schemalägga fokustid för att få dem gjorda. Den innehåller även en sammanfattning av dina möten och relevanta dokument för dagen. Cortana extraherar information från en användares e-postmeddelanden och lagrar den i sin Exchange Online-postlåda tills den konsolideras i genomgångsmeddelandet. På nolltid är personlig information tillgänglig utanför Exchange Online postlåda. Användarna får åtkomst till genomgångsmeddelandet endast om de har licenser som Exchange Online tjänsteabonnemanget.

### <a name="how-to-opt-out-of-briefing-email"></a>Så här avanmäler du dig från genomgångsmeddelande

Administratörer kan konfigurera genomgång för organisationen med [Hjälp av PowerShell](/briefing/be-admin) i Exchange Online. Enskilda personer kan avanmäla sig Cortana av e-post om att avsluta **prenumerationen** i meddelandets sidfot.
