# <a name="update-your-mx-records-to-transition-to-the-global-exchange-online-service"></a>Uppdatera dina MX-poster till över gången till den globala Exchange Online-tjänsten

1. Logga in på [administrations portalen för Microsoft 365](https://admin.microsoft.com)och gå till **inställnings**  >  **domäner**

2. Statusen visas till höger för varje domän. Om organisationens domäner pekar på Microsoft Cloud Germany Exchange Online måste du uppdatera din MX-post.

3. Klicka på domänen och klicka sedan på **DNS-fel som upptäckts, klicka här för att Visa**.

4. På den här sidan finns instruktioner för hur du kan åtgärda MX-posten. Om domänens registrator använder [Domain Connect](../setup/add-domain.md#registrars-with-domain-connect)kan du klicka på "åtgärda mina poster" överst. I annat fall kan du följa länken i guiden till **steg-för-steg-instruktioner** för din registrator.
