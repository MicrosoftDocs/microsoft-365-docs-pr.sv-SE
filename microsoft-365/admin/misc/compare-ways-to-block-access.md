# <a name="compare-ways-to-block-access"></a>Jämföra sätt att blockera åtkomst

När en medarbetare lämnar organisationen, på gott villkor eller dåligt, måste du blockera deras åtkomst till Microsoft 365. Det kan du göra på några olika sätt.
  
||||
|:-----|:-----|:-----|
|**Olika sätt att blockera åtkomst** <br/> |**Definition** <br/> |**Bästa metod** <br/> |
|Blockera inloggning  <br/> |Ett sätt att blockera en användare från att komma åt Microsoft 365 är att ändra sin inloggningsstatus till **Inloggning blockerad**. Detta förhindrar att de loggar in på Microsoft 365 från sina datorer och mobila enheter, även om de fortfarande kan visa tidigare nedladdad eller synkroniserad e-post och dokument. Om du använder Blackberry Enterprise Service kan du inaktivera deras åtkomst där också.  <br/> |Använd den här metoden när en anställd planerar att lämna organisationen eller planerar att ha en lång ledighet.  <br/> |
|Återställa användarlösenord  <br/> |Ett annat sätt att förhindra att en användare kommer åt Microsoft 365 är att återställa sitt lösenord. Det gör att de inte kan använda sitt konto, men de kan fortfarande visa tidigare nedladdad eller synkroniserad e-post och dokument. Du kan sedan logga in som dem och ändra lösenord till ett eget.  <br/> |Använd den här metoden när en anställd slutar plötsligt och permanent, och du anser att det finns anledning att oroa sig för affärsdata.  <br/> |
|Ta bort alla tilldelade licenser  <br/> |Ett annat alternativ är att ta bort alla Microsoft 365-licenser som tilldelats användaren. Detta förhindrar att de använder program och tjänster som Office-paketet, Office-program för webben, Yammer och SharePoint Online. De kan fortfarande logga in, men kan inte använda de här tjänsterna.  <br/> |Använd när du känner att den här användaren inte längre behöver åtkomst till specifika funktioner i Microsoft 365.  <br/> <br> **Viktigt:** När du tar bort en licens tas användarens postlåda bort inom 30 dagar.
