# <a name="compare-ways-to-block-access"></a>Jämföra olika sätt att blockera åtkomst

När en anställd lämnar organisationen, på egen villkor eller får hjälp, måste du blockera deras åtkomst till Microsoft 365. Det kan du göra på några olika sätt.
  
|Olika sätt att blockera åtkomst|Definition|Metodtips|
|:-----|:-----|:-----|
|Blockera inloggning  <br/> |Ett sätt att blockera en användare från Microsoft 365 att ändra deras inloggningsstatus till **Inloggning blockerad**. Det gör att de inte kan logga Microsoft 365 från sina datorer och mobila enheter, men de kan fortfarande visa tidigare nedladdad eller synkroniserad e-post och dokument. Om du använder Blackberry Enterprise Service kan du inaktivera deras åtkomst där också.  <br/> |Använd den här metoden när en anställd planerar att lämna organisationen eller planerar att ha en lång ledighet.  <br/> |
|Återställa användarlösenord  <br/> |Ett annat sätt att hindra en användare från att Microsoft 365 att återställa deras lösenord. Det gör att de inte kan använda sitt konto, men de kan fortfarande visa tidigare nedladdad eller synkroniserad e-post och dokument. Du kan sedan logga in som dem och ändra lösenord till ett eget.  <br/> |Använd den här metoden när en anställd slutar plötsligt och permanent, och du anser att det finns anledning att oroa sig för affärsdata.  <br/> |
|Ta bort alla tilldelade licenser  <br/> |Ett annat alternativ är att ta Microsoft 365 licenser som tilldelats till användaren. Det hindrar dem från att använda program och tjänster som Office-paketet, Office-appar för webben, Yammer och SharePoint Online. De kan fortfarande logga in, men kan inte använda de här tjänsterna.  <br/> |Använd den här när du anser att användaren inte längre behöver åtkomst till specifika funktioner Microsoft 365.  <br/> <br> **Viktigt!** När du tar bort en licens tas användarens postlåda bort efter 30 dagar.
