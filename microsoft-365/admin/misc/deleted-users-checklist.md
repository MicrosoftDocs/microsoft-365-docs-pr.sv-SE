# <a name="quick-help-deleted-users-checklist"></a>Snabbhjälp: Checklista för borttagna användare

Du har alltså bestämt dig för att ta bort en användare från organisationen. Du kan till exempel ta bort en användare när personen lämnar organisationen. Det finns några saker du bör göra innan du tar bort och efter att du tagit bort användaren. Den här checklistan hjälper dig med det.
  
## <a name="remove-the-microsoft-365-license-from-the-user"></a>Ta bort Microsoft 365 licens från användaren

Du kan ta bort en licens från en användare genom att  söka efter användaren på startsidan, eller från sidan Aktiva användare, och sedan välja dem. Avmarkera **kryssrutan för den** licens du vill ta bort från användaren på fliken Licenser och appar. [Läs hela artikeln](../manage/remove-licenses-from-users.md).
  
## <a name="block-sign-in"></a>Blockera inloggning

Du kan blockera användaren så att han eller hon inte kan logga in i din organisation. Du kan söka efter användare på startsidan, eller från sidan **Aktiva användare**, och sedan välja dem. Högst upp i fönstret, under användarens namn,  väljer du ikonen Blockera den här användaren och markerar sedan kryssrutan Blockera den här användaren från att logga **in.** [Läs hela artikeln](../add-users/assign-admin-roles.md).
  
## <a name="get-their-data"></a>Hämta data

Du kan skicka användarens e-post till en annan anställd eller användare i organisationen genom att följa dessa [steg](../add-users/remove-former-employee.md). Du kan även säkerhetskopiera och spara data från den tidigare anställde genom att följa dessa [steg](../add-users/get-access-to-and-back-up-a-former-user-s-data.md).
  
## <a name="delete-user"></a>Delete user

Du kan ta bort användaren genom att söka efter användaren på startsidan, eller från sidan **Borttagna användare**. Markera användaren och välj Ta **bort användare.** [Läs hela artikeln](../add-users/delete-a-user.md).
  
## <a name="what-to-do-with-the-unassigned-license"></a>Vad gör jag med den borttagna licensen

Om du har en ny anställd eller person som ska ansluta till organisationen kan du tilldela licensen till dem så att de kan använda Microsoft 365. Du kan söka efter användare på startsidan, eller från sidan **Aktiva användare**, och sedan välja dem. På fliken **Licenser och** appar markerar du kryssrutan för den licens du vill lägga till för den nya användaren. [Läs hela artikeln](../manage/assign-licenses-to-users.md).
  
## <a name="remove-license-so-you-dont-have-to-pay-for-it"></a>Ta bort en licens så att du slipper betala för den

Du kan också ta bort en licens från prenumerationen så att du slipper betala för den. Skriv prenumeration på startsidan för administrationscentret **och** välj **Visa prenumerationer.** Välj **Lägg till/ta bort licenser.** I rutan **Totalt antal** licenser anger du det totala antalet licenser som du behöver för prenumerationen och väljer sedan **Skicka ändring.** Om du till exempel hade 5 licenser och du behöver ta bort en av dem skriver du **4**. [Läs hela artikeln](../../commerce/licenses/buy-licenses.md).
