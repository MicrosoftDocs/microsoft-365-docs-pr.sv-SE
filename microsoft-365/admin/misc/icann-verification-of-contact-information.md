# <a name="icann-verification-of-contact-information"></a>ICANN-verifiering av kontaktinformation

När du köper en domän eller gör ändringar i kontaktinformationen för en domän måste registratorn verifiera din kontaktinformation för att följa ICANN-bestämmelser.

För validering skickas ett e-postmeddelande till registrantens adress.

 **Bekräfta din kontaktinformation genom att välja länken i e-postbekräftelsen.** Om domänägarna inte verifierar e-postadressen stängs domänen av. Gå sedan till sidan [Domäner i](https://admin.microsoft.com/adminportal/home?ref=Domains) administrationscentret och kontrollera din domänstatus.
