# <a name="microsoft-365-reports-in-the-admin-center---microsoft-browser-usage"></a>Microsoft 365-rapporter i administrationscentret – Microsoft-webbläsaranvändning

Instrumentpanelen Microsoft **365-rapporter** visar en översikt över aktiviteter för produkter i din organisation. Där kan du gå in på detaljnivå i rapporter för enskilda produkter för att få mer detaljerad insyn i aktiviteterna inom varje produkt. Ta en titt på [översiktsavsnittet för Rapporter](activity-reports.md). I rapporten om Microsoft-användning av webbläsare får du inblick i Internet Explorer, Microsoft Edge Legacy och ny Microsoft Edge-användning. Användningsrapporteringen baseras på Microsoft 365-onlinetjänster som är tillgängliga via en Microsoft-webbläsare.

 > [!NOTE]
 > Du måste vara global administratör, global läsare eller rapportläsare i Microsoft 365 eller Exchange-, SharePoint- eller Skype för företag-administratör för att kunna se rapporter.

## <a name="how-to-get-to-the-microsoft-browser-usage-report"></a>Så här kommer du till microsofts användningsrapport för webbläsare

1. I administrationscentret går du till sidan **Rapporter** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">Användning</a>. 
2. På startsidan för instrumentpanelen klickar du på **knappen Visa mer** på Microsoft-kortet för webbläsaranvändning.

## <a name="interpret-the-microsoft-browser-usage-report"></a>Tolka Microsofts användningsrapport för webbläsare
