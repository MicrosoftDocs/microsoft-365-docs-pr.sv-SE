# <a name="microsoft-365-reports-in-the-admin-center---yammer-groups-activity-report"></a>Microsoft 365 Rapporter i administrationscentret – Yammer aktivitet i grupper

På Microsoft 365 **Rapporter** ser du en översikt över aktiviteter i organisationens produkter. Här kan du gå in på detaljnivå i rapporter för enskilda produkter för att få bättre insikter om aktiviteterna inom varje produkt. Ta en titt på [översiktsavsnittet för Rapporter](activity-reports.md). I rapporten Aktivitet i Yammer-grupper får du inblick i aktiviteten i Yammer-grupperna i organisationen och du kan se hur många Yammer-grupper som används.
  
> [!NOTE]
> Du måste vara global administratör, global läsare eller rapportläsare i Microsoft 365 eller Exchange-, SharePoint-, Teams-tjänst, Teams Communications- eller Skype för företag-administratör för att kunna se rapporter.  
 
## <a name="how-do-i-get-to-the-yammer-groups-activity-report"></a>Hur kommer jag åt Yammer gruppaktivitetsrapport?

1. I administrationscentret går du till sidan **Rapporter** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">Användning</a>. 
2. På startsidan för instrumentpanelen klickar du på **knappen Visa** mer på Yammer kort.

  
## <a name="interpret-the-yammer-groups-activity-report"></a>Tolka rapporten Aktivitet i Yammer-grupper

Du kan visa gruppaktiviteterna i rapporten Yammer genom att välja **fliken Gruppaktivitet.**<br/>![Microsoft 365 – Aktivitetsrapport Yammer Microsoft-grupper.](../../media/3afdafe5-9269-402e-8264-c7695ceb227d.png)

Välj **Välj kolumner för** att lägga till eller ta bort kolumner i rapporten.  <br/> ![Yammer över gruppaktivitet – välj kolumner](../../media/54744932-34fe-48c3-9779-1d10c3f05be1.png)

Du kan också exportera rapportdata till en Excel .csv fil genom att välja **länken** Exportera. Då exporteras data för alla användare och du kan göra enkel sortering och filtrering för vidare analys. Om du har mindre än 2 000 användare kan du sortera och filtrera i tabellen i själva rapporten. Om du har fler än 2 000 användare måste du exportera data för att kunna filtrera och sortera. 
  
|Objekt|Beskrivning|
|:-----|:-----|
|**Metrisk**|**Definition**|
|Gruppnamn  <br/> |Namnet på gruppen. <br/> |
|Gruppadministratör  <br/> |Namnet på gruppadministratören eller ägaren.  <br/> |
|Borttagen  <br/> |Antalet borttagna Yammer grupper. Om gruppen tagits bort, men uppvisade aktivitet under rapporteringsperioden visas det i tabellen om här flaggan är satt till sant.  <br/> |
|Typ  <br/> |Typ av grupp, offentlig eller privat. <br/> |
|Ansluten till Office 365  <br/> |Anger om gruppen Yammer också är en Microsoft 365 grupp. <br/> |
|Datum för senaste aktivitet (UTC)  <br/> | Det senaste datumet då ett meddelande lästs, publicerats eller gillats av gruppen.  <br/> |
|Medlemmar  <br/> | Antalet medlemmar i gruppen.  <br/> |
|Publicerad  <br/> |Antalet meddelanden som publicerats i Yammer gruppen under rapporteringsperioden. <br/>|
|Läsa  <br/> |Antalet konversationer som lästs Yammer gruppen under rapporteringsperioden.  <br/> |
