# <a name="mail-flow-map-report"></a>Kartrapport för e-postflöde

Den här rapporten innehåller insikter om hur e-post flödar genom din Office 365-organisation. Du kan använda den här informationen för att lära dig mönster, identifiera avvikelser och åtgärda problem när de uppstår.

![Mapprapporten för e-postflöde i instrumentpanelen för e-postflöde i Säkerhets- & Compliance Center](../../media/mail-flow-map-selected.png)

## <a name="mail-flow-map-widget"></a>Widgeten Mappning för e-postflöde

Som standard visar e-postflödeskartan det snabba e-postflödesmönstret från föregående dag. Du kan använda vänster- och högerpilarna för olika dagar. När du håller muspekaren över varje område i rapporten visas volymen e-post till och från Office 365-organisationen enligt följande diagram:

![Vänster- och högerpilar i widgeten E-postflödeskarta](../../media/mail-flow-map-widget.png)

## <a name="overview"></a>Översikt

Om du klickar på **widgeten E-postflödesmappning** tar du dig till **kartrapporten För e-postflöde.** Här kan du se mer detaljerad rapportnivå, du kan klicka på Visa informationstabell för att se detaljerade data. Du kan också hämta den detaljerade rapporten genom att klicka på Begäran-rapport.

![Översiktsvy i rapporten Karta över e-post](../../media/mail-flow-map-overview.png)

## <a name="details"></a>Information

Som standard anges **Visa data för** värdet **Översikt**. När du klickar på nedrullningen och väljer **Detalj**växlar vyn till domännivåinformationen.

![Välj Detalj i Visa data för i översiktsvyn i kartrapporten E-postflöde](../../media/mail-flow-map-select-detail.png)

De övre avsändar- och mottagardomänerna visas och resten kommer att placeras i **Andra** enligt följande diagram:

![Informationsvyn i kartrapporten E-postflöde](../../media/mail-flow-map-detail.png)

## <a name="related-insights"></a>Relaterade insikter

Relaterade insikter visas under flödeskartan E-post om de är tillgängliga (till exempel sändardomäninsikt eller e-postloopens insikt).

## <a name="see-also"></a>Se även

Mer information om andra insikter om e-postflöde på instrumentpanelen för e-postflöde finns [i E-postflödesinsikter i Security & Compliance Center](mail-flow-insights-v2.md).
