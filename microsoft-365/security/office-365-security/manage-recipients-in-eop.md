# <a name="manage-recipients-in-eop"></a>Hantera mottagare i EOP

Microsoft Exchange Online Protection (EOP) erbjuder flera sätt att hantera dina e-postmottagare. Som administratör kan du utföra vissa hanteringsuppgifter i Administrationscenter för Exchange (EAC) eller använda fjärrdata Windows PowerShell och verifiera andra hanteringsuppgifter som utförs i Microsoft 365-administrationscentret.

EOP stöder följande typer av mottagare:

- **E-postanvändare:** E-postanvändare är mottagare i dina EOP-hanterade domäner. Dessa mottagare har inloggningsuppgifter i organisationen, men de har externa e-postadresser, vilket innebär att deras mottagarpostlådor finns utanför din molnorganisation.

  Du kan lägga till e-postanvändare så att de kan ta emot e-post och du kan också skapa regler för e-postflöde (kallas även transportregler) för specifika användare. Du kan också tilldela roller till e-postanvändare i organisationen. användare med behörighet för hanteringsrollgrupp kan komma åt Administrationscenter för Exchange (EAC) och utföra vissa hanteringsuppgifter. Mer information om användarroller och hur du tilldelar användarroller i EOP finns [i Hantera administratörsrollgruppbehörigheter i EOP](manage-admin-role-group-permissions-in-eop.md).

  Mer information om hur du hanterar e-postanvändare i EOP finns [i Hantera e-postanvändare i EOP](manage-mail-users-in-eop.md).

- **Grupper**: E-postanvändare kan grupperas tillsammans i distributionsgrupper eller säkerhetsgrupper.

Mer information om hur du hanterar grupper i EOP finns [i Hantera grupper i EOP](manage-groups-in-eop.md).
