# <a name="manage-recipients-in-eop"></a>Hantera mottagare i EOP

Microsoft Exchange Online Protection (EOP) erbjuder flera sätt att hantera dina e-postmottagare. Som administratör kan du utföra vissa hanteringsuppgifter i Administrationscentret för Exchange (EAC) eller använda fjärr- och Windows PowerShell och verifiera andra hanteringsuppgifter som utförs i administrationscentret för Microsoft 365.

EOP stöder följande typer av mottagare:

- **E-postanvändare:** E-postanvändare är mottagare i dina EOP-hanterade domäner. Dessa mottagare har inloggningsuppgifter i din Office 365-organisation, men de har externa e-postadresser, vilket innebär att deras mottagarpostlådor finns utanför molnorganisationen.

  Du kan lägga till e-postanvändare så att de kan ta emot e-post och du kan också skapa regler för e-postflöde (kallas även transportregler) för specifika användare. Du kan också tilldela roller till e-postanvändare i organisationen. användare med behörighet för hanteringsrollgrupp kan komma åt Administrationscentret för Exchange (EAC) och utföra vissa hanteringsuppgifter. Mer information om användarroller och hur du tilldelar användarroller i EOP finns i [Hantera administratörsrollgruppbehörigheter i EOP](manage-admin-role-group-permissions-in-eop.md).

  Mer information om hur du hanterar e-postanvändare i EOP finns i [Hantera e-postanvändare i EOP](manage-mail-users-in-eop.md).

- **Grupper**: E-postanvändare kan grupperas tillsammans i distributionsgrupper eller säkerhetsgrupper.

Mer information om hur du hanterar grupper i EOP finns [i Hantera grupper i EOP](manage-groups-in-eop.md).
