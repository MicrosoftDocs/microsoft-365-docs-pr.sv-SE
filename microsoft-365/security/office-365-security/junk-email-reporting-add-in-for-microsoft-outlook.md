# <a name="junk-email-reporting-add-in-for-microsoft-outlook"></a>Tillägg för skräppostrapportering för Microsoft Outlook

Med tillägget Skräppostrapportering för Microsoft Outlook kan användare av Exchange Online- och Exchange Online Protection enkelt rapportera skräppost till Microsoft för analys för att minska antalet och effekterna av framtida skräppostmeddelanden som filtrerats av tjänsten. Kompatibel med Microsoft Outlook 2013, Outlook 2010 eller Outlook 2007 med Windows 7, Windows 8 eller Windows 10, rapporteringstillägget ger följande fördelar:
  
- En klickrapportering som gör det möjligt för användare att välja skräppost och skicka den till Microsoft för analys.
    
- Möjligheten att välja och sedan skicka flera e-postmeddelanden med ett enda klick.
    
- Valda e-postmeddelanden flyttas till mappen Skräppost i Microsoft Outlook.
    
- Stöd för flera språk.
    
Följande avsnitt innehåller vägledning om hur du distribuerar, använder och felsöker tillägget Microsoft Skräppostrapportering för Outlook:
  
- [Installera tillägget Skräppostrapportering för Microsoft Outlook](install-the-junk-email-reporting-add-in-for-microsoft-outlook.md) Innehåller information som administratörer kan installera och distribuera tillägget i organisationen. Den innehåller också information om hur du avinstallerar tillägget. 
    
- [Rapportera skräppost meddelanden till Microsoft](report-junk-email-messages-to-microsoft.md) Innehåller information för slutanvändare om hur du använder tillägget för att rapportera skräppost meddelanden till Microsoft. 
    
- [Felsökning och supportinformation](troubleshooting-and-support-information.md) Innehåller felsökningssteg för slutanvändare och administratörer och information om hur du kontaktar teknisk support för hjälp.
