# <a name="send-and-receive-smime-signed-and-encrypted-email-in-exchange-online"></a>Skicka och ta emot S/MIME signerad och krypterad e-post i Exchange Online

Att skicka eller svara på ett S/MIME-krypterat meddelande i Microsoft Outlook påminner mycket om upplevelsen med ett icke-krypterat meddelande. Mer information om hur du läser eller skickar S/MIME-krypterade meddelanden från ett e-postprogram som Outlook på webben (tidigare kallat Outlook Web App) finns i [Kryptera meddelanden med hjälp av S/MIME i Outlook på webben](https://support.office.com/article/878c79fc-7088-4b39-966f-14512658f480).
