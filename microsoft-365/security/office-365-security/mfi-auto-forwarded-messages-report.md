# <a name="auto-forwarded-messages-report"></a>Rapport över vidarebefordrade meddelanden automatiskt

Rapporten **Meddelanden som vidarebefordras automatiskt** i instrumentpanelen för e-postflöde visar information om meddelanden som automatiskt vidarebefordras från Office 365-organisationen till mottagare i externa domäner.

![Den automatiska vidarebefordrade meddelanden insikt i Office 365 Security & Compliance Center](../../media/8bc2600b-71c3-4b37-b4d0-9435fe0cfc8d.png)

![Rapporten Meddelanden som vidarebefordras automatiskt i instrumentpanelen för e-postflöde i Office 365 Security & Compliance Center](../../media/8bc2600b-71c3-4b37-b4d0-9435fe0cfc8d.png)

## <a name="auto-forwarded-messages-details"></a>Information om vidarebefordrade meddelanden automatiskt

När du klickar på antalet meddelanden i widgeten visas ett utfällbart fönster som visar statusen för automatiskt vidarebefordrade meddelanden. Du kan se informationen genom att klicka på länken **Vidarebefordra rapport.**

![Utfällbara uppgifter för rapporten Vidarebefordrade meddelanden automatiskt i Office 365 Security & Compliance Center](../../media/87d0fb1e-d2ef-4901-b17c-ec32d23a539e.png)

## <a name="insights"></a>Insikter

Två insikter genereras baserat på rapportdata: **Nya vidarebefordran användare** och **nya vidarebefordran domäner**. Varje insikt innehåller en sammanfattning av antalet nya vidarebefordrare eller domäner med en länk till den **nya vidarebefordringsrapporten** som ger mer information om de två typerna av data. I **den nya vidarebefordringsrapporten** visas också en tidslinjevy över ny aktivitet och informationstabellen identifierar vem som började vidarebefordra och till vilka domäner.
