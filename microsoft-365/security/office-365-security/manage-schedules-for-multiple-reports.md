# <a name="manage-schedules-for-multiple-reports-in-the-security-amp-compliance-center"></a>Hantera scheman för flera rapporter &amp; i Security Compliance Center

I Security &amp; Compliance Center finns flera [rapporter och insikter](reports-and-insights-in-security-and-compliance.md) tillgängliga för att hjälpa organisationens säkerhetsteam att minska och hantera hot mot din organisation. Om du är medlem i organisationens säkerhetsteam kan du hantera scheman för en eller flera rapporter. 
  
## <a name="manage-schedules-for-reports"></a>Hantera scheman för rapporter

> [!IMPORTANT]
> Kontrollera att du har de behörigheter som krävs [i &amp; Office 365 Security Compliance Center](permissions-in-the-security-and-compliance-center.md). I allmänhet kan Office 365 globala administratörer, säkerhetsadministratörer och &amp; säkerhetsläsare komma åt rapporter i Security Compliance Center. 
  
![I Security &amp; Compliance Center \> väljer du Rapporter Hantera scheman](../../media/efa5e2f9-bf73-4f85-acea-f1ca7e2bca5e.png)

1. Gå [https://protection.office.com](https://protection.office.com) till och logga in. Detta tar dig till Security & Compliance Center.

2. Gå till &amp; **Rapporter** \> Hantera scheman i **Säkerhetsefterlevnadscenter**.
    
3. Markera ett objekt i listan.
    
4. Granska informationen för rapportens schema.
    
5. Vid behov redigera eller ta bort schemat och klicka sedan på **Stäng**.
