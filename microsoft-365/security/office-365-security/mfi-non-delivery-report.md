# <a name="non-delivery-report"></a>Rapport om utebliven leverans

**Rapporten Icke-leverans** visar de mest påträffade felkoderna i rapporter som inte har leverans (kallas även NDR: er eller avvisningsmeddelanden) för användare i organisationen. Den här rapporten är avsedd att visa information om NDRs, så att du kan felsöka problem.

![Rapporten Utebliven leverans i instrumentpanelen för e-postflöde i Security & Compliance Center](../../media/non-delivery-report-selected.png)

Du kan visa information för alla NDR-koder eller en viss kod genom att klicka på **Visa data för**. Du kan också klicka på **Visa informationstabell** om du vill visa en mer detaljerad vy som visas i följande diagram:

![Visa informationstabell i rapporten Ej levererad](../../media/non-delivery-report-view-details-table.png)

När du väljer en rad i tabellen för att visa information om den specifika NDR kan du använda den detaljerade informationen för att felsöka och vidta nödvändiga åtgärder.

![Välj en rad i informationstabellen i rapporten Ej leverans](../../media/non-delivery-report-details-table-select-row.png)
