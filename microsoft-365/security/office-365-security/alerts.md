# <a name="alerts-in-the-office-365-security-amp-compliance-center"></a>Alerts in the Office 365 Security &amp; Compliance Center

Använd varningsfunktionerna i Office 365 Security &amp; Compliance Center för att visa och hantera aviseringar för din Office 365-organisation, inklusive hantering av avancerade aviseringar som en del av Microsoft Cloud App [Security-översikten](https://docs.microsoft.com/cloud-app-security/what-is-cloud-app-security).
  
## <a name="how-to-get-to-the-office-365-alerts-features"></a>Så här tar du dig till funktionerna för Office 365-aviseringar

Aviseringar i Office 365 &amp; finns i Säkerhetsefterlevnadscenter. Så här kommer du till sidan.
  
 **Så här går &amp; du direkt till Security Compliance Center:**
  
1. Gå till [https://protection.office.com](https://protection.office.com).
    
2. Logga in i Office 365 med ditt arbets- eller skolkonto. 
    
3. Klicka på **Aviseringar** i den vänstra rutan för att se aviseringarna. 
    
 **Så här går &amp; du till Security Compliance Center med startprogrammet för Office 365:**
  
1. Logga in i Office 365 med ditt arbets- eller skolkonto. 
    
2. Klicka på startprogrammet i det övre vänstra hörnet och klicka sedan på **Säkerhetsefterlevnad &amp; **.
    
    Hittar du inte appen du söker? Från appfönstret kan du välja **Alla appar** för att se en alfabetisk lista över tillgängliga Office 365-appar. Därifrån kan du söka efter en specifik app. 
    
3. Klicka på **Aviseringar** i den vänstra rutan för att se aviseringarna. 
    
## <a name="alerts-features"></a>Varningsfunktioner

I följande tabell beskrivs de **Alerts** verktyg som &amp; är tillgängliga under Aviseringar i Säkerhetsefterlevnadscenter. 
  
|**Verktyg**|**Beskrivning**|
|:-----|:-----|
|[Hantera aviseringar](../../compliance/create-activity-alerts.md) <br/> |Använd aktivitetsaviseringar för att skicka e-postmeddelanden till dig själv eller andra administratörer när användare utför specifika aktiviteter i Office 365. Aktivitetsaviseringar liknar att söka i granskningsloggen för Office 365 efter händelser, förutom att du får ett e-postmeddelande när en händelse som du har skapat en avisering för inträffar.  <br/> |
|[Hantera avancerade aviseringar](https://docs.microsoft.com/cloud-app-security/what-is-cloud-app-security) <br/> |Använd funktionen **Hantera avancerade aviseringar** i Office 365 Cloud App Security för att ställa in principer som kan varna dig för misstänkt och avvikande aktivitet i Office 365. När du har larmats kan du undersöka situationer som kan vara problematiska och vid behov vidta åtgärder för att lösa säkerhetsproblem.  <br/> |
