# <a name="messaging-policy-and-compliance-in-eop"></a>Meddelandepolicy och efterlevnad i EOP

Microsoft Exchange Online Protection tillhandahåller meddelandeprinciper och efterlevnadsfunktioner som kan hjälpa dig att hantera dina e-postdata.

## <a name="messaging-policy-and-compliance-documentation"></a>Meddelandepolicy och efterlevnadsdokumentation

|**Ämne**|**Beskrivning**|
|:-----|:-----|
|[Granskningsrapporter i EOP](auditing-reports-in-eop.md)|Granskningsrapporter ger information om hur du kör rapporten för administratörsrollgrupp, som visar ändringarna i rollgrupper för hantering i organisationen inom en viss tidsram, och administratörsgranskningsloggen, som registrerar alla konfigurationsändringar som gjorts i organisationen.|
|[Regler för e-postflöde (transportregler) i Exchange Online Protection](mail-flow-rules-transport-rules-0.md)|Innehåller information om regler för e-postflöde (kallas även transportregler), som ger dig flexibiliteten att tillämpa dina egna företagsspecifika principer på e-post.|
