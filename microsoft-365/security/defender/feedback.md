# <a name="provide-feedback-on-microsoft-365-defender"></a>Ge feedback om Microsoft 365 Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Gäller för:**
- Microsoft 365 Defender

Din feedback hjälper oss att bli bättre i din miljö mot avancerade attacker. Dela din upplevelse, intryck och förfrågningar genom att ge feedback.

1. Välj Lämna feedback från valfri del **av portalen.** 

    ![Bild på feedbackknappen](../../media/feedback.png)

2. Betygsätta din upplevelse och ge information om vad du gillade eller var förbättringar kan göras. Du kan också välja att bli kontaktad om din feedback. 

3. Välj **Skicka**.
