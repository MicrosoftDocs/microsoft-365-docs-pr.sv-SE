# <a name="whats-coming-to-microsoft-secure-score"></a>Vad kommer till Microsoft Secure Score

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

Microsoft Secure Score hittar du på https://security.microsoft.com/securescore Microsoft 365 [Säkerhetscenter.](overview-security-center.md)

## <a name="proposed-changes"></a>Föreslagna ändringar

Vi kommer att göra några ändringar i framtiden för att [microsoft Secure Score](microsoft-secure-score.md) ska bli en bättre representant för säkerheten och förbättra användbarheten. Ditt poängresultat och högsta möjliga poäng kan komma att ändras.

### <a name="july-2021"></a>Juli 2021

#### <a name="add-improvement-action-related-to-microsoft-teams"></a>Lägg till förbättringsåtgärd relaterad till Microsoft Teams

- Begränsa uppringningsanvändare från att kringgå en möteslobby.
- Begränsa externa deltagares möjlighet att ha kontroll i Teams möte.
- Hindra anonyma användare från att Teams möten.
- Kräv att samtalslobbyn konfigureras för Teams möten.
- Konfigurera vilka användare som tillåts vara närvarande i Teams möten.

#### <a name="add-improvement-action-related-to-microsoft-defender-for-endpoint"></a>Lägg till förbättringsåtgärd relaterad till Microsoft Defender för Slutpunkt
- Åtgärda Microsoft Defender för slutpunkts sensordatasamling för macOS
- Åtgärda Microsoft Defender för nedsatt kommunikation med slutpunkt för macOS
- Ange en minsta lösenordslängd på 15 eller fler tecken i macOS
- Ange "Framtvinga lösenordshistorik" till 24 eller fler lösenord i macOS
- Ange maximal ålder för lösenord till 90 dagar eller färre, men inte 0 i macOS
- Ange tröskelvärdet för kontolåsning till 5 eller lägre i macOS
- Aktivera brandväggen på macOS
- Aktivera Gatekeeper
- Aktivera System Integrity Protection (SIP)
- Aktivera FileVault diskkryptering
- Ange att skärmen ska låsas när skärmsläckaren startar i macOS
- Kontrollera att skärmsläckaren är inställd att starta om 20 minuter eller mindre i macOS
- Säkra hemmappar
- Aktivera Microsoft Defender Antivirus realtidsskydd för macOS
- Aktivera PUA Microsoft Defender Antivirus skydd i blockeringsläge för macOS
- Aktivera Microsoft Defender Antivirus moln levererat skydd för macOS
- Uppdatera Microsoft Defender Antivirus för macOS
- Åtgärda Microsoft Defender för endpoint sensordatasamling för Linux
- Åtgärda Microsoft Defender för slutpunkts nedsatt kommunikation för Linux
- Obegränsade Access-konton
- Aktivera Microsoft Defender Antivirus realtidsskydd för Linux
- Aktivera PUA Microsoft Defender Antivirus i blockeringsläge för Linux
- Aktivera Microsoft Defender Antivirus moln levererat skydd för Linux
- Uppdatera Microsoft Defender Antivirus definitioner för Linux
