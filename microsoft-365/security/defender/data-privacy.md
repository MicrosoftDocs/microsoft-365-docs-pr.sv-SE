# <a name="microsoft-365-defender-data-security-and-privacy"></a>Microsoft 365 Defender datasäkerhet och sekretess

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Gäller för:**
- Microsoft 365 Defender

Microsoft 365 Defender arbetar Microsoft Azure datacenter i EU, Storbritannien och USA. Kunddata som samlas in av tjänsten lagras vilan i (a) innehavarens geografiska plats som identifieras under etableringen eller(b) om Microsoft 365 Defender använder en annan Microsoft-onlinetjänst för att bearbeta sådana data, den geolokalisering som definieras av datalagringsreglerna för den andra onlinetjänsten.

Kunddata i anonymiserad form kan också lagras i centrala lagrings- och bearbetningssystem i USA.

- [Microsoft Cloud App Security datasäkerhet och sekretess](/cloud-app-security/cas-compliance-trust)
