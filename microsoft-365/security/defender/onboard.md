# <a name="configure-and-manage-microsoft-defender-for-endpoint-capabilities"></a>Konfigurera och hantera Microsoft Defender för slutpunktsfunktioner

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

**Gäller för:**

- [Microsoft Defender för Endpoint](https://go.microsoft.com/fwlink/p/?linkid=2069559)

Konfigurera och hantera alla Microsoft Defender för Slutpunkt-funktioner för att få det bästa säkerhetsskyddet för din organisation. 


## <a name="in-this-section"></a>I det här avsnittet 
Ämne | Beskrivning 
:---|:---
[Konfigurera funktioner för att minska attackytan](/windows/security/threat-protection/microsoft-defender-atp/configure-attack-surface-reduction) |  Genom att säkerställa att konfigurationsinställningarna är korrekt inställda och utnyttja tekniker för minskningar tillämpas, kommer dessa uppsättning funktioner att motarbeta attacker och användning. 
[Konfigurera nästa generations skydd](/windows/security/threat-protection/windows-defender-antivirus/configure-windows-defender-antivirus-features) | Konfigurera nästa generations skydd för att fånga alla typer av nya hot.
[Konfigurera microsoft Threat Experts-funktioner](/windows/security/threat-protection/microsoft-defender-atp/configure-microsoft-threat-experts) | Konfigurera och hantera hur du vill få information om hot mot cyberhot från Microsoft Threat Experts.
[Konfigurera Microsoft 365 Defender-integrering](/windows/security/threat-protection/microsoft-defender-atp/threat-protection-integration)| Konfigurera andra lösningar som kan integreras med Microsoft Defender för Endpoint.
[Hanterings- och API-support](/windows/security/threat-protection/microsoft-defender-atp/management-apis)| Hämta aviseringar till ditt SIEM eller använd API:er för att skapa anpassade aviseringar. Skapa och skapa Power BI-rapporter. 
[Konfigurera inställningar för Microsoft Defender Säkerhetscenter](/windows/security/threat-protection/microsoft-defender-atp/preferences-setup) |  Konfigurera portalrelaterade inställningar, till exempel allmänna inställningar, avancerade funktioner, aktivera förhandsgranskning och andra.
