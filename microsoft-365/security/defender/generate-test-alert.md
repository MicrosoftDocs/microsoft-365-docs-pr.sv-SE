# <a name="generate-a-test-alert-in-your-microsoft-365-defender-evaluation-lab"></a>Generera en testavisering i Microsoft 365 Defender-utvärderingslabb  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Gäller för:**
- Microsoft 365 Defender

Nu när du har slutfört konfigurationen och konfigurationen av utvärderingslabben i Microsoft 365 Defender kan du köra en simulering av hotattack och upptäcka, varna korrelation och självreparationsfunktioner i den integrerade säkerhetslösningen.  

1. Logga in på https://security.microsoft.com
2. På startsidan letar du efter en panel med namnet **Simulering**.  Gå igenom genomgångsguiden för anvisningar om hur du skapar ett testhändelse i Microsoft 365 Defender.
<br>![Bild av simuleringskort på Microsoft 365 Säkerhetsinstrumentpanel](../../media/mtp-eval-73.png) <br>
