# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Dataövervakning och rapportering i Microsoft 365-säkerhetscentret

Kategorin **Data** hjälper till att spåra användaraktivitet som kan leda till obehörigt dataavslöjande. Det här är en omarbetning av befintliga DLP-principrapporter plus en DLP-principmatchningsrapport från tredje part.

Du kan se:

* Användare som delar flest filer från molnappar
* Hur många DLP-principmatchningar inträffade
* Hur många DLP-principer åsidosätter eller falska positiva identifieringar rapporteras
* Hur många DLP-principmatchningar som har hänt i molntjänster från tredje part via Microsoft Cloud App Security
