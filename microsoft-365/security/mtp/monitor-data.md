# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Data övervakning och rapportering i säkerhets Center för Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Data** kategorin hjälper till att spåra användar aktivitet som kan leda till att obehöriga data lämnas ut. De håller på att arbeta med befintliga DLP policy-rapporter samt en matchnings rapport från tredje part.

Du kan se:

- Användare som delar de flesta filer från molnappar
- Hur många matchningar av DLP-principer som uppstår
- Hur många DLP-principer som åsidosätter eller falsk identifiering rapporteras
- Hur många olika DLP-principer som görs i moln tjänster från tredje part via Microsoft Cloud App Security
