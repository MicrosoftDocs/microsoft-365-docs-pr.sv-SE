# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Dataövervakning och rapportering i Microsoft 365 säkerhetscenter

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


Kategorin **Data** hjälper till att spåra användaraktivitet som kan leda till obehörig åtkomst till data. De är omarbetningar av befintliga DLP-principrapporter plus en rapport om matchning av DLP-princip från tredje part.

Du kan se:

- Användare som delar mest filer från molnappar
- Hur många DLP-principmatchningar har inträffat
- Hur många DLP-principer som åsidosätts eller falska positiva resultat rapporteras
- Hur många DLP-principmatchningar som har hänt i molntjänster från tredje part via Microsoft Cloud App Security
