# <a name="microsoft-threat-protection-data-security-and-privacy"></a>Data säkerhet och sekretess för Microsoft Threat Protection

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Gäller för:**
- Microsoft Hotskydd

Microsoft Threat Protection fungerar i Microsoft Azure Data Centers i Europeiska unionen, Storbritannien och USA. Kunddata som samlas in av tjänsten bevaras i vila (a) den geografiska placeringen av innehavaren som identifieras under etableringen eller (b) om ett skydd mot Microsoft-hotet använder en annan Microsoft Online-tjänst för att bearbeta sådana data är den geografisk placeringen enligt reglerna för data lagring för den andra online-tjänsten.

Kunddata i pseudonymized-format kan även lagras i centrala förvarings-och process system i USA.


Mer information om data lagring och integritets information för de särskilda produkterna finns i:
- [Microsoft Defender ATP-data lagring och integritet](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/data-storage-privacy)
- [Säkerhets information och integritet för Microsoft Cloud App-appen](https://docs.microsoft.com/cloud-app-security/cas-compliance-trust)
- [Sekretess, säkerhet och öppenhet för Microsoft 365](https://docs.microsoft.com/office365/servicedescriptions/office-365-platform-service-description/privacy-security-and-transparency#advanced-threat-protection)
