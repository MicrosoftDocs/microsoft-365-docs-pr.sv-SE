# <a name="microsoft-threat-protection-data-security-and-privacy"></a>Microsoft Threat Protection datasäkerhet och sekretess

**Gäller:**
- Skydd av Hot mot Microsoft



Genom att använda förhandsversionen av Microsoft Threat Protection godkänner du följande villkor:

Dina tillämpliga kunddata (enligt definitionen i villkoren för onlinetjänster ("OST") överförs från andra Microsoft-tjänster till Microsoft Threat Protection och från Microsoft Threat Protection tillbaka till tillämpliga Microsoft-tjänster. Under den här förhandsversionen följer användningen av dina kunddata i Microsoft Threat Protection datahanteringsstandarderna och åtagandena för Microsoft Defender Advanced Threat Protection (Microsoft Defender ATP). Du bekräftar att sådana åtaganden kan skilja sig från de tjänster som kunddata överförs från. Dessutom lagras kunddata som lagras i Microsoft Threat Protection i resten i geon som du har valt för lagring av dina Microsoft Defender ATP-kunddata, vilket kan skilja sig från den Geo du valde i samband med andra tjänster. Microsoft överför inte kunddata utanför en sådan geo förutom vad som anges i avsnittet "Dataplats" i Microsoft Trust Center som finns på [Microsoft Trust Center](https://www.microsoft.com/trust-center).

Även om du avgör vilka av dina användare som kan komma åt Microsoft Threat Protection tillåter Microsoft Threat Protection för närvarande inte rollbaserad åtkomstkontroll ("RBAC"). Du bekräftar att i den mån Microsoft Threat Protection tar emot data från en Microsoft-tjänst som stöder RBAC gäller åtkomstnivåer i den tjänsten inte i Microsoft Threat Protection.
