# <a name="microsoft-365-defender-data-security-and-privacy"></a>Microsoft 365 Defender data säkerhet och integritet

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Gäller för:**
- Microsoft 365 Defender

Microsoft 365 Defender fungerar i Microsoft Azure Data Centers i Europeiska unionen, Storbritannien och USA. Kunddata som samlas in av tjänsten bevaras i vila (a) den geografiska placeringen av innehavaren som identifieras under etableringen eller (b) om Microsoft 365 Defender använder en annan Microsoft Online-tjänst för att bearbeta sådana data, det vill säga geografiskheten enligt data lagrings reglerna för den andra online-tjänsten.

Kunddata i pseudonymized-format kan även lagras i centrala förvarings-och process system i USA.


Mer information om data lagring och integritets information för de specifika produkterna finns i:
- [Microsoft Defender för slut punkts data lagring och integritet](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/data-storage-privacy)
- [Säkerhets information och integritet för Microsoft Cloud App-appen](https://docs.microsoft.com/cloud-app-security/cas-compliance-trust)
- [Sekretess, säkerhet och öppenhet för Microsoft 365](https://docs.microsoft.com/office365/servicedescriptions/office-365-platform-service-description/privacy-security-and-transparency#advanced-threat-protection)
