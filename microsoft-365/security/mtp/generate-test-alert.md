# <a name="generate-a-test-alert-in-your-microsoft-threat-protection-evaluation-lab"></a>Generera en testvarning i microsofts utvärderingslabb för hotskydd  

**Gäller:**
- Microsoft Hotskydd

Nu när du har slutfört installationen och konfigurationen för Microsoft Threat Protection-utvärderingslabbet kan du köra en simulering av hotattacker för att uppleva identifierings-, varningskorrelationen och självläkande reparationsfunktionerna i den integrerade säkerhetslösningen.  

1. Logga in påhttps://security.microsoft.com
2. Leta efter en panel med namnet **Simulering**på startsidan .  Gå igenom-guiden för steg för att skapa en testincident i Microsoft Threat Protection.
