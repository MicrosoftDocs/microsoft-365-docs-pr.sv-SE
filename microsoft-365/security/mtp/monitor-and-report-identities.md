# <a name="identity-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Identitetsövervakning och rapportering i Microsoft 365-säkerhetscentret

Du kan övervaka identiteterna i organisationen och hålla reda på misstänkta eller riskfyllda beteenden. I kategorin **Identiteter** **i rapporter**kan du spåra:

* Användare med de mest identifierade avvikelserna
* Hur många användare som rapporteras i riskzonen av principer för villkorlig åtkomst
* Antalet globala administratörer i din organisation

![Kategori för identiteter av rapportsidan](../../media/identities.png)

För användare med specifika identifieringar kan du utforska den specifika aviseringen och undersöka i Microsoft Defender Security Center. Identifieringar inkluderar avvikelser, till exempel användare som loggar in från okända platser.

En fullständig uppsättning riskhändelser finns i [Azure Active Directory-riskhändelser](https://docs.microsoft.com/azure/active-directory/reports-monitoring/concept-risk-events).
