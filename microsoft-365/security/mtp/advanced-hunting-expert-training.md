<span data-ttu-id="ddd25-140">Följande utdrag från en CSL-fil visar en omfattande uppsättning vägledningar som är markerade som kommentarer med `//` .</span><span class="sxs-lookup"><span data-stu-id="ddd25-140">The following excerpt from a CSL file shows a comprehensive set of guidance marked as comments with `//`.</span></span>
