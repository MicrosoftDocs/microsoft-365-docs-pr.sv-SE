# <a name="whats-coming-to-microsoft-secure-score"></a>Det här kommer till Microsoft Secure Score

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

Microsoft Secure Score finns på https://security.microsoft.com/securescore Microsoft [365 säkerhetscenter.](overview-security-center.md)

## <a name="proposed-changes"></a>Föreslagna ändringar

Vi kommer att göra några ändringar inom kort för att [göra Microsoft Secure Score](microsoft-secure-score.md) till en bättre representant för din säkerhet och förbättra användbarheten. Ditt resultat och högsta möjliga poäng kan ändras.

### <a name="no-planned-changes"></a>Inga planerade ändringar

Det finns inga planerade ändringar för stunden.
