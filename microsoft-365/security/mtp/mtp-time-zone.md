# <a name="set-the-time-zone-for-microsoft-threat-protection"></a>Ange tids zonen för skydd mot Microsoft Threat

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Gäller för:**
- Microsoft Hotskydd



Microsoft Threat Protection kan visa information om datum och tid med antingen din lokala tidszon eller UTC. Den valda tids zonen tillämpas på all information om datum och tid som visas i följande funktioner i Microsoft 365-säkerhets Center:
- Incidenter
- Automatisk undersökning och reparation, inklusive åtgärds Center
- Avancerad jakt

Om du vill ställa in tids zonen för de här funktionerna går du till **Inställningar**för  >  **skydds**  >  **tids zoner**från Microsoft Threat Protection.
