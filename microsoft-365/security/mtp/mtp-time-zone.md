# <a name="set-the-time-zone-for-microsoft-threat-protection"></a>Ange tidszon för Microsoft Threat Protection

**Gäller:**
- Skydd av Hot mot Microsoft



Microsoft Threat Protection kan visa datum- och tidsinformation med antingen din lokala tidszon eller UTC. Den valda tidszonen gäller för all datum- och tidsinformation som visas i följande funktioner i Microsoft 365-säkerhetscentret:
- Incidenter
- Automatiserad undersökning och reparation, inklusive åtgärdscentret
- Avancerad jakt

Om du vill ange tidszonen för dessa funktioner går du till**Tidszonen** **Inställningar** > **för Microsoft Hotskydd** > .
