# <a name="make-the-switch-to-microsoft-defender-for-endpoint"></a>Gå över till Microsoft Defender för Slutpunkt

**Gäller för:**
- [Microsoft Defender för Endpoint](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)

> Vill du uppleva Microsoft Defender för Slutpunkt? [Registrera dig för en kostnadsfri utvärderingsversion.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-exposedapis-abovefoldlink)

## <a name="migration-guides"></a>Migreringsguider

Om du funderar på att flytta över till Defender för Endpoint har vi hjälp. Granska scenarierna i följande tabell. Välj det scenario som bäst representerar din situation och se den rekommenderade vägledningen.

| Scenario | Vägledning |
|:----|:----|
| Du har ännu inte någon lösning för slutpunktsskydd, och du vill veta mer om Defender för Slutpunkt. Du vill se hur Defender för Slutpunkt fungerar innan du distribuerar det i din miljö.  | [Microsoft Defender för utvärderingslabb med slutpunkt](evaluation-lab.md)   |
| Du har redan Defender för Endpoint och du vill ha hjälp med att konfigurera allt.  | [Distributionsguide för Microsoft Defender för slutpunkt](deployment-phases.md)  |
| Du planerar att byta från en lösning som inte är en Microsoft-slutpunktsskyddslösning till Defender för Endpoint och Microsoft Defender Antivirus. Du vill få en översikt över migreringsprocessen och hur du gör bytet. |[Gå över till Microsoft Defender för Slutpunkt](switch-to-microsoft-defender-migration.md)   |
| Du har redan migrerat eller migrerat till Defender för Endpoint. Du vill ha hjälp med nästa steg, till exempel att hantera dina säkerhetsinställningar, konfigurera fler funktioner eller finjustera dina säkerhetsprinciper. | [Hantera Microsoft Defender för Slutpunkt, efter migrering](manage-atp-post-migration.md) |
