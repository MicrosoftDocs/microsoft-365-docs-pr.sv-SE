# <a name="verify-data-storage-location-and-update-data-retention-settings-for-microsoft-defender-for-endpoint"></a>Verifiera datalagringsplatsen och uppdatera datalagringsinställningarna för Microsoft Defender för Endpoint

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Gäller för:**
- [Microsoft Defender för Endpoint](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)


>Vill du använda Defender för Slutpunkt? [Registrera dig för en kostnadsfri utvärderingsversion.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-gensettings-abovefoldlink)

Under onboardingprocessen tar en guide dig igenom inställningarna för datalagring och lagring av Defender för Slutpunkt. 

När du har slutfört introduktionen kan du verifiera ditt val på sidan inställningar för datalagring.

## <a name="verify-data-storage-location"></a>Verifiera datalagringsplatsen
Under fasen [Konfigurera skulle](production-deployment.md)du ha valt plats för att lagra data. 

Du kan kontrollera dataplatsen genom att gå till **Inställningar**  >  **Datalagring**.

## <a name="update-data-retention-settings"></a>Uppdatera inställningar för datalagring

1. Välj Inställningar Datalagring **i**  >  **navigeringsfönstret.**

2. Välj varaktigheten för datalagringen i listrutan.

    > [!NOTE]
    > Andra inställningar kan inte redigeras.

3. Klicka **på Spara inställningar.**
