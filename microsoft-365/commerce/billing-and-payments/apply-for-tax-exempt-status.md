# <a name="apply-for-tax-exempt-status-for-office-365-for-business"></a>Ansöka om undantag från beskattning för Office 365 för företag

Om du är berättigad till status som skattebefriad i ditt land eller din region kontaktar du [supporten](../../admin/contact-support-for-business-products.md) för att skapa en tjänstbegäran för att fastställa skattebefriad status för din prenumeration. Du måste tillhandahålla följande dokumentation när du skickar din begäran: 
  
|**Plats**|**Dokumentation**|
|:-----|:-----|
|USA och Kanada  <br/> |Momscertifikat för moms för försäljning  <br/> |
|Puerto Rico  <br/> |Certificado de Compras Exentas  <br/> |
|Irland  <br/> |13B/56A Intyg om skattebefrielse  <br/> |
|Internationella organisationer som har skattebefrielse  <br/> |Bekräftelse av certifiering/brev från relevanta lokala skattemyndigheter  <br/> |
