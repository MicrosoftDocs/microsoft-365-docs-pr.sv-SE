# <a name="buy-a-subscription-to-office-365-for-business-from-your-free-trial"></a>Köpa ett abonnemang på Office 365 för företag från en kostnadsfri utvärderingsversion

Du har provat Office 365 och är nu redo att köpa det. Du har två alternativ när du köper Office 365 från utvärderingsversionen:
  
- [Köp samma abonnemang som din utvärderingsversion](#buy-the-same-plan-as-your-trial).
- [Köp ett annat abonnemang än din utvärderingsversion](#buy-a-different-plan-than-your-trial)

> [!NOTE]
> I slutet av din kostnadsfria provperiod konverteras provprenumerationen automatiskt till en betald prenumeration. Den betalda prenumerationen kommer som standard att vara den plan som du för närvarande har registrerat dig för. Du kan fortfarande köpa en annan plan genom att följa stegen i [avsnittet Köp en annan plan än utvärderingsavsnittet](#buy-a-different-plan-than-your-trial) nedan.

## <a name="buy-the-same-plan-as-your-trial"></a>Köp samma abonnemang som din utvärderingsversion

Det är enkelt, och när du köper samma typ av prenumeration som utvärderingsversionen behöver du inte oroa dig för att förlora data, konton eller konfiguration.

::: moniker range="o365-worldwide"

1. I administrationscentret går du till sidan **Fakturering** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Produkter och tjänster</a>.

2. Leta reda på den prenumeration som du vill köpa på sidan **Produkter & tjänster.** Välj **Köp prenumeration**i avsnittet **Licenser.**

3. Välj antingen ett månatligt eller årligt åtagande för din prenumeration och välj sedan **Checka ut**.

4. Kontrollera prenumerationen på nästa sida och välj **Checka ut**.

5. På nästa sida kontrollerar du såld **till** adress, **fakturerad till** information och **artiklar i den här ordningen**. Om du behöver göra några ändringar väljer du **Ändra** bredvid det aktuella avsnittet.

6. När du är klar väljer du **Acceptera avtal & platsorder**.

::: moniker-end

::: moniker range="o365-germany"

1. Gå till sidan **Fakturering** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Prenumerationer</a> i administrationscentret.

2. På sidan **Prenumerationer** väljer du **Köp nu** för den utvärderingsversion som du vill köpa.

3. Granska prisinformationen och välj **Nästa**.

4. Ange din betalningsinformation och välj sedan **Gör order**.

5. På bekräftelsesidan väljer du **Gå till Admin Home**.

::: moniker-end

::: moniker range="o365-21vianet"

1. Gå till sidan **Fakturering** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Prenumerationer</a> i administrationscentret.

2. På sidan **Prenumerationer** väljer du **Köp nu** för den utvärderingsversion som du vill köpa.

3. Granska prisinformationen och välj **Nästa**.

4. Ange din betalningsinformation och välj sedan **Gör order**.

5. På bekräftelsesidan väljer du **Gå till Admin Home**.

::: moniker-end

## <a name="buy-a-different-plan-than-your-trial"></a>Köp ett annat abonnemang än din utvärderingsversion

Om du köper en annan typ av prenumeration än utvärderingsversionen måste du tilldela om dina användarlicenser från utvärderingsplanen till ditt nya abonnemang (innan din 90-dagars respitperiod avslutas när utvärderingsprenumerationen upphör) för att behålla dina data, konton och konfiguration.

::: moniker range="o365-worldwide"

> [!NOTE]
> Om du inte använder det nya administrationscentret för Microsoft 365 kan du aktivera det genom att välja **Prova det nya administrationscentret** längst upp på startsidan.

1. Gå till sidan <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">Faktureringsköpstjänster</a> i **administrationscentret.** \>

2. På sidan **Köptjänster** väljer du den prenumeration som du vill köpa och väljer sedan **Köp**.

3. Ange hur många användarlicenser du behöver och om du vill betala varje månad eller för hela året. Välj om du vill att licenser automatiskt ska tilldelas alla användare som inte redan har en licens. Välj sedan **Checka ut nu**.

4. Din kundvagn öppnas. Om din varukorg försvinner från skärmen väljer du **Varukorg** för att öppna den igen, granska beställningen och välj sedan **Checka ut**.

5. Granska prisinformationen och välj **Nästa**.

6. Ange betalningsinformation och välj **sedan Beställ** \> **Gå till Admin Home**.

::: moniker-end

::: moniker range="o365-germany"

1. Gå till sidan **Fakturering** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Prenumerationer</a> i administrationscentret.

2. Välj Lägg till **prenumerationer**på sidan **Prenumeration.**

    Du ser de prenumerationer som är tillgängliga för din organisation. Alla prenumerationer som du redan har köpt identifieras som **I Prov eller** **Köpt**.
  
3. Leta reda på prenumerationen som du vill köpa och välj sedan **Köp nu**.
  
4. Ange antalet användarlicenser du behöver, välj om du vill betala varje månad eller för hela året och välj sedan **Lägg till i kundvagnen**.

5. Din kundvagn öppnas. Om din varukorg försvinner från skärmen väljer du **Varukorg** för att öppna den igen, granska beställningen och välj sedan **Checka ut**.

6. Granska prisinformationen och välj **Nästa**.

7. Ange din betalningsinformation och välj sedan **Gör order**.

8. På bekräftelsesidan väljer du **Gå till Admin Home**.

::: moniker-end

::: moniker range="o365-21vianet"


1. Gå till sidan **Fakturering** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Prenumerationer</a> i administrationscentret.

2. Välj Lägg till **prenumerationer**på sidan **Prenumeration.**

    Du ser de prenumerationer som är tillgängliga för din organisation. Alla prenumerationer som du redan har köpt identifieras som **I Prov eller** **Köpt**.
  
3. Leta reda på prenumerationen som du vill köpa och välj sedan **Köp nu**.
  
4. Ange antalet användarlicenser du behöver, välj om du vill betala varje månad eller för hela året och välj sedan **Lägg till i kundvagnen**.

5. Din kundvagn öppnas. Om din varukorg försvinner från skärmen väljer du **Varukorg** för att öppna den igen, granska beställningen och välj sedan **Checka ut**.

6. Granska prisinformationen och välj **Nästa**.

7. Ange din betalningsinformation och välj sedan **Gör order**.

8. På bekräftelsesidan väljer du **Gå till Admin Home**.

::: moniker-end
