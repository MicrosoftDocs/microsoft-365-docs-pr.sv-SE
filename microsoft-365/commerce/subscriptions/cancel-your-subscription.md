# <a name="cancel-your-subscription"></a>Avbryt prenumerationen

::: moniker range="o365-21vianet"

> [!NOTE]
> Administrationscentret förändras. Om dina erfarenheter inte överensstämmer med uppgifterna som visas här kan du läsa mer i [Om det nya administrationscentret för Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet).

::: moniker-end

*Berättigande:* Om du har färre än 25 licenser tilldelade till användare kan du avbryta utvärderings versionen av Microsoft 365 för företag eller betalat online i administrations centret för Microsoft 365 när som helst. Om du har fler än 25 licenser tilldelade till användare kan [du ringa supporten för att avbryta prenumerationen](../../admin/contact-support-for-business-products.md).

*Åter betalning:* Eventuell uppskattad kredit kommer att återföras till dig inom nästa fakturerings period.

> [!NOTE]
> Om du har flera prenumerationer på samma produkt, till exempel Microsoft 365 Business Premium, påverkar inte någon av dem de inköpta licenserna eller tjänsterna i de andra abonnemangen.

## <a name="what-happens-when-you-cancel-a-subscription"></a>Vad händer när du avbryter ett abonnemang

Om du avbryter ett abonnemang innan giltighets tiden flyttas prenumerationens status direkt till inaktiverat. För de flesta abonnemang, i de flesta länder och regioner, varar det inaktiverade tillståndet 90 dagar. Administratörer kan fortfarande komma åt och säkerhetskopiera data för organisationen när prenumerationen är i inaktiverat läge, men vi rekommenderar att administratörer [säkerhetskopierar sina data](back-up-data-before-switching-plans.md) innan de annullerar ett abonnemang, särskilt om det är sin enda prenumeration. Administratörer kan även återaktivera prenumerationen när den är inaktive rad.

Efter 90 visas abonnemanget i Borttaget. Alla data som du lämnar följer kan tas bort efter det att 90 dagar tagits 180 bort efter att du har avbrutits. Du kan inte ta bort en betalnings metod från ett annullerat abonnemang tills du når det borttagna tillståndet.

### <a name="what-to-expect-for-you-and-your-users-if-you-cancel-a-subscription"></a>Det här kan du förvänta dig för dig själv och dina användare om du avbryter en prenumeration
  
- **Administratörs åtkomst** Administratörer kan fortfarande logga in och komma åt administrations centret och köpa andra abonnemang efter behov. Som global administratör eller faktureringsadministratör har du 90 dagar på dig att [återaktivera prenumerationen](reactivate-your-subscription.md) med alla data intakta.

- **Användar åtkomst** Användarna kan inte använda tjänster som OneDrive för företag eller komma åt kunddata som e-post eller dokument på grupp webbplatser. Office-program, som Word och Excel, flyttas så småningom till ett skrivskyddat läge med nedsatt funktionalitet och [meddelanden om ej licensierad produkt](https://support.microsoft.com/office/0d23d3c0-c19c-4b2f-9845-5344fedc4380.aspx) visas.

Om du vill veta mer kan du läsa [Vad händer med mina data och åtkomst när min prenumeration](what-if-my-subscription-expires.md) går ut?

> [!IMPORTANT]
> Om du vill att dina prenumerations data ska tas bort innan den normala inaktiverade perioden är över kan du [avsluta ditt konto](../close-your-account.md).

## <a name="other-steps-you-might-have-to-take"></a>Andra åtgärder du kanske måste vidta

### <a name="change-custom-domain-settings"></a>Ändra anpassade domän inställningar

Om du använder en egen domän med ditt abonnemang finns det några extra steg som du måste göra innan du kan avbryta prenumerationen. Om du inte har en egen domän kan du hoppa över för att [Spara dina data](#save-your-data).

#### <a name="change-your-domain-nameserver-records-if-needed"></a>Ändra dina domän namnserver poster (om det behövs)

Om du har konfigurerat en egen domän lägger du till DNS-poster så att domänen fungerar med Microsoft 365-tjänster. Innan du tar bort domänen bör du uppdatera DNS-posterna, till exempel din domän MX-post, hos din DNS-värd.

Byt till exempel MX-posten hos din DNS-värd. E-post som skickas till din domän upphör till din Microsoft-adress och går till din nya e-postleverantör istället. (En MX-post bestämmer vart e-post för domänen skickas.)

- Om dina namnserver (NS-poster) [pekar på Microsoft 365 namnservrar](../../admin/setup/add-domain.md)ska ändringarna av MX-posten inte tillämpas förrän du ändrar ns-posterna så att de pekar på din nya DNS-värd (se steg 2).

- Innan du uppdaterar MX-posten kan du låta användarna veta det datum du planerar att byta e-postmeddelande och den nya e-postleverantör som du planerar att använda. Om dina användare vill flytta sina befintliga Microsoft-e-postmeddelanden till den nya leverantören måste de ta extra steg.

- Följ resterande steg i den här artikeln den dag du ändrar MX-posten.

#### <a name="update-your-domain-mx-and-other-dns-records-if-youre-using-a-custom-domain"></a>Uppdatera domänens MX och andra DNS-poster (om du använder en egen domän)

Om du har bytt dina namnserver (NS) till Microsoft 365 när du konfigurerar din domän måste du konfigurera eller uppdatera MX-posten och andra DNS-poster hos den DNS-värd du planerar att använda och sedan ändra NS-posten till DNS-värden.

Om du inte bytte NS-poster när du konfigurerade din domän när du ändrar MX-posten börjar din e-post direkt med den nya adressen.

Information om hur du ändrar NS-poster finns i [ta bort en domän](../../admin/get-help-with-domains/remove-a-domain.md).

### <a name="save-your-data"></a>Spara dina data

När annulleringen träder i kraft förlorar användarna åtkomst till deras data. Innan du avbryter prenumerationen kan de spara sina OneDrive för företag-eller SharePoint Online-filer till en annan plats. Alla kunddata som du lämnar följer kan tas bort efter 30 dagar och är inte senare än 180 dagar efter annullering.

- Om du vill flytta e-post, kontakter, uppgifter och kalenderinformation till ett annat konto hittar du information i artikeln om att [exportera eller säkerhetskopiera e-post, kontakter och kalender till en Outlook-datafil (.pst)](https://support.microsoft.com/office/14252b52-3075-4e9b-be4e-ff9ef1068f91.aspx).

- Om du vill spara ett dokumentbibliotek eller listinnehåll (till exempel kontakter) från en SharePoint Online-miljö (OneDrive för företag eller gruppwebbplatser) på filresurser eller på en lokal dator kan du läsa mer i artikeln om [manuell migrering av SharePoint Online-innehåll](https://support.microsoft.com/kb/2783484).

### <a name="uninstall-office-optional"></a>Avinstallera Office (valfritt)

Om du har avbrutit ditt abonnemang och inte har [flyttat användare till ett annat abonnemang](move-users-different-subscription.md) som innehåller Microsoft 365 körs Microsoft 365 i läget Nedsatt funktionalitet. När det här inträffar kan användarna endast läsa och skriva ut dokument och Microsoft 365-program visar [olicensierade produkt meddelanden](https://support.microsoft.com/office/0d23d3c0-c19c-4b2f-9845-5344fedc4380.aspx). Se till att användarna [avinstallerar Office](https://support.microsoft.com/office/9dd49b83-264a-477a-8fcc-2fdf5dbf61d8.aspx) från sina datorer för att undvika missförstånd.
