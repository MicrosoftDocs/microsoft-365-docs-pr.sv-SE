# <a name="document-understanding-model-usage-analytics"></a>Dokument förstå modell användnings analys

</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4CSoL]

</br>

Med innehålls Center för Microsoft SharePoint Syntex får du mer information om hur modeller som har publicerats från innehålls centret används. Här ingår en sammanfattning av följande information:

- Var modellerna används
- Hur många filer håller processer över tid

 ![Modell analys](../media/content-understanding/model-analytics.png) </br>

## <a name="total-model-percentage"></a>Total modell procent

   ![Total modell procent](../media/content-understanding/total-model-percentage.png) </br>

I det sammanslagna modell diagrammet i **procent** visas varje publicerad modell som en procent andel av de totala filerna som bearbetas av alla publicerade modeller i innehålls centret.

Varje modell visar alltid **hela**antalet filer som har laddats upp av modellen. En nedsatt fullständighet-taxa kan innebära att det finns problem med antingen modellen eller filerna som analyseras.

## <a name="files-processed-over-time"></a>Filer som behandlas över tid

   ![Bearbetade filer](../media/content-understanding/files-processed-over-time.png) </br>

I diagrammet **bearbetade filer över tids** stapeldiagram visas inte bara antalet filer som har behandlats över tiden för varje modell, men som också visar de dokument bibliotek som modellen användes på.

   ![Liggande stapeldiagram](../media/content-understanding/bar-chart-models.png) </br>
