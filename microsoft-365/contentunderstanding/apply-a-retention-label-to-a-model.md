# <a name="apply-a-retention-label-to-a-document-understanding-model-preview"></a>Använda en bevarande etikett på en dokument förståelses modell (för hands version)

> [!Note] 
> Innehållet i den här artikeln gäller för projekt cortex privat för hands version. [Lär dig mer om Project cortex](https://aka.ms/projectcortex).

</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4CSoL]

</br>

Du kan enkelt använda en [behållnings etikett](https://docs.microsoft.com/microsoft-365/compliance/retention?view=o365-worldwide#retention-labels) på ett dokument för att förstå modellen i Project cortex.

Med behållnings etiketter kan du tillämpa inställningar för behållning för dokument som ska identifieras av dokumentet.  Du kanske till exempel vill att din modell inte bara ska identifiera eventuella förnyelse dokument som laddas upp till ditt dokument bibliotek, men om du också vill att dessa dokument ska tas bort.

Du lägger till en behållnings etikett till dokumentet förstås med modell inställningarna. 

> [!Note]
> För att lagrings etiketterna ska vara tillgängliga för din modell måste de [skapas och publiceras i Microsoft 365 Compliance Center](https://docs.microsoft.com/microsoft-365/compliance/create-apply-retention-labels?view=o365-worldwide#how-to-create-and-publish-retention-labels).
