# <a name="learn-about-document-understanding-models-through-a-sample-model"></a>Lär dig mer om hur du förstår dokument med hjälp av en exempel modell

Microsoft SharePoint Syntex innehåller en exempel modell som du kan använda för att undersöka och ge dig en bättre förståelse för hur du skapar egna modeller. Med exempel modellen kan du också undersöka modell komponenter, till exempel klassificerare, utdrag och förklaringar. Du kan också använda exempelfilerna för att träna modellen.

## <a name="import-the-sample-model"></a>Importera exempel modellen

För att få åtkomst till exempel modellen måste du först importera modellen till ditt innehålls Center.

1. I innehålls Center väljer du **modeller** för att visa listan med modeller.</br>
2. Välj **import Sample Model**på sidan **modeller** .</br>

    ![Importera exempel modell](../media/content-understanding/import-sample-model.png) </br>

3. Leta efter exempel modellen med rubriken *BenefitsChangeNotice. klassificerare* som visas i din modell lista.</br>

    ![Exempel modell](../media/content-understanding/sample-model.png) </br>

4. Välj *BenefitsChangeNotice. klassificerare*för att öppna modellens start sida.</br>
  
     ![Exempel på Start sida](../media/content-understanding/sample-home-page.png)

5. Kontrol lera modellen bättre från start sidan för modellen för att se hur den skapades.
 
- Visa de exempelfiler som används för att träna modellen.
- Undersök modellens enhets utdrag mer noggrant för att se hur exempel modellen konfigurerade förklaringarna.

   ![Exempel på modell utdrag](../media/content-understanding/entity-extractors.png)
