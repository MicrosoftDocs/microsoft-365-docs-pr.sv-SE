## <a name="what-does-end-of-support-mean"></a>Vad innebär *slut på support* ?

Precis som de flesta Microsoft-produkter är PerformancePoint Server 2007 SP3, ProClarity-programmet och SharePoint Server 2007 SP3 en support livs cykel, under vilken Microsoft tillhandahåller nya funktioner, fel korrigeringar och säkerhets uppdateringar. Livs cykeln för en produkt räcker normalt för 10 år från produktens första utgivning. Slutet av livs cykeln kallas produkt Supportens slut. Som ProClarity, PerformancePoint Server och SharePoint Server 2007 har Microsoft inte längre rätt att:
  
- Teknisk support för problem som kan uppstå.
    
- Program korrigeringar för problem som upptäcks och som kan påverka servrarnas stabilitet och användbarhet.
    
- Säkerhets korrigeringar för säkerhets problem som upptäcks och som kan göra servrar och program sårbara för säkerhets brott.
    
- Tids zons uppdateringar.
    
Din installation av ProClarity, SharePoint Server 2007 SP3 och PerformancePoint Server 2007 SP3 fortsätter köras även om supporten har avslut ATS. Vi rekommenderar att du migrerar från dessa program så snart som möjligt.

### <a name="what-if-i-dont-upgrade-now"></a>Vad gör jag om jag inte uppgraderar nu?

Du kan välja att inte uppgradera direkt. Dina befintliga servrar och program kommer att fortsätta köras. Men du kommer inte att få några fler uppdateringar, inklusive säkerhets uppdateringar, eftersom stöd har avslut ATS. Och om det går fel med dina serverprogram kan du inte få hjälp från Microsoft Technical Support.

### <a name="use-power-bi-in-the-cloud-or-on-premises"></a>Använda Power BI (i molnet eller lokalt)

Power BI är en serie företags analys verktyg för att analysera data och dela insikter. Med Power BI kan du använda lokala eller online-datakällor för att skapa interaktiva rapporter och instrument paneler. Personer kan visa och använda rapporter och instrument paneler på sina datorer och mobila enheter.
  
Power BI är inte en del av Microsoft 365 eller SharePoint Server. Det är ett separat erbjudande som innehåller Power BI-skrivbord, Power BI-gateway och Power BI-tjänsten. Power BI är också integrerat med SharePoint Online. Du kan komma igång med Power BI gratis. Baserat på dina behov av data användning och företag kan du senare uppgradera till Power BI Pro med Microsoft 365 E5. Mer information finns i [Vad är Power BI?](https://go.microsoft.com/fwlink/?linkid=841341)
  
### <a name="use-reporting-services-on-premises"></a>Använda repor ting Services (lokalt)

SQL Server repor ting Services erbjuder en robust rapporterings lösning. Du kan konfigurera repor ting Services i antingen enhetligt läge eller i SharePoint-integrerat läge. Du kan använda flera olika verktyg för att skapa rapporter, inklusive Rapportdesignern, rapport verktyget och Power View. Med den senaste versionen av SQL Server kan du även använda SQL Server Mobile rapport Publisher för att leverera rapporter som anpassas till vilken skärm storlek som helst. Det gör att läsarna förbrukar rapporter på sina mobila enheter. Mer information finns i [SQL Server repor ting Services (SSRS): skapa, distribuera och hantera mobila och sid rapporter](https://go.microsoft.com/fwlink/?linkid=841342).
  
### <a name="use-performancepoint-services-on-premises"></a>Använda PerformancePoint-tjänster (lokalt)

PerformancePoint Server 2007 såldes separat från SharePoint Server 2007. Från och med SharePoint Server 2010 är PerformancePoint-tjänster ett tjänst program i SharePoint Server. Därför behöver du inte köpa separata Server licenser eller hård vara för att använda PerformancePoint-tjänster.
  
Om du vill flytta från PerformancePoint Server 2007 till PerformancePoint-tjänster flyttar du till en nyare version av SharePoint Server och konfigurerar PerformancePoint-tjänster. Den version av SharePoint Server som du flyttar till avgör om du kan importera befintligt instrument panels innehåll från PerformancePoint Server 2007 till PerformancePoint Services.
  
- Om du uppgraderar till SharePoint Server 2010 kan du importera PerformancePoint Dashboard-innehåll från PerformancePoint Server 2007 till PerformancePoint Services i SharePoint Server 2010. Mer information finns i [guiden Importera: PerformancePoint Server 2007-innehåll till SharePoint Server 2010](https://go.microsoft.com/fwlink/?linkid=838873).
    
- Om du flyttar till SharePoint Server 2013 eller SharePoint Server 2016 måste du antagligen skapa ett nytt instrument panels innehåll (data källor, rapporter, styrkort och instrument panels sidor).
    
För att komma igång med uppgraderings abonnemanget för PerformancePoint-tjänster, se följande resurser:
  
- [SharePoint Server 2007 slut på support översikt](sharepoint-2007-end-of-support.md)
    
- När du vet vilken version av SharePoint som du flyttar till kan du läsa motsvarande artikel för PerformancePoint-tjänster:
    
  - [Plan för PerformancePoint-tjänster (SharePoint Server 2010)](https://go.microsoft.com/fwlink/?linkid=841363)
    
  - [PerformancePoint Services i SharePoint Server 2013 – Översikt](https://go.microsoft.com/fwlink/?linkid=841551)
    
  - [PerformancePoint Services i SharePoint Server 2016 – översikt](https://go.microsoft.com/fwlink/?linkid=874704)
    
När du uppgraderar till PerformancePoint-tjänster får du flera nya funktioner och förbättringar. PerformancePoint Services erbjuder förbättrade styrkort; nya visualiseringar, till exempel rapporten nedbrytnings träd och KPI-information; fler diagram typer; bättre funktioner för tidsinformations filtrering. och förbättrade hjälpmedels funktioner. Mer information finns i [Nyheter i PerformancePoint Services (SharePoint Server 2010)](https://go.microsoft.com/fwlink/?linkid=841343).
