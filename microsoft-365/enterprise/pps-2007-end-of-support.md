## <a name="what-does-end-of-support-mean"></a>Vad innebär slut på support?

Microsoft-produkter, till exempel PerformancePoint Server 2007 SP3, ProClarity-programvara och SharePoint Server 2007 SP3 har en support livs cykel under vilken Microsoft tillhandahåller nya funktioner, program korrigeringar och säkerhets uppdateringar. Livs cykeln för en produkt räcker i tio år från datumet för produktens första utgivning och slutet av livs cykeln kallas produkt Supportens slut. Eftersom ProClarity, PerformancePoint Server och SharePoint Server 2007 har nått slutet av supporten tillhandahåller Microsoft inte längre:
  
- Teknisk support för problem som kan uppstå
    
- Program korrigeringar för problem som upptäcks och som kan påverka servrarnas stabilitet och användbarhet
    
- Säkerhets korrigeringar för säkerhets problem som upptäcks och som kan göra servrar och program som är utsatta för säkerhets brott
    
- Tids zons uppdateringar
    
Din installation av ProClarity, SharePoint Server 2007 SP3 och PerformancePoint Server 2007 SP3 fortsätter köras även om supporten har avslut ATS. Vi rekommenderar att du migrerar från dessa program så snart som möjligt.

### <a name="what-happens-if-i-dont-upgrade-now"></a>Vad händer om jag inte uppgraderar nu?

Du kan välja att inte uppgradera just nu. Dina befintliga servrar och program kommer att fortsätta köras. Men du får inga fler uppdateringar – inklusive säkerhets uppdateringar – efter att supporten har avslut ATS. Och om det går fel med dina serverprogram kan du inte få hjälp från Microsoft Technical Support.

### <a name="use-power-bi-in-the-cloud-or-on-premises"></a>Använda Power BI (i molnet eller lokalt)

Power BI är en serie företags analys verktyg för att analysera data och dela insikter. Med Power BI kan du skapa interaktiva rapporter och instrument paneler med lokala eller online-datakällor. Personer kan visa och använda rapporter och instrument paneler med deras datorer eller mobila enheter.
  
Power BI ingår inte i Microsoft 365 eller SharePoint Server, men är ett separat erbjudande som innehåller Power BI-datorer, Power BI-gateway och Power BI-tjänsten. Power BI är också integrerat med SharePoint Online. Du kan komma igång med Power BI gratis och beroende på hur data används och företagets behov, uppgradera till Power BI Pro med Microsoft 365 E5. Mer information finns i [Vad är Power BI?](https://go.microsoft.com/fwlink/?linkid=841341)
  
### <a name="use-reporting-services-on-premises"></a>Använda repor ting Services (lokalt)

SQL Server repor ting Services erbjuder en robust rapporterings lösning tillsammans med möjligheten att installera och konfigurera repor ting Services i antingen enhetligt eller SharePoint-integrerat läge. Du kan skapa rapporter med flera verktyg, till exempel Rapportdesignern, rapport verktyget och Power View. Med den senaste versionen av SQL Server kan du även använda SQL Server Mobile rapport Publisher för att leverera rapporter som kan anpassas till vilken skärm storlek som helst, vilket gör det möjligt för din organisation att använda rapporter på sina mobila enheter. Mer information finns i [SQL Server repor ting Services (SSRS): skapa, distribuera och hantera mobila och sid rapporter](https://go.microsoft.com/fwlink/?linkid=841342).
  
### <a name="use-performancepoint-services-on-premises"></a>Använda PerformancePoint-tjänster (lokalt)

Som du vet köpte PerformancePoint Server 2007 separat från SharePoint Server 2007. Från och med SharePoint Server 2010 är PerformancePoint-tjänster ett tjänst program i SharePoint Server. Det innebär att du inte behöver köpa separata Server licenser eller maskin vara för att kunna använda PerformancePoint-tjänster.
  
Om du vill flytta från PerformancePoint Server 2007 till PerformancePoint-tjänster flyttar du till en nyare version av SharePoint Server och konfigurerar PerformancePoint-tjänster. Vilken version av SharePoint Server som du flyttar till avgör om du kan importera befintligt instrument panels innehåll från PerformancePoint Server 2007 till PerformancePoint Services.
  
- Om du uppgraderar till SharePoint Server 2010 kan du importera PerformancePoint Dashboard-innehåll från PerformancePoint Server 2007 till PerformancePoint Services i SharePoint Server 2010. Mer information om hur det här fungerar finns i [guiden Importera: PerformancePoint Server 2007-innehåll till SharePoint Server 2010](https://go.microsoft.com/fwlink/?linkid=838873).
    
- Om du flyttar till SharePoint Server 2013 eller till SharePoint Server 2016 måste du antagligen skapa ett nytt instrument panels innehåll (data källor, rapporter, styrkort och instrument panels sidor).
    
För att komma igång med uppgraderings abonnemanget för PerformancePoint-tjänster, se följande resurser:
  
1. [SharePoint Server 2007 slut på support översikt](sharepoint-2007-end-of-support.md)
    
2. När du vet vilken version av SharePoint som du flyttar till kan du läsa motsvarande artikel för PerformancePoint-tjänster:
    
  - [Plan för PerformancePoint-tjänster (SharePoint Server 2010)](https://go.microsoft.com/fwlink/?linkid=841363)
    
  - [PerformancePoint Services i SharePoint Server 2013 – Översikt](https://go.microsoft.com/fwlink/?linkid=841551)
    
  - [PerformancePoint Services i SharePoint Server 2016 – översikt](https://go.microsoft.com/fwlink/?linkid=874704)
    
När du uppgraderar till PerformancePoint-tjänster får du flera nya funktioner och förbättringar. PerformancePoint Services erbjuder förbättrade styrkort, nya visualiseringar, till exempel nedbrytnings träd och KPI-information, samt fler diagram typer, bättre funktioner för tidsinformation och bättre tillgänglighet. Mer information finns i [Nyheter i PerformancePoint Services (SharePoint Server 2010)](https://go.microsoft.com/fwlink/?linkid=841343).
