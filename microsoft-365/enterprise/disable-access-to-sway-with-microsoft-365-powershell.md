# <a name="disable-access-to-sway-with-powershell-for-microsoft-365"></a>Inaktivera åtkomst till Sway med PowerShell för Microsoft 365

*Denna artikel gäller för både Microsoft 365 Enterprise och Office 365 Enterprise.*

Med ManageSway.ps1 PowerShell-skript kan du visa och inaktivera tjänster i din Microsoft 365 organisation, inklusive Sway. Det här skriptet automatiserar procedurerna som beskrivs i följande avsnitt:
  
- [Visa licenser och tjänster med PowerShell](view-licenses-and-services-with-microsoft-365-powershell.md)
    
- [Inaktivera åtkomst till tjänster med PowerShell](disable-access-to-services-with-microsoft-365-powershell.md)
    
Du måste ladda ned de två filer som är kopplade till skriptet:
  
- Skriptet ManageSway.ps1 vid [https://go.microsoft.com/fwlink/p/?LinkId=785070](https://go.microsoft.com/fwlink/p/?LinkId=785070)
    
- Hjälpfilen för skriptet vid [https://go.microsoft.com/fwlink/p/?LinkId=785072](https://go.microsoft.com/fwlink/p/?LinkId=785072)
