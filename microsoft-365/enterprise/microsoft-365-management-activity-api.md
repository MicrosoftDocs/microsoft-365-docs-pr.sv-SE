# <a name="office-365-management-activity-api"></a>Administrations aktivitets API för Office 365

Microsoft tillhandahåller repor ting Services som du kan använda för att få en samlad transaktions information om din Office 365-klient. [API-modulen för hantering av Office 365](https://docs.microsoft.com/office/office-365-management-api/office-365-management-apis-overview#office-365-management-activity-api) använder en branschstandardiserade RESTful-design och OAuth v2 för autentisering. Det gör det enkelt att börja experimentera med data och insugning till visualiserings verktyg och program. API-funktionen ger en datafeed med information om användare, administratör, åtgärder och säkerhets aktivitet i Office 365. Uppgifterna kan bevaras i reglerings syfte, eller kombineras med data från en lokal infrastruktur eller andra källor. Detta hjälper till att skapa en övervaknings lösning för verksamhet, säkerhet och efterlevnad i hela företaget.

API för hanterings aktivitet i Office 365 ger information om olika åtgärder för användare, administratörer, system och händelser från Office 365 och Azure Active Directory-aktiviteter. API tillhandahåller ett enhetligt gransknings schema med över 10 fält gemensamt för alla tjänster. API gör det möjligt för organisationer att skapa enkla anslutningar mellan händelser och gör det möjligt för nya sätt att ta hand om data.

Dussin tals oberoende program varu tillverkare som samarbetar med Microsoft och har inbyggda lösningar baserade på API: t. Vissa lösningar är bara fokus på Office 365-data och andra källdata från flera moln leverantörer och lokala system för att skapa en enhetlig vy över operationer, säkerhet och efterlevnad. 

Mer information finns i referens för [hanterings AKTIVITETS API för Office-365](https://docs.microsoft.com/office/office-365-management-api/office-365-management-activity-api-reference).
