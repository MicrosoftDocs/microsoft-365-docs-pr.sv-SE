# <a name="internet-sites-in-microsoft-azure-using-sharepoint-server-2013"></a>Internet-webbplatser i Microsoft Azure med SharePoint Server 2013

 Internet webbplatser som använder SharePoint Server 2013 fördelar sig med Azure Infrastructure Services som värd. Den här artikeln innehåller resurser för utformning och implementering av den här lösningen.
  
## <a name="using-azure-infrastructure-services-for-internet-sites"></a>Använda Azure Infrastructure Services för Internet-webbplatser

Microsoft Azure tillhandahåller ett övertygande alternativ för Internet-webbplatser baserade på SharePoint Server 2013. Fördelarna är följande:
  
- Fokusera på att utveckla en fantastisk webbplats i stället för att bygga infrastruktur.
    
- Flexibilitet för att skala din lösning baserat på begäran genom att bygga ut och in.
    
- Betala endast för de resurser du behöver och Använd.
    
- Utnyttja Azure Active Directory för kund konton.
    
- Lägga till funktioner som för närvarande inte är tillgängliga i Microsoft 365, till exempel djupgående rapportering och analys.
