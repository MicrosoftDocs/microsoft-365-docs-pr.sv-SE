# <a name="windows-and-office-deployment-lab-kit"></a>Deployment Lab Kit för Windows och Office

I dessa nedladdningsbara labbar ligger fokus på distribution av Windows 10 och Office 365, samt överväganden kring konfiguration och hantering efter distributionen. Den här utbildningen rekommenderas starkt för organisationer som behöver migrera från Windows 7 eftersom supporten upphört, men gäller även för dig som har Windows 10 och Office 365 plus eller Office 2019. I utbildningen ingår guider för Windows 10, Office 365 ProPlus, Enterprise Mobility + Security och relaterade produkter och tjänster.

Dessa labbar är utformade för att hjälpa dig att planera, testa och verifiera din distribution och hantering av datorer med Windows 10 Enterprise och Office 365 ProPlus. Labbarna omfattar de steg och verktyg som beskrivs i skrivbordsdistributionshjulet, bland annat Microsoft Endpoint Configuration Manager, Desktop Analytics, Office Customization Tool, OneDrive och Windows Autopilot.

Vi rekommenderar att du börjar med att skapa en sandbox-miljö eller isolerad labbmiljö som en del i [skrivbordsdistributionsprocessen](https://www.aka.ms/howtoshift) för Windows 10 och Office 365 ProPlus när du börjar utforska uppdateringarna i distributionsverktygen och testa distributionsrelaterad automatisering.

Labbpaketet är kostnadsfritt att ladda ned och innefattar utvärderingsversioner av programvara.

### <a name="download-the-windows-and-office-deployment-lab-kit"></a>[**Ladda ned Windows and Office Deployment Lab Kit**](https://www.microsoft.com/evalcenter/evaluate-lab-kit)

## <a name="a-complete-lab-environment"></a>**En komplett labbmiljö**

Du får en virtuell labbmiljö som etableras automatiskt, inklusive domänanslutna skrivbordsklienter, domänkontrollant, Internetgateway och en helt konfigurerad ConfigMgr-instans. Utvärderingsversioner av följande produkter ingår:

  - Windows 10 Enterprise, version 1909
  - Windows 7 Enterprise
  - Microsoft Endpoint Configuration Manager, version 1902
  - Windows Assessment and Deployment Kit för Windows 10, version 1903
  - Microsoft Deployment Toolkit
  - Microsoft Application Virtualization (App-V) 5.1
  - Microsoft BitLocker Administration and Monitoring 2.5 SP1
  - Windows Server 2019
  - Microsoft SQL Server 2017

Dessutom är labben utformad för anslutning till utvärderingsversioner av: 

  - Microsoft 365 E5

Eller
  - Office 365 Enterprise E5
  - Enterprise Mobility + Security

## <a name="step-by-step-labs"></a>**Stegvisa labbuppgifter**

Du får utföra flera labbuppgifter för olika distributions- och hanteringsscenarier med detaljerad vägledning. Labbuppgifterna har uppdaterats för de senaste versionerna av Intune och Configuration Manager (version 1910).  

### <a name="device-and-app-readiness"></a>**Beredskap för enheter och appar**

  - Desktop Analytics
  - Företagsläge och webbplatslista för företagsläge för Internet Explorer

### <a name="directory-and-network-readiness"></a>**Katalog- och nätverksberedskap**

  - Grundläggande konfiguration för Azure Active Directory och Microsoft 365
  - Nätverksoptimering med hjälp av leveransoptimering, peer-cachelagring i ConfigMgr och LEDBAT
  - Samhantering av ConfigMgr och Microsoft Intune
  - Fjärråtkomst (VPN)

### <a name="office-and-lob-app-delivery"></a>**Leverans av Office- och LOB-appar**

  - Office 365 ProPlus-distribution med Microsoft Endpoint Configuration Manager
  - Office 365 ProPlus-distribution med Microsoft Intune
  - Distribution och hantering av appar med Microsoft Intune
  - Distribution och egen installation av appar med Microsoft Store för företag
  - Programkonvertering till UWP med Brygga för skrivbordsversion
  - Windows App Certification Kit
  - Åtgärder för webbläsarkompatibilitet med företagsläge för IE

### <a name="user-file-and-settings-migration"></a>**Migrering av användares filer och inställningar**

  - User State Migration Tool i aktivitetssekvenser för datoråterställning och -ersättning i ConfigMgr och MDT
  - Flytt av känd mapp i OneDrive
  - Enterprise State Roaming

### <a name="security-and-compliance"></a>**Säkerhet och efterlevnad**

  - BitLocker-diskkryptering
  - Windows Defender Antivirus
  - Windows Hello för företag
  - Konvertering från BIOS till UEFI för att möjliggöra virtualiseringsbaserad säkerhet
  - Windows Defender Credential Guard
  - Windows Defender Application Guard
  - Windows Defender Exploit Guard
  - Windows Defender-programreglering
  - Microsoft Defender Avancerat skydd

### <a name="os-deployment-and-feature-updates"></a>**Distribution av operativsystem och uppdateringar av funktioner**

  - Skapa operativsystemavbildning
  - Aktivitetssekvenser för OS-distribution i ConfigMgr
      - Bare Metal
      - Återställning
      - Ersättning
      - Uppgradering
  - Aktivitetssekvenser för OS-distribution i MDT
  - Funktionsuppdateringar med hjälp av aktivitetssekvenser för uppgradering i ConfigMgr
  - Windows Autopilot

### <a name="windows-and-office-servicing"></a>**Underhåll av Windows och Office**

  - Hantering av programuppdateringar med Configuration Manager
  - Uppdateringshantering för Office 365 ProPlus i Configuration Manager
  - Hantering av mobila enheter för Windows 10 med Microsoft Intune

### <a name="new-windows-virtual-desktop"></a>**NYTT! Windows Virtual Desktop**
  - Windows Virtual Desktop: förbereda, distribuera, optimera
