# <a name="manage-microsoft-365-with-powershell"></a>Hantera Microsoft 365 med PowerShell

*Denna artikel gäller för både Microsoft 365 Enterprise och Office 365 Enterprise.*

PowerShell för Microsoft 365 är ett kraftfullt Management-verktyg som kompletterar Microsoft 365 Admin Center. Du kan till exempel använda PowerShell Automation för att enkelt hantera flera användar konton och licenser och skapa rapporter.

Välj bland följande avsnitt för att lära dig hur du använder PowerShell för att hantera Microsoft 365:
  
- [**Komma igång**](getting-started-with-microsoft-365-powershell.md)

    Börja här om du inte är bekant med PowerShell för Microsoft 365 och vill installera Microsoft 365-modulerna och ansluta till prenumerationen.

- [**Användar konton, licenser och grupper**](manage-user-accounts-and-licenses-with-microsoft-365-powershell.md)

    Börja här om du vill veta mer om hur du använder automatiserings kommandon för att hantera användar konton, licenser och grupper.

- [**SharePoint**](manage-sharepoint-online-with-microsoft-365-powershell.md)

    Börja här om du vill använda automatiserings kommandon för att hantera SharePoint.

- [**Exchange Online**](https://docs.microsoft.com/powershell/exchange/exchange-online-powershell)

    Börja här om du vill hantera Exchange Online.

- [**Migrera e-post**](use-powershell-for-email-migration-to-microsoft-365.md)

    Börja här om du vill migrera din e-post från befintliga system.

- [**Säkerhets- och efterlevnadscenter**](https://docs.microsoft.com/powershell/exchange/scc-powershell)

    Börja här om du vill hantera säkerhets & funktioner för regelefterlevnad.

- [**Partners för delegerad åtkomst behörighet (DAP)**](manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-dap-p.md)

    Börja här om du vill använda syndikerings-och KRYPTOGRAFIPROVIDER-partners för att hantera dina Microsoft 365-klienttjänster.

- [**Skype för företag online**](manage-skype-for-business-online-with-microsoft-365-powershell.md)

    Börja här för att hantera Skype för företag – online.
