# <a name="manage-microsoft-365-with-powershell"></a>Hantera Microsoft 365 med PowerShell

*Den här artikeln gäller både Microsoft 365 Enterprise och Office 365 Enterprise.*

PowerShell för Microsoft 365 är ett kraftfullt Management-verktyg som kompletterar Microsoft 365 Admin Center. Du kan till exempel använda PowerShell-automatisering för att snabbare hantera flera användar konton och licenser och skapa rapporter. Lär dig hur du använder PowerShell för Microsoft 365-användare och-licenser, Skype för företag – Online, SharePoint Online, Exchange Online och säkerhets & Compliance Center.
  
Välj ämnet utifrån dina behov:
  
- [Komma igång](getting-started-with-microsoft-365-powershell.md)

    Börja här om du inte är bekant med PowerShell för Microsoft 365 och vill installera Microsoft 365-modulerna och ansluta till Microsoft 365-prenumerationen.

- [Användar konton, licenser och grupper](manage-user-accounts-and-licenses-with-microsoft-365-powershell.md)

    Börja här om du har installerat Microsoft 365-modulerna och vill veta mer om hur du använder automatiserings kommandon för att hantera användar konton, licenser och grupper.

- [SharePoint Online](manage-sharepoint-online-with-microsoft-365-powershell.md)

    Börja här om du har installerat Microsoft 365-moduler och vill använda automatiserings kommandon för att hantera SharePoint Online.

- [Exchange Online PowerShell](https://docs.microsoft.com/powershell/exchange/exchange-online/exchange-online-powershell)

    Börja här om du vill använda automatiserings kommandon för att hantera Exchange Online.

- [Migrera e-post](use-powershell-for-email-migration-to-microsoft-365.md)

    Börja här om du har installerat PowerShell 365-modulerna och vill migrera din e-post från befintliga system.

- [Säkerhets- och efterlevnadscenter](https://docs.microsoft.com/powershell/exchange/office-365-scc/office-365-scc-powershell)

    Börja här om du vill använda automatiserings kommandon för att hantera säkerhets & Compliance Center.

- [Partners för delegerad åtkomst behörighet (DAP)](manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-dap-p.md)

    Börja här om du vill använda syndikerings-och KRYPTOGRAFIPROVIDER-partners för att hantera dina Microsoft 365-klienttjänster.

- [Skype för företag Online](manage-skype-for-business-online-with-microsoft-365-powershell.md)

    Börja här om du har installerat PowerShell-moduler och vill hantera Skype för företag – online.
