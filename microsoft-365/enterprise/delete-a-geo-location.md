# <a name="delete-a-satellite-location-in-microsoft-365-multi-geo"></a>Ta bort en satellitplats i Microsoft 365 Multi-Geo

Om du inte längre behöver en satellitplats kan du ta bort den från klientorganisationen SharePoint administrationscentret.

> [!WARNING]
> Alla användardata på satellitplatsen tas bort permanent. Det omfattar allt OneDrive för företag, webbplatser SharePoint postlådor Exchange postlådor, inklusive postlådor Microsoft 365 grupper. Du måste migrera data till en annan satellitplats eller den centrala platsen innan du tar bort satellitplatsen. Den här åtgärden kan inte ångras.

Endast globala administratörer kan ta bort satellitplatser.

![Skärmbild av fler geoadministrationscenter som visar användargränssnittet för Ta bort geoplats](../media/multi-geo-delete-satellite-location.png)

Ta bort en satellitplats

1. Öppna SharePoint administrationscenter

2. Gå till **fliken Geoplatser.**

3. Klicka på den geoplats du vill ta bort på kartan.

4. Klicka på **Ta bort plats.**

5. Bekräfta borttagningen genom att markera kryssrutorna för bekräftelse.

6. Klicka på **Ta bort**.
