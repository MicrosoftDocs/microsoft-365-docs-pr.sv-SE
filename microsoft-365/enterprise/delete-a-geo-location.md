# <a name="delete-a-satellite-location-in-microsoft-365-multi-geo"></a>Ta bort en satellit plats i Microsoft 365 multi-geo

Om du inte längre behöver en satellit plats kan du ta bort den från klient organisationen från administrations centret för SharePoint.

> [!WARNING]
> Alla användar data på satellit platsen tas bort permanent. Detta inkluderar allt OneDrive för företag-innehåll, SharePoint-webbplatser och Exchange-postlådor inklusive Microsoft 365 Group-postlådor. Du måste migrera data till en annan satellit plats eller på Central platsen innan du tar bort satellit platsen. Åtgärden kan inte ångras.

Endast globala administratörer kan ta bort satellit platser.

![Skärm bild av multi-geo Admin Center som visar användar gränssnittet för geo-platsen](../media/multi-geo-delete-satellite-location.png)

Ta bort en satellit plats

1. Öppna administrations centret för SharePoint

2. Gå till fliken **geo platser** .

3. Klicka på den Geo-plats som du vill ta bort på kartan.

4. Klicka på **ta bort plats**.

5. Bekräfta borttagningen genom att markera kryss rutorna.

6. Klicka på **Ta bort**.
