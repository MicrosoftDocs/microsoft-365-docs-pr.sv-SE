# <a name="plan-for-microsoft-365-enterprise"></a>Planera för Microsoft 365 Enterprise

När du flyttar över en stor verksamhet Microsoft 365 företag är det viktigt att planera i förväg och fatta viktiga designbeslut som effektiviserar IT-distributionen och införandet av användare. 

## <a name="planning-with-microsoft-365-fasttrack"></a>Planering med Microsoft 365 FastTrack

[FastTrack för Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) är den bästa metoden för att få hjälp från Microsoft för att planera din Microsoft 365 distribution. FastTrack kan hjälpa dig genom de vanligaste designövervägandena och kan svara på frågor på vägen. 

>[!Note]
>Du kan också få hjälp från en [Microsoft-partner.](https://www.microsoft.com/solution-providers/home)
>

## <a name="do-it-yourself-planning-for-microsoft-365"></a>Gör det själv när du planerar Microsoft 365

För att planera Microsoft 365 på egen hand går du igenom planerings- och designbesluten för dessa områden:

- Din Microsoft 365 klient

  Omfattar planering för nätverksanslutningar till Internet, Microsoft 365 identiteter och integrering med appar, lokalt, Azure och andra element. Gå till [Prenumerationer, licenser, konton och klienter för Microsofts molntjänster för](subscriptions-licenses-accounts-and-tenants-for-microsoft-cloud-offerings.md) mer information.

- Stöd för dina klienter

  Innehåller certifikatbaserad autentisering, hantering av mobila enheter, autentiseringsalternativ och samarbete mellan klientorganisationen. Gå till [Certifikatbaserad autentisering för](microsoft-365-client-support-certificate-based-authentication.md) mer information.

- Stöd för hybrid modern autentisering

  Innehåller planering för modern autentisering när du använder hybridkonfigurationer av viktiga Microsoft 365 produktivitetsprogram. Gå till [Översikt över hybrid modern autentisering och förutsättningar för](hybrid-modern-auth-overview.md) mer information.

- Äldre Office klienter och servrar

  Innehåller migreringsinformation för Office 2007 och Office 2010-klient- och serverprodukter. Mer information [finns i Planera uppgraderingen.](plan-upgrade-previous-versions-office.md)

Du kan också logga in på Microsoft 365-prenumerationen och använda installationsguiderna för [Microsoft 365 och Office 365 tjänster.](setup-guides-for-microsoft-365.md)
