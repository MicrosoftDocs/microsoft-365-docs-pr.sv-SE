# <a name="plan-for-microsoft-365-enterprise"></a>Abonnemang för Microsoft 365 Enterprise

När du flyttar en företags organisation till Microsoft 365 är det viktigt att planera steget före och fatta avgörande beslut som effektiviserar IT-distributionen och användarens antagande. 

## <a name="planning-with-microsoft-365-fasttrack"></a>Planera med Microsoft 365 FastTrack

[FastTrack för microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) är den bästa metoden för att få hjälp från Microsoft för planering av en Microsoft 365-distribution. FastTrack kan hjälpa dig genom de vanligaste design aspekterna och kan besvara frågor på samma sätt. 

>[!Note]
>Du kan också få hjälp från en [Microsoft-partner](https://www.microsoft.com/solution-providers/home).
>

## <a name="do-it-yourself-planning-for-microsoft-365"></a>Gör det själv – planera det själv för Microsoft 365

För att planera för Microsoft 365 själv, gå igenom planerings-och design besluten för dessa områden:

- Din Microsoft 365-klient organisation

  Inkluderar planering för nätverks anslutningar till Internet, dina Microsoft 365-identiteter och integrering med appar, lokala, Azure och andra element. Gå till [prenumerationer, licenser, konton och innehavare för Microsofts moln tjänster](subscriptions-licenses-accounts-and-tenants-for-microsoft-cloud-offerings.md) för mer information.

- Support för dina klienter

  Inkluderar certifikatbaserad inloggningsautentisering, hantering av mobila enheter, autentiseringsalternativ och samarbete mellan klienter. Gå till [certifikatbaserad inloggningsautentisering](microsoft-365-client-support-certificate-based-authentication.md) för mer information.

- Stöd för Hybrid modern inloggningsautentisering

  Inkluderar planering för modern lösenordsautentisering när Hybrid konfigurationer för viktiga Microsoft 365-produktivitets program används. Om du vill ha mer information går du till [hybrid modern verifierings översikt och förutsättningar](hybrid-modern-auth-overview.md) .

- Äldre Office-klienter och-servrar

  Innehåller information om migrering för Office 2007 och Office 2010-klient och Server produkter. Gå till [Planera din uppgradering](plan-upgrade-previous-versions-office.md) för mer information.

Du kan även logga in på ditt Microsoft 365-abonnemang och använda [installations guiderna för microsoft 365 och Office 365-tjänsterna](setup-guides-for-microsoft-365.md).
