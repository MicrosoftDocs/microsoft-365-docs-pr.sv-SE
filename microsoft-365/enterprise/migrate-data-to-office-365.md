# <a name="migrate-your-organization-data-to-microsoft-365-enterprise"></a>Migrera organisationens data till Microsoft 365 Enterprise

## <a name="ready-to-migrate-your-data-to-microsoft-365"></a>Är du redo att migrera dina data till Microsoft 365?

Våra [grundläggande instruktioner hjälper dig](https://support.office.com/article/Set-up-Office-365-for-business-6a3a29a0-e616-4713-99d1-15eda62d04fa) att få alla att använda Microsoft 365-tjänsterna och -programmen på kortast möjliga tid. Då hoppar vi över hybridscenarion och mer avancerade metoder för migrering till Microsoft 365. 
  
Om du behöver hjälp med att konfigurera Microsoft 365 är [FastTrack](https://fasttrack.microsoft.com/office) det enklaste sättet att distribuera Microsoft 365. Du kan också logga in och använda installationsguiderna för [Microsoft 365-tjänster.](setup-guides-for-microsoft-365.md)

## <a name="migrate-email-to-microsoft-365"></a>Migrera e-post till Microsoft 365

- Läs mer om olika sätt [att migrera e-post till Microsoft 365.](https://support.office.com/article/Ways-to-migrate-multiple-email-accounts-to-Office-365-0a4913fe-60fb-498f-9155-a86516418842)
- Hitta alternativa sätt [för personer i organisationen att migrera sina egna e-postmeddelanden, kontakter och kalendrar.](https://support.office.com/article/Migrate-email-and-contacts-to-Office-365-for-business-a3e3bddb-582e-4133-8670-e61b9f58627e)
