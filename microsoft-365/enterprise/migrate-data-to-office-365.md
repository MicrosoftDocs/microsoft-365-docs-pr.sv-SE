# <a name="migrate-your-organization-data-to-microsoft-365-enterprise"></a>Migrera organisationens data till Microsoft 365 Enterprise

## <a name="ready-to-migrate-your-data-to-microsoft-365"></a>Är du redo att migrera dina data till Microsoft 365?

Våra [grundläggande inställnings instruktioner](https://support.office.com/article/Set-up-Office-365-for-business-6a3a29a0-e616-4713-99d1-15eda62d04fa) hjälper dig att få alla att använda dina Microsoft 365-tjänster och-program på kortast möjliga tid. Detta hoppar över hybrid scenarier och de mer avancerade metoderna för migrering till Microsoft 365. 
  
Om du vill få hjälp med att komma igång med Microsoft 365 är [FastTrack](https://fasttrack.microsoft.com/office) det enklaste sättet att distribuera Microsoft 365, men du kan även logga in och använda [installations guider för Microsoft 365-tjänsterna](setup-guides-for-microsoft-365.md).
