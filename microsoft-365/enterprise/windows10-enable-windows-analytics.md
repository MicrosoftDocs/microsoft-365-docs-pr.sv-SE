# <a name="step-4-monitor-device-health-and-compliance"></a>Steg 4: Övervaka enhetens hälsa och efterlevnad

![Fas 3: Windows 10 Enterprise](../media/deploy-foundation-infrastructure/win10enterprise_icon-small.png)

Windows Analytics använder diagnostikdata för att tillhandahålla omfattande och användbar information som hjälper dig att få djup insikt i driftseffektiviteten och hälsan hos Windows 10-enheter i din miljö.

Funktionen Enhetshälsa i Windows Analytics ger proaktiva insikter för att identifiera och åtgärda problem som påverkar slutanvändare. Den här nya tjänsten använder diagnostikdata för att ge sådana insikter utan ytterligare infrastrukturkrav. Proaktivt åtgärda slutanvändarproblem gör att du kan minska supportkostnaderna och förbättra effektiviteten. Leta efter ytterligare funktioner som ska släppas snart, vilket kommer att förbättra funktionerna och värdet av denna nya tjänst.

Se [Övervaka hälsotillståndet för enheter med Enhetshälsa](https://docs.microsoft.com/windows/deployment/update/device-health-monitor) om du vill veta mer, komma igång och använda Enhetshälsa.

Som en interimskontrollpunkt kan du se [avslutningskriterierna](windows10-exit-criteria.md#crit-windows10-step4) som motsvarar det här steget.
