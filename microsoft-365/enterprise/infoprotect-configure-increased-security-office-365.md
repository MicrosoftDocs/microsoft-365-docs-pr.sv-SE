# <a name="step-3-configure-increased-security-for-microsoft-365"></a>Steg 3: Konfigurera ökad säkerhet för Microsoft 365

*Det här steget är obligatoriskt och gäller både E3- och E5-versionerna av Microsoft 365 Enterprise*

![Fas 6: Informationsskydd](../media/deploy-foundation-infrastructure/infoprotection_icon-small.png)

Om du vill vara säker på att din Microsoft 365-prenumeration och tillhörande data skyddas mot skadliga hot redan från början och framåt konfigurerar du följande:

- [Ställ in principer för hantering av hot](https://docs.microsoft.com/office365/securitycompliance/tenant-wide-setup-for-increased-security#tune-threat-management-policies-in-the-microsoft-365-security-center)
- [Ytterligare inställningar för hela Exchange Online-klientorganisationen](https://docs.microsoft.com/office365/securitycompliance/tenant-wide-setup-for-increased-security#configure-additional-exchange-online-tenant-wide-settings)
- [Delningsprinciper för hela klientorganisationen i administrationscentret för SharePoint](https://docs.microsoft.com/office365/securitycompliance/tenant-wide-setup-for-increased-security#configure-tenant-wide-sharing-policies-in-sharepoint-admin-center)
- [Inställningar i Azure Active Directory (Azure AD)](https://docs.microsoft.com/office365/securitycompliance/tenant-wide-setup-for-increased-security#configure-settings-in-azure-active-directory)

När det är konfigurerat kan du få information om säkerhetsstatusen från:

- [Instrumentpaneler och rapporter i Microsoft Security Center](https://docs.microsoft.com/office365/securitycompliance/tenant-wide-setup-for-increased-security#view-dashboards-and-reports-in-the-security-and-compliance-centers)
- [Microsoft Secure Score](https://docs.microsoft.com/office365/securitycompliance/microsoft-secure-score)

En extra säkerhetsfunktion är [Office 365 Advanced Threat Protection (ATP)](https://docs.microsoft.com/office365/securitycompliance/office-365-atp) som hjälper din organisation att samarbeta säkrare genom att:

- Skydda [länkar](https://docs.microsoft.com/office365/securitycompliance/atp-safe-links) och [bifogade filer](https://docs.microsoft.com/office365/securitycompliance/atp-safe-attachments) i e-post. 
- Tillhandahålla information om bedrägerier och anti-phishing för e-post i Exchange Online och [filer i SharePoint Online, OneDrive för företag och Microsoft Teams](https://docs.microsoft.com/office365/securitycompliance/atp-for-spo-odb-and-teams). 

Office 365 ATP är endast tillgängligt med Microsoft 365 E5.
