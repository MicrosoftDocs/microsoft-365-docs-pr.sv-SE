# <a name="getting-started-with-powershell-for-microsoft-365"></a>Komma igång med PowerShell för Microsoft 365

Lär dig varför PowerShell för Microsoft 365-färdigheter är avgörande för att hantera Microsoft 365, hur du ansluter till ditt Microsoft 365-abonnemang, skapar rapporter och får mer information och hjälp från Microsoft 365 Community.

- [Ansluta till alla Microsoft 365-tjänster i ett enda fönster i Windows PowerShell](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)
