# <a name="get-started-with-powershell-for-microsoft-365"></a>Börja använda PowerShell för Microsoft 365

*Denna artikel gäller för både Microsoft 365 Enterprise och Office 365 Enterprise.*

Du kan använda kommandon och skript i PowerShell för Microsoft 365 hantera Microsoft 365 och effektivisera ditt dagliga arbete. Använd följande information för att ta reda på varför PowerShell för Microsoft 365 är avgörande för hantering av Microsoft 365, hur du ansluter till din Microsoft 365-prenumeration och skapar rapporter och var du kan få mer information från Microsoft 365-communityn.
  
Välj bland följande avsnitt:
  
- [**Varför du behöver använda PowerShell för Microsoft 365**](why-you-need-to-use-microsoft-365-powershell.md)
    
    Börja här om du är nybörjare i PowerShell för Microsoft 365. Lär dig varför du ska använda PowerShell för Microsoft 365.
    
- [**Ansluta till Microsoft 365 med PowerShell**](connect-to-microsoft-365-powershell.md)
    
    Börja här för att ansluta till Microsoft 365-prenumerationen med hjälp av PowerShell för Microsoft 365 och utföra administrativa uppgifter från kommandoraden.
    
- [**Ansluta till alla Microsoft 365-tjänster i ett enda PowerShell-fönster**](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)
    
    Du kan hantera Microsoft 365 i separata fönster för Skype för företag Online, SharePoint Online, Microsoft Exchange Online och Microsoft 365 och licenser. Eller så kan du hantera dem alla från ett enda fönster. I den här artikeln förklaras hur.
    
- [**Använda PowerShell för att skapa rapporter i Microsoft 365**](use-windows-powershell-to-create-reports-in-microsoft-365.md)
    
    Börja här om du har installerat PowerShell för Microsoft 365-moduler och vill lära dig mer om att snabbt använda automatiseringskommandon för att skapa rapporter.
    
- [**Cmdlet-referenser för Microsoft 365-tjänster**](cmdlet-references-for-microsoft-365-services.md)
    
    Läs mer om cmdletarna för PowerShell för Microsoft 365-moduler.
    
- [**Microsoft 365-communityresurser för PowerShell**](microsoft-365-powershell-community-resources.md)
    
    Börja här för att ansluta till PowerShell-communityn och få mer information om hur du använder PowerShell för Microsoft 365.
