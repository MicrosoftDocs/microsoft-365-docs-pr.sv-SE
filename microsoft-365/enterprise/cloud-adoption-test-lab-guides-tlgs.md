# <a name="test-microsoft-365-with-test-lab-guides-tlgs"></a>Testa Microsoft 365 med testlabb guider (Molntjänsterhttps://aka.MS/k53sfj)

Molntjänsterhttps://aka.MS/k53sfj hjälper dig att snabbt lära dig mer om Microsoft-produkter. De är bra om du behöver utvärdera en teknik eller en konfiguration innan du bestämmer om det är rätt för dig och innan du påbörjar design, planering och distribution för användare. Den "Jag har byggt det själv och det fungerar" med praktisk upplevelse kan du förstå distributions kraven för en ny produkt eller lösning så att du bättre kan planera för att vara med i produktionen.
  
Molntjänsterhttps://aka.MS/k53sfj skapar också representativa miljöer för utveckling och testning av program, även kallat utvecklings-och test miljöer.
  
![Test laboratorie guider i Microsoft Cloud](../media/24ad0d1b-3274-40fb-972a-b8188b7268d1.png)
  
## <a name="microsoft-365-devtest-environment"></a>Microsoft 365 dev/test miljö

Använd dessa artiklar för att bygga din Microsoft 365 dev/test miljö:
  
- [Den enkla baskonfigurationen](lightweight-base-configuration-microsoft-365-enterprise.md)
    
    Skapa en Microsoft 365 Enterprise, E5 prov abonnemang.

- [Baskonfiguration för simulerat företag](simulated-ent-base-configuration-microsoft-365-enterprise.md)
    
    Skapa ett förenklat intranät som körs i Microsoft Azure Infrastructure Services och en Microsoft 365 Enterprise, E5 prov abonnemang. 

    Det här är valfritt och krävs om du vill bygga en simulerad företags konfiguration för Hybrid identitet.
    
Mer Molntjänsterhttps://aka.MS/k53sfj som gäller både Office 365 och Microsoft 365 finns i [test laboratorie guider](m365-enterprise-test-lab-guides.md).
