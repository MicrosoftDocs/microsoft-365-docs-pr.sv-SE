# <a name="cloud-services-roadmap-for-microsoft-365"></a>Översikt över molntjänster för Microsoft 365

Distribuera de molntjänster som bäst passar organisationens behov för att dra nytta av kreativitet, samarbete och produktivitet i Microsoft 365 för företag.

Om du har befintliga serverbaserade distributioner för Exchange, SharePoint eller Skype för företag och vill migrera hela organisationen till Microsoft 365 [](client-server-software-roadmap-microsoft-365.md)för företag kan du gå till översikten över klient- och serverprogramvaran. Den här översikten Microsoft Office våra klientprodukter, lokala Office Server-produkter och Microsoft &ndash; Windows-baserade enheter.

## <a name="deploy"></a>Distribuera

Så här distribuerar du dina molntjänster:

- [Förbereda dina tjänster](configure-services-and-applications.md)
- [Migrera lokala data till Microsoft 365](migrate-data-to-office-365.md)
- Konfigurera dina molntjänster för användarna
  - [Exchange Online](/Exchange/exchange-online)
  - [SharePoint Online](/sharepoint/sharepoint-online)
  - [Skype för företag](/SkypeForBusiness/skype-for-business-online)
  - [Microsoft Teams](/MicrosoftTeams/Teams-overview)
  - [Yammer](https://support.office.com/article/e1464355-1f97-49ac-b2aa-dd320b179dbe)
- [Utbilda dina användare](/office365/admin/admin-overview/get-started-with-office-365#training-resources-for-your-users)

## <a name="how-microsoft-does-cloud-services-for-microsoft-365"></a>Hur Microsoft använder molntjänster för Microsoft 365

För information om hur Microsoft IT har distribuerat eller hanterar Microsoft 365 molntjänster:

1. Gå till [Microsoft IT Showcase.](https://www.microsoft.com/itshowcase)
2. Välj **Sök innehåll**.
3. Välj **IT-Pro** **under** **Målgrupp** under Förfina resultat och välj **sedan** en molntjänst under Produkt.
