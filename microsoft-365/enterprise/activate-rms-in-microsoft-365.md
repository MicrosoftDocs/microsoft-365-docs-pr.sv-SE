# <a name="activate-rights-management-in-the-admin-center"></a>Aktivera Rights Management i administrations centret

Du måste aktivera Rights Management-tjänsten (RMS) innan du kan använda IRM-funktionerna (Information Rights Management) i Microsoft 365-program och-tjänster. När du har aktiverat RMS kan organisationen börja skydda viktiga dokument och e-postmeddelanden med Azure RMS. Denna informations skydds lösning kan skydda alla filtyper och integreras med klient program som Excel, Microsoft Word och andra, Exchange Online och SharePoint Online och servrar som Microsoft Exchange och Microsoft SharePoint.
  
> [!TIP]
> Om du inte är säker på om du behöver rättighets hantering kontrollerar du om din organisation har ett eller flera av [dessa affärs problem eller krav](https://docs.microsoft.com/azure/information-protection/what-is-azure-rms#business-problems-solved-by-azure-rights-management). 
  
Använd de här länkarna om du vill ha mer information om RMS:
  
- Om du vill veta mer om RMS läser du [Vad är Azure Rights Management](https://docs.microsoft.com/rights-management/understand-explore/what-is-azure-rms).

- Om du är nybörjare på RMS läser du [Översikt över Azure Rights Management](https://docs.microsoft.com/rights-management/understand-explore/azure-rights-management).

- En översikt över distributions stegen finns i [kart kartan för Azure Rights Management](https://docs.microsoft.com/rights-management/plan-design/deployment-roadmap).

- Anvisningar om hur du aktiverar RMS för Microsoft 365 finns i [aktivera skydds tjänsten från Azure information Protection](https://docs.microsoft.com/azure/information-protection/activate-service).
