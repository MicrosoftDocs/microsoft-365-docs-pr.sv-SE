# <a name="cmdlet-references-for-microsoft-365-services"></a>Cmdlet-referenser för Microsoft 365-tjänster

*Denna artikel gäller för både Microsoft 365 Enterprise och Office 365 Enterprise.*

Den här artikeln innehåller cmdlet-referenser för de Microsoft 365-tjänsterna och anslutningsinstruktionerna för Microsoft 365 tjänst som PowerShell har stöd för.

> [!NOTE]
> Om du vill ansluta till alla tjänster på en gång Anslut du [till Microsoft 365 tjänster i ett Windows PowerShell fönster.](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)

Instruktioner Microsoft 365 PowerShell-anslutning finns i Anslut [för Microsoft 365 med PowerShell.](connect-to-microsoft-365-powershell.md)

Anslutningsinstruktioner för Exchange Online PowerShell finns i Anslut [till Exchange Online PowerShell.](/powershell/exchange/connect-to-exchange-online-powershell)

Mer information finns i [Rapportering av cmdlets i Exchange Online.](/powershell/exchange/exchange-online-powershell)
