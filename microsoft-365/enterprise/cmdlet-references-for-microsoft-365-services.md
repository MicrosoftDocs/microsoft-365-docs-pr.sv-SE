# <a name="cmdlet-references-for-microsoft-365-services"></a>Cmdlet-referenser för Microsoft 365-tjänster 

*Denna artikel gäller för både Microsoft 365 Enterprise och Office 365 Enterprise.*

Den här artikeln innehåller cmdlet-referenser för olika Microsoft 365-tjänster och anslutningsinstruktioner för varje Microsoft 365-tjänst som PowerShell har stöd för.

> Information om hur du ansluter till alla tjänster på en gång [finns i Ansluta till alla Microsoft 365-tjänster i ett enda Windows PowerShell-fönster.](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)
