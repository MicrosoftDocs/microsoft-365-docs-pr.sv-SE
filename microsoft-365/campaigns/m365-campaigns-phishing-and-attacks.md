# <a name="protect-yourself-against-phishing-and-other-attacks"></a>Skydda dig mot nätfiske och andra attacker

Microsoft 365 har många skydd för att bevaka attacker, men det finns saker du kan göra själv för att begränsa och minimera risken för onlineattacker.

- **Skräppost** Det finns många orsaker till att du kan få skräppost och inte all skräppost är likadan. Du kan dock minska risken för attacker genom att filtrera bort skräppost.
- **Nätfiske** En nätfiskebedrägeri är ett e-postmeddelande som verkar legitimt, men som är ett försök att komma åt din personliga information eller stjäla dina pengar.
- **Förfalskning** Bedragarna kan också använda en teknik som kallas förfalskning, så att det verkar som om du har fått ett e-postmeddelande från dig själv. 
- **Skadlig** programvara är skadlig programvara som kan installeras på datorn, vanligtvis när du har klickat på en länk eller öppnat ett dokument från ett e-postmeddelande. Det finns olika typer av skadlig programvara (till exempel utpressningstrojaner, när datorn tas över), men du vill inte ha någon av dem. 

## <a name="best-practices"></a>Metodtips

### <a name="reduce-spam-mail"></a>Minska skräppost

Följ de [här 10 tipsen om hur du kan minska skräppost.](https://support.microsoft.com/en-us/office/10-tips-on-how-to-help-reduce-spam-55f756e8-688b-41c3-a086-8f68ccc592f6)

### <a name="report-it"></a>Rapportera det!

Rapportera nätfiske eller andra e-postmeddelanden som du får. Markera meddelandet och välj **Rapportmeddelande i** menyfliksområdet.
Här finns mer information om rapportering av [skräppost- och nätfiskemeddelanden.](https://support.office.com/article/Use-the-Report-Message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2)

### <a name="avoid-phishing"></a>Undvik nätfiske

- Svara aldrig på e-postmeddelanden där du uppmanas att skicka personlig information eller kontoinformation.
- Om du får ett e-postmeddelande som ser misstänkt ut eller frågar om den här typen av information ska du aldrig klicka på länkar som tar dig till ett företags webbplats.
- Öppna aldrig en bifogad fil i ett e-postmeddelande som ser misstänkt ut.
- Om e-postmeddelandet ser ut att komma från ett företag kontaktar du företagets kundtjänst via telefon eller webbläsare för att se om e-postmeddelandet är legitimt.
- Sök på webben efter ämnesraden i e-postmeddelandet följt av ordet bluff för att se om någon annan har rapporterat bedrägeriet.

Läs om fem vanliga typer av bedrägerier [med missbruk, nätfiske eller förfalskning.](https://support.office.com/article/Deal-with-abuse-phishing-or-spoofing-in-Outlook-com-0d882ea5-eedc-4bed-aebc-079ffa1105a3)

### <a name="make-sure-your-emails-look-legitimate-to-others"></a>Kontrollera att dina e-postmeddelanden ser legitima ut för andra

Hjälp dina kunder att lita på din kommunikation genom att lägga till en digital signatur som bevisar att den kommer från dig. Visa [Säkra meddelanden med hjälp av en digital signatur.](https://support.office.com/article/secure-messages-by-using-a-digital-signature-549ca2f1-a68f-4366-85fa-b3f4b5856fc6)
