# <a name="share-files-and-videos"></a>Dela filer och videor

När du behöver styra vem som kan visa och vem som kan redigera dina filer måste du lagra filerna på en säker plats där du kan se till att behörigheterna tillämpas på rätt sätt. Du kan Microsoft Teams lagra filer och sedan dela filer inom eller utanför ditt företag, öva eller kampanj genom att använda Microsoft Teams eller genom att skicka SharePoint länkar. Om du skickar en länk i stället för en e-postbilaga betyder det att du vet vem som visar och ändrar filerna och att de inte kan visas eller ändras utan din tillåtelse.

![Diagram över Microsoft Teams filfönster med fliken Filer och hämta länk på menyn](../media/m365-democracy-teams-sharefiles.png)

Med dina filer i Microsoft Teams och SharePoint kan du också arbeta med filerna tillsammans och granska varandras ändringar. Använd Microsoft Teams att dela filer i ett företag, en övning eller kampanj. Om du behöver dela externt med personer utanför organisationen kan du lägga till dem som gäster i ett team eller skicka dem en säker SharePoint länk.

Du kan också använda Microsoft Stream för att lagra och dela videor internt. De här videoklippen är inte offentliga för att visa dem, så de är perfekta för interna kampanjmeddelanden.

## <a name="best-practices"></a>Metodtips

Använd de här metoderna för att dela filer och videor på ett säkert sätt:

1. Lagra filer i Microsoft Teams eller SharePoint, och se till att bara de personer som behöver åtkomst till dessa filer har dem.
2. När du vill dela ska du inte bifoga filer i ett e-postmeddelande. Välj i stället **Hämta länk från** Microsoft Teams eller SharePoint och skicka länken i ett e-postmeddelande.
3. Om du vill dela en fil externt lägger du till användaren som gäst i teamet eller använder SharePoint för att få en säker länk för att dela just den filen.
4. Använd Microsoft Stream som värd för videor som du vill att kampanjen ska se.
5. Använd Microsoft Teams eller SharePoint för att lagra videofiler som du behöver i din grupp för att samarbeta eller dela.
