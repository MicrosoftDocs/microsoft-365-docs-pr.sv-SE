# <a name="set-up-meetings"></a>Skapa möten

Möten i Microsoft Teams innehåller ljud, video och delning. Eftersom de är online har du alltid ett mötesrum (utan att behöva ett rum eller projektor!), även om personalen är geografiskt fördelad eller arbetar på distans. Microsoft Teams möten är ett bra sätt att samlas med personalen både inom och utanför organisationen. Du behöver inte vara medlem i din organisation eller ha ett konto för att ansluta till ett möte. Du kan schemalägga och köra onlinemöten med Microsoft Teams. Under ett möte kan du dela din skärm, dela filer, tilldela uppgifter och mycket mer. Politiska kampanjer kan omfatta personal, frivilliga och gäster utanför organisationen i mötet. Små företag eller metoder kan träffa sin personal eller träffa kunder eller partners över Microsoft Teams.

[![En illustration av två användare i ett möte](../media/HostOnlineMeeting-thumb-358x201.png)](https://go.microsoft.com/fwlink/?linkid=2078712)

Ladda ned en infografik [i PDF](https://go.microsoft.com/fwlink/?linkid=2078712) [eller PowerPoint](https://go.microsoft.com/fwlink/?linkid=2079515) för att få en snabb överblick över hur du kan ansluta till eller vara värd för ett onlinemöte med Microsoft Teams.

## <a name="best-practices"></a>Metodtips

Följ de här metodtipsen för onlinemöten:

- Schemalägg dina onlinemöten direkt i Microsoft Teams. Du kan välja ett team och en kanal Teams automatiskt bjuda in deltagarna i det teamet eller kanalen.
- Behöver du ett improviserad möte? Om du chattar privat väljer du Möte  nu för att starta ett video- eller ljudsamtal med den person du chattar med.
