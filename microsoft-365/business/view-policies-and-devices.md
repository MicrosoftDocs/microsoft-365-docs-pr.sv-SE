# <a name="view-policies-and-devices"></a>Visa principer och enheter

## <a name="view-device-policies"></a>Visa principer för enheter

1. Logga in på [Microsoft 365 Business](https://portal.office.com) som global administratör. 
    
2. Välj **enheter** på vänster navigeringsfält \> **principer** \> **Lägg till**.
    
    På den här sidan kan du skapa, redigera, ändra målgruppen eller ta bort en princip.
    
    ![Screenshot of the Policies page](media/27ebb1d3-d04b-4221-a13f-8583045b5077.png)
  
## <a name="view-device-actions"></a>Visa enhetsåtgärder

1. Logga in på [Microsoft 365 Business](https://portal.office.com) som global administratör. 
    
2. Från kortet **Enhetsåtgärder** i administratörscentret väljer du **Enhetsåtgärder** för att öppna sidan **Enhetsåtgärder**. 
    
    På den här sidan kan du välja en eller flera enheter och ta bort företagsdata. För Windows 10-enheter som du har angett skyddsinställningar för kan du också välja att återställa enheten till fabriksinställningarna.
    
    ![Device actions page.](media/6d2ad0c4-9c96-4489-ab93-c4e38e317d45.PNG)
