# <a name="pre-requisites-for-protecting-data-on-devices-with-microsoft-365-business"></a>Förutsättningar för att skydda data på enheter med Microsoft 365 Business

Det första steget för att införa Microsoft 365 Business i din organisation är att kontrollera att förutsättningarna är uppfyllda.
  
## <a name="pre-requisites"></a>Förutsättningar

- Windows-enheter måste köras med Windows 7 Professional, Windows 8 Pro eller Windows 8.1 Pro.
    
    [Uppgradera Windows-enheter till Windows Pro Creators Update](upgrade-to-windows-pro-creators-update.md)
    
- Ta bort enheter från mobila hanteringslösningar (Mobile Iron, AirWatch med mera). Alla personer i organisationen kommer att registreras för mobil hantering i Microsoft 365 Business.
    
- Apple iOS 8.0 och senare.
    
    Google Android 4.0 och senare (inklusive Samsung KNOX Standard 4.0 och senare). Mer information finns i artikeln om [enheter med stöd för Intune](https://go.microsoft.com/fwlink/p/?linkid=852307).
    
- Om befintliga Office-program installerats på användardatorer, hittar du information om stegen som du kan behöva ta innan du kan konfigurera Microsoft 365 Business att installera Office 2016 på användarnas datorer i artikeln om att [förbereda för installation av Office-klienten](prepare-for-office-client-deployment.md).
