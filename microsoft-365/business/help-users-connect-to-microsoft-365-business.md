# <a name="help-users-connect-to-microsoft-365-business"></a>Hjälpa användare att ansluta till Microsoft 365 Business

Standardprinciperna för enheten är redo att användas när installationen är slutförd. Principerna används när användare, som laddat ned Office-program på sina mobila enheter, loggar in med sitt arbets- eller skolkonto. Det återstår en del arbete, exempelvis uppgradering under drift, innan principerna kan användas på Windows-datorer. Ta del av, och förmedla, följande avsnitt för att hjälpa användarna att komma igång och bli anslutna.
  
[Konfigurera Windows-enheter för Microsoft 365 Business-användare](set-up-windows-devices.md)
  
[Använda stegvisa anvisningar för att lägga till AutoPilot-enheter och -profiler](add-autopilot-devices-and-profile.md)
  
[Konfigurera mobila enheter för Microsoft 365 företag-användare](set-up-mobile-devices.md)
