# <a name="multi-factor-authentication"></a>Multifaktorautentisering

MFA (Multifaktor authentication) kräver att användare loggar in med en andra metod för att verifiera sin identitet med ett telefonsamtal eller med en autentiseringsapp.

## <a name="set-up-mfa-in-the-microsoft-365-admin-center"></a>Konfigurera MFA i administrationscentret för Microsoft 365

[![Etikett som gör att du kan se att administrationscentret ändras och mer information finns på aka.ms/aboutM365preview.](../media/m365admincenterchanging.png)](https://docs.microsoft.com/office365/admin/microsoft-365-admin-center-preview)

1. Logga in på [Microsoft 365 administrationscenter](https://admin.microsoft.com) med hjälp av dina globala administratörsuppgifter. 
2. Välj **Installationsprogrammet**i den vänstra navigeringsfältet .
3. På sidan Installationsprogrammet väljer du **Visa** på **MFA-kortet (Multi Factor Authentication).**
4. På sidan **Aktivera MFA** väljer du **Kom igång**eller **Hantera** om du redan har konfigurerat MFA och vill göra ändringar. 

  :::image type="content" source="../media/turnonmfa.png" alt-text="Skärmbild av aktivera MFA-sida.":::

5. Kontrollera både eller en av **Kräv multifaktorautentisering för administratörer** **på säkerhetspanelen För att stärka inloggningen** och välj sedan **Skapa princip**.
