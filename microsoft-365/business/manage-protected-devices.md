# <a name="manage-protected-devices-with-microsoft-365-business"></a>Hantera skyddade enheter med Microsoft 365 Business

När en användare har anslutit sin enhet till organisationen genom att logga in med sitt arbetskonto på enheten eller office-appar på enheten kan du skydda enheten med de principer du konfigurerar. Anslutna enheter visas på sidan **Enheter.** Om en enhet försvinner eller blir stulen kan du ta bort företagsdata från den här sidan. Du kan även återställa Windows 10-enheter till deras fabriksinställning för att ta bort alla anpassade inställningar. 

Mer information om de uppgifter som du kan använda för att hantera enheter finns i följande artiklar: 
  
[Ta bort företagsdata från enheter](remove-company-data.md)
  
[Återställa Windows 10-enheter till fabriksinställningarna](reset-devices-to-factory-settings.md)
