# <a name="manage-protected-devices-with-microsoft-365-business"></a>Hantera skyddade enheter med Microsoft 365 Business

När en användare ansluter sin enhet till din organisation genom att logga in på sitt arbetskonto eller ett Office-program, kan vi skydda enheten med de principer du har konfigurerat och som visas på den sidan **Enheter**. Om en enhet försvinner eller blir stulen kan du ta bort företagsdata från den här sidan. Du kan även återställa Windows 10-enheter till deras fabriksinställning för att ta bort alla anpassade inställningar. Mer information om hur du kan hantera enheter finns i följande artiklar: 
  
[Ta bort företagsdata från enheter](remove-company-data.md)
  
[Återställa Windows 10-enheter till fabriksinställningarna](reset-devices-to-factory-settings.md)
