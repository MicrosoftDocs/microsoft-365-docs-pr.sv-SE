# <a name="microsoft-365-business-frequently-asked-questions"></a>Vanliga och frågor svar om Microsoft 365 Business

## <a name="general"></a>Allmänt

### <a name="what-is-microsoft-365-business"></a>Vad är Microsoft 365 Business? 
Microsoft 365 är en integrerad lösning som sammanför bäst i klassen produktivitetsverktyg, säkerhets- och hanteringskapacitet för enheten för små till medelstora företag. 

**En heltäckande uppsättning Affärsverktyg för produktivitet och samarbete** 
* Word, Excel, PowerPoint, Outlook, OneNote, Publisher och Access 
* Exchange, OneDrive, Skype for Business, Microsoft team SharePoint 
* Business apps från Office (bokningar, Outlook Customer Manager, MileIQ<sup>[1](#footnote1)</sup>, Microsoft listor<sup>[1](#footnote1)</sup>, Microsoft anslutningar<sup>[1](#footnote1)</sup>, Microsoft fakturering<sup>[1](#footnote1)</sup>) 
 
**Företagsverktyg enhetsfunktioner för hantering och säkerhet** 
- Ger skydd mot externa hot som nätfiske & sofistikerade skadliga program med Office 365 Advanced Threat Protection och Windows Defender utnyttja Guard 
-  Hjälper till att kontrollera & hantera hur känslig information är tillgänglig och överförs med Office 365 förhindra dataförlust principer (ska vara tillgängliga i sommaren 2018) och Azure Information Protection Plan 1 
-  Hjälper till att skydda, bevara och säkerhetskopiera dina data med Exchange Online-arkivering  
-   App-skydd för Office & andra mobila appar med Intune App Protection 
-  Enhetshantering för datorer med Windows 10, MacOS & Mobile-enheter med Intune-enhetshantering 
-  Konsekvent säkerhetskonfiguration över enheter - skydd av företagets data över enheter - Windows Defender alltid- och uppdaterade 
 
 
**Förenklad Enhetsinställningar för distributions- och** 
* Enstaka admin console till installationsprogrammet och hantera användare och enheter 
* Automatisk installation av Office-program på datorer med Windows 10 
* Alltid uppdaterade Office + Windows 10 
* Strömlinjeformad driftsättning av datorer med Windows AutoPilot

### <a name="who-should-consider-adopting-microsoft-365-business"></a>Som bör överväga att anta Microsoft 365 Business? 
Microsoft 365 Business byggdes för små och medelstora kunder som har lite eller inga IT-resurser för personal och bäst i klassen produktivitets- och samarbetsverktyg funktionerna i Office 365 med enheten lösningar för hantering och säkerhet som skydda affärsdata. Microsoft 365 Business kunden är redo att flytta sin IT-verksamhet till molnet och är intresserade av att förebygga effekten av åtgärder för att skydda data på både företag och anställd ägda enheter.

### <a name="how-can-i-get-microsoft-365-business-for-my-business"></a>Hur kan jag få Microsoft 365 Business för mitt företag? 
Microsoft 365 Business kan köpas via en <a href="https://www.microsoft.com/solution-providers/search" target="_blank">Microsoft-Partner</a> eller direkt från <a href="https://www.microsoft.com//microsoft-365/business" target="_blank">Microsoft</a>. Att välja om du ska köpa direkt från Microsoft eller via en Microsoft-Partner, bör du tänka på Personal-kapacitet och önskar att underhålla IT-infrastruktur. En Microsoft-Partner kan hjälpa dig att distribuera och hantera din IT-infrastruktur inklusive Microsoft-lösningar.

### <a name="how-much-does-microsoft-365-business-cost"></a>Hur mycket kostar Microsoft 365 Business? 
Microsoft 365 Business erbjuds USD$ 20,00 användare/månad är baserat på ett _årligt avtal_ om köpt direkt från Microsoft. När du har köpt via en Microsoft-Partner, kan prissättning variera beroende på tjänster ger partnern och deras prissättning modell för Microsoft 365 Business. 

### <a name="is-there-a-cap-to-how-many-microsoft-365-business-seats-a-customer-can-have"></a>Finns det en fjärrskrivbordsanslutning till hur många Microsoft 365 Business säten som en kund kan ha? 
Microsoft 365 Business har utformats för små och medelstora företag med låg till medelhög komplexitetskrav för IT. Kunder kan köpa upp till 300 Microsoft 365 Business-licenser för organisationen. Kunder kan mixa och matcha cloud-abonnemang därmed, beroende på organisationens IT-krav, kunder kan lägga till Microsoft 365 Enterprise licenser till samma konto. 
 
När du funderar på en miljö som består av flera olika typer av abonnemang ska kunder fungera med deras betrodda rådgivare IT att bestämma bästa att hantera och skydda olika prenumerationer som Microsoft 365 Business och Microsoft 365 Enterprise olika funktioner för att skydda och hantera program och data.

### <a name="can-i-combine-microsoft-365-business-with-other-microsoft-subscription-offerings"></a>Kan jag kombinera Microsoft 365 Business med andra Microsoft-erbjudanden för prenumeration? 
Ja, kunder kan kombinera sina Microsoft 365 Business abonnemang med planer och tillägg från Azure Dynamics 365, Enterprise Mobility + säkerhet och Office 365.

### <a name="is-everyone-in-my-business-required-to-have-a-microsoft-365-business-subscription"></a>Är alla i företaget måste ha en Microsoft 365 Business-prenumeration? 
Nej, alla behöver inte ett Microsoft 365 Business abonnemang även om fördelarna för säkerhet och hantering är bara tillgängliga för användare med enheter som hanteras med en Microsoft 365 Business-prenumeration. 
 
Standardisering av IT-miljön fungerar för att minska underhåll och säkerhet kostnader över tiden och är ett företag bör sträva efter att uppnå. Vi inser emellertid att vissa storlek för små och medelstora kunder uppdatera sina program i första hand när de uppgraderar sin maskinvara under en längre tid. Företag kan distribuera Microsoft 365 Business till en del av deras organisation, men rekommenderas för bästa skydd av känslig företagsinformation och konsekvent samarbete erfarenheter distribution till alla användare.

### <a name="how-can-i-know-if-the-hardware-and-software-i-run-today-is-compatible-with-microsoft-365-business"></a>Hur vet jag om maskin- och programvara som körs idag är kompatibel med Microsoft 365 Business? 
Om maskinvaran du använder idag kör Windows 7 Pro eller senare, det sannolikt uppfyller minimikraven för Microsoft 365 Business. Vissa Windows 10-funktioner som Cortana, Windows Hej och multitouch kräver specifik maskinvara som bara är tillgänglig på nyare datorer. Se <a href="https://www.microsoft.com/windows/windows-10-specifications" target="_blank">systemkraven för Windows 10 Pro</a> för ytterligare information. 
 
Befintliga stationära (Win32) programkompatibilitet är starkt i Windows 10 med de flesta program fungerar utan några ändringar. Kunder och deras betrodda rådgivare IT bör läsa testprocessen rekommenderade program för <a href="https://docs.microsoft.com/windows/deployment/planning/windows-10-compatibility#recommended-application-testing-process" target="_blank">kompatibilitet med Windows 10</a> och granska <a href="https://products.office.com/office-system-requirements#subscription-plans-section" target="_blank">systemkraven för Office</a> för att säkerställa en smidig övergång till Microsoft 365 Business. 

### <a name="what-is-windows-10-business"></a>Vad är Windows 10 företag? 
Windows 10 Business är en uppsättning molnet tjänster och enheter hanteringsfunktioner som kompletterar Windows 10 Pro och aktivera Microsoft 365 Business centraliserad hantering och säkerhet kontroller. Windows 10 Business levereras också med Windows AutoPilot, en tjänst som förenklar driftsättning av nya Windows 10-datorer. Om du har enheter som är licensierade för Windows 7, 8 och 8.1 Professional, Microsoft 365 Business så här en uppgradering till Windows 10 Pro som är en förutsättning för att distribuera Windows 10 Business. 

### <a name="how-does-microsoft-365-business-help-support-our-bring-your-own-device-byod-policy"></a>Hur Microsoft 365 Business hjälper våra Supportpolicy för sätta din egen enhet (BYOD)? 
Många anställda som vill använda sina egna mobiltelefoner eller tabletter access personal och information om arbete i stället utföra flera enheter för varje ändamål. Användning av personliga enheter för arbetet och vardagen, ökar risken för att företagets information lätt kan hamna i fel händer. Många konkurrerande mobila data protection lösningar kräver att växla till ett visst läge på sin enhet eller använda en annan komplex mekanism för att användare kan hitta påträngande och därför undvika med hjälp av användarna. 
 
Microsoft 365 företag erbjuder sina kunder en enkla men kraftfulla sätt att aktivera anställda att använda sina personliga enheter för arbete och ger företaget möjlighet att förhindra att dessa enheter från att komma åt, behålla och dela information . Mer specifikt: 
* **App-skydd för Office mobile apps** skyddar Office data, inklusive e-post, kalender, kontakter och dokument på iOS och Android mobila enheter genom tvingande principer som automatiskt tar bort business data efter en föreskriven tid inte ansluta till tjänsten, som kräver att information lagras endast för OneDrive för företag som kräver en PIN-kod/fingeravtryck kontroll att få tillgång till Office-program och förhindra att företagets data som kopieras från ett Office-program till personliga appar. Hantering av mobila program för andra mobila appar via Intune är också tillgängligt för Microsoft 365 Business prenumeranter. 
  
* **Device Management för Windows 10 datorer** aallows företag kan välja att ställa in och använda funktioner som Windows Defender skydd mot skadlig kod, Automatiska uppdateringar och stänga av skärmar efter en föreskriven tid. Dessutom kan borttappade eller stulna enheter för Windows 10 vara helt torkas av program och data via administratörscenter. 

* **Enhetshantering för iOS, Android & MacOS** funktioner hjälper företag att hantera ett ekosystem med olika enhet med iOS, Android, Windows och macOS enheter på ett säkert sätt. 

### <a name="how-does-microsoft-365-business-help-protect-pcs-in-my-organization-from-malicious-attacks"></a>Hur Microsoft 365 Business skyddar datorer i organisationen från skadliga attacker? 
Datorer som hanteras med Microsoft 365 Business skyddas med Windows Defender, som är det Nej. 1 antivirus-funktionen på Windows 10, skydda fler datorer mot virus, skadlig programvara, spionprogram och andra hot än andra lösningar.  Med Microsoft 365 Business säkerställa företag skydd för Windows Defender fungerar alltid uppdaterad på sina Windows 10-enheter.  
 
Windows 10 Business även Windows Defender utnyttja Guard, en ny uppsättning funktioner för förebyggande av intrång. En av dess funktioner, kontrollerade mappåtkomst, stoppar ransomware i dess spår genom att låsa mappar och hindra obehöriga program från att komma åt viktiga filer. 

### <a name="whats-the-difference-between-office-365-business-premium-microsoft-365-business-and-microsoft-365-enterprise"></a>Vad är skillnaden mellan Office 365 Business Premium, Microsoft 365 Business och Microsoft 365 Enterprise? 
Microsoft har olika produktivitet och säkerhet infrastrukturhantering som små och medelstora kunder kan tänka på när du uppgraderar sina skrivbord och enheten infrastruktur var att samla allt mer kraftfulla funktioner. 

**Office 365 Business Premium** ger bäst i klassen produktivitet med Office 365-appar och tjänster men inte program skydds- och hanteringsfunktioner för Microsoft 365 Business.

**Microsoft 365 Business** kombinerar Office 365-appar och tjänster med hantering av mobila program och Windows 10 Pro om du vill aktivera fjärrhantering och skydda enheter mot virus och skadlig programvara. Den innehåller en förenklad hanteringskonsol principer kan administreras via vilken enheten och data. Många små och medelstora företag kan bäst tillgodoses med Microsoft 365 Business. 
 
**Microsoft 365 Enterprise** är en uppsättning planer som ger ökat för kompatibilitet och hantering över Microsoft 365 Business och är utformade för företagskunder och kunder som har över 300 användare-licensiering. Dessutom ger Microsoft 365 Enterprise planer ytterligare funktioner inklusive business intelligence och analytics-verktyg.

### <a name="can-i-switch-my-office-365-plan-to-microsoft-365-business"></a>Kan jag växla min Office 365-plan till Microsoft 365 Business? 
Ja, kunder kan växla planerna från en kvalificerande Office 365-plan Microsoft 365 Business. Beroende på kundens aktuella planen det får en minskning eller ökning av månadsavgifter.

### <a name="in-what-regions-is-microsoft-365-business-available"></a>Inom vilka områden det finns Microsoft 365 Business? 
Microsoft 365 Business kommer att vara tillgänglig för alla partners och kunder där Office 365 är tillgängligt. Visa en lista över <a href="https://products.office.com/business/international-availability" target="_blank">Office 365 internationella tillgänglighet för språk, länder och regioner</a>. 

### <a name="is-there-a-microsoft-365-business-trial-i-may-use-to-evaluate-the-offer"></a>Finns det en Microsoft 365 Business prövning som jag kan använda för att utvärdera erbjudandet? 
En utvärderingsversion av Microsoft 365 Business är tillgänglig för CSP: er. En prövning för direkta kunder kommer att vara tillgänglig senare. 
 
### <a name="what-should-customers-and-partners-know-before-running-microsoft-365-business-within-their-organization"></a>Vad bör kunder och partner känna till innan du kör Microsoft 365 Business inom organisationen? 
Kunder som vill få fullständig funktionerna i Microsoft 365 Business måste köra Windows 7, 8.1 eller 10 Pro<sup>[2](#footnote2)</sup> på sina befintliga stationära datorer. Befintliga Windows 10 Pro datorer bör köra skapare uppdatering om de inte redan har gjort. 

## <a name="deployment"></a>Distribution

### <a name="what-should-customers-consider-when-planning-a-microsoft-365-business-deployment"></a>Vad ska kunderna tänka på när du planerar en distribution med Microsoft 365 Business? 
Den mest direkta vägen till en lyckad distribution av Microsoft 365 Business är att kommunicera med en Microsoft-Partner. De har omfattande utbildning och erfarenhet av en mängd olika kundscenarier och är bäst utrustade för att förstå din miljö och behov. Kunder som har haft IT på personal kan använda <a href="https://support.office.com/article/496e690b-b75d-4ff5-bf34-cc32905d0364" target="_blank">Microsoft 365 Business komma igång</a> för att bistå dem i deras Microsoft 365 Business distribution. 

### <a name="does-microsoft-365-business-include-the-full-capabilities-of-microsoft-intune"></a>Innehåller Microsoft 365 Business utökar Microsoft Intune? 
Ja, Microsoft 365 Business prenumeranter är licensierade för att använda fullständiga Intune-funktioner för iOS, Android, MacOS och andra plattformar enhetshantering. Funktioner som inte finns i en förenklad hanteringskonsol Microsoft 365 Business som 3: e part app hantering och konfiguration av WiFi-profiler, VPN-certifikat, etc. kan hanteras i full Intune-konsolen.   
  

### <a name="does-azure-active-directory-premium-p1-come-with-microsoft-365-business"></a>Kommer Azure Active Directory Premium P1 med Microsoft 365 Business? 
Microsoft 365 Business bygger på teknik från hela Microsoft och när den delar vissa funktioner med Azure Active Directory, är inte en fullständig version. Säkerhet och av hanteringsprinciper som skapats i Microsoft 365 Business förlitar sig på Azure funktionalitet, men innehåller inte alla funktioner (t.ex. Self service funktioner, funktioner för villkorad tillgång och rapportering). Kunder kan välja att köpa Azure Active Directory Premium P1 och P2 som ett tillägg till Microsoft 365 Business. Läs [Servicebeskrivningen för Microsoft 365 Business](https://technet.microsoft.com/library/b042d1fd-c997-46d8-8c02-1df4c932f148) för mer information 
 
 
### <a name="does-microsoft-365-business-allow-customers-to-manage-macs"></a>Tillåter Microsoft 365 Business kunder att hantera Macintosh? 
Intune hjälper dig att på ett säkert sätt hantera enheter med iOS, Android, Windows och macOS. 
 
### <a name="what-is-windows-autopilot"></a>Vad är Windows AutoPilot? 
Windows AutoPilot är en tjänst som förenklar driftsättning av nya Windows 10-datorer. Den här processen kan utföras när slutanvändaren loggar in på Microsoft 365 Business för första gången – utan att någonsin vidrör enheten IT – genom att använda kontroller för centraliserad hantering av Microsoft 365 Business. Du kan också använda Windows AutoPilot för befintliga datorer som kör Windows 10 Professional skapare Update (eller senare) och har fabriksinställningarna. Information om Windows AutoPilot kan hittas i <a href="https://blogs.technet.microsoft.com/windowsitpro/2017/06/29/modernizing-windows-deployment-with-windows-autopilot/" target="_blank">juni, 2017 blogginlägg</a>.

## <a name="compatibility"></a>Kompatibilitet

### <a name="can-i-add-office-365-add-ons-to-microsoft-365-business"></a>Kan jag lägga till Office 365-tillägg för Microsoft 365 företag? 
Alla tillägg som kan läggas till Office 365 Business Premium kan läggas till Microsoft 365 Business. Detta innebär att du kan köpa Advanced Threat Protection, Office 365 Cloud App säkerhet, avancerad överensstämmelse, hot intelligens, MyAnalytics, PowerBI Pro och ljud konferens. 
 
### <a name="can-i-add-phone-system-and-calling-plans-to-microsoft-365-business"></a>Kan jag lägga till telefonsystemet och anropa planer till Microsoft 365 Business? 
Nej, telefonsystem och anropa planerar är reserverade för kunder som har mer avancerade behov. Kunder som kräver dessa funktioner bör titta på Microsoft 365 Enterprise erbjudanden. 
 
### <a name="can-microsoft-365-business-customers-use-windows-defender-advanced-threat-protection"></a>Microsoft 365 Business-kunder kan använda Windows Defender Avancerat skydd? 
Nej, kunder som kräver Windows Defender Avancerat skydd måste Windows 10 Enterprise E5 eller Microsoft 365 Enterprise E5. 
 


### <a name="can-customers-create-hosted-windows-10-vms-with-a-microsoft-365-business-subscription"></a>Kunder kan skapa värd Windows 10 VMs med Microsoft 365 Business-prenumeration? 
Nej, kunder som kräver virtualisering ska köpa Windows 10 Enterprise eller ett Microsoft 365 Enterprise-abonnemang.

### <a name="what-is-the-gdpr-and-how-does-microsoft-365-business-help-customers-with-their-compliance-obligations"></a>Vad är BNPR och hur Microsoft 365 Business hjälper kunder med deras skyldigheter att?
Allmänna Data Protection förordning (BNPR) är en omfattande ny sekretess lag som ger invånare i Europeiska unionen (EU) större kontroll över sina ”personuppgifter” och kräver organisationer att upprätthålla den personliga informationens integritet. BNPR kräver organisationer som kontroll eller bearbeta personuppgifter knutna till EU: S invånare att bara använda tredje parts databehandlare som uppfyller kraven på BNPR för behandling av personliga data. I mars 2017 gjort Microsoft tillgängliga avtalsenliga garantier som ger dessa garantier. Kunder som har frågor om hur Microsoft kan hjälpa dem att uppfylla sina skyldigheter för ytterligare BNPR bör Lär dig avancerade överensstämmelse och säkerhetsfunktioner som är tillgängliga som tillägg (t.ex. Azure informationsskydd, förhindra dataförlust, Avancerat Threat Protection, etc.) och i andra paket (exempelvis Microsoft 365 Enterprise E5). Om du vill veta mer kan du besöka [www.microsoft.com/gdpr](https://www.microsoft.com/gdpr).
