# <a name="add-additional-users-to-microsoft-365-business"></a>Lägga till ytterligare användare i Microsoft 365 Business

## <a name="add-new-users"></a>Lägg till nya användare

1. Gå till administratörscenter på <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>. 
2. Välj **användare** \> **aktiva användare**i det vänstra navigeringsfältet.
1. På sidan **aktiva användare** väljer **du Lägg till en användare**.
 4. I panelen **Ny användare** skriver du in informationen som begärs. 
  
    Du kan också ange ytterligare information under **Kontaktinformation**, välja hur du anger lösenord under inställningen **Lösenord** och tilldela roller under **Roller**.
      
    ![Enter user information in the New user card](media/f04d39ca-48be-4868-8330-8552a4754c8b.png)
      
    I avsnittet Produktlicenser sätter du licensinställningen **Microsoft 365 Business** till **På**.
      
    ![Set the license setting to On position](media/7404f7f7-93bc-44a3-9ffb-4208b5b17402.png)
  
Se även [lägga till användare individuellt eller i bulk](https://docs.microsoft.com/office365/admin/add-users/add-users).
