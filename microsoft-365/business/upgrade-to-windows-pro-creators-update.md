## <a name="upgrade-to-windows-10-pro"></a>Uppgradera till Windows 10 Pro
  
Om du vill uppgradera till Windows 10 Pro har du flera alternativ. Du kan:
    
- Installera uppgraderingen från webbplatsen för nedladdning av [Microsoft-programvara](https://go.microsoft.com/fwlink/?LinkID=836951 ) Välj det här alternativet om enheten du är inloggad på är på samma enhet som den du &ndash; vill uppdatera. 

    - Från webbplatsen för nedladdning av programvara klickar du **på Uppdatera nu** för att börja uppgradera enheten till Windows 10 Pro Creators Update. 
    
- Skapa ett installationsmedia [](https://go.microsoft.com/fwlink/?LinkID=836960) med hjälp av verktyget för att skapa media Välj det här alternativet om du vill skapa ett installationsmedia för &ndash; Windows 10 Pro Creators Update (USB-flashenhet eller ISO-fil) för att installera Windows 10 på en annan dator än den du använder.

    - Läs anvisningarna om hur du använder verktyget och skapar ditt installationsmeddelar. 

> [!NOTE]
> Om du har Windows-enheter med Windows 7 Pro, Windows 8 Pro eller Windows 8.1 Pro ger Microsoft 365 för företag-prenumerationen dig rätt att uppgradera till Windows Pro 10.
    
Se [Konfigurera Windows-enheter för Microsoft 365 för företag-användare för](set-up-windows-devices.md) att slutföra konfigurera Windows 10-enheter. 
  
Se [Konfigurera mobila enheter för Microsoft 365 för företag-användare för att](set-up-mobile-devices.md) slutföra konfigurera Android- och iOS-enheter.
