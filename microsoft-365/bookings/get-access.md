# <a name="get-access-to-microsoft-bookings"></a>Få tillgång till Microsoft Bookings

I den här artikeln beskrivs hur du får tillgång till Bookings för första gången. Den innehåller även information om hur du aktiverar eller inaktiverar Bookings i organisationen. Här förklaras också hur du kan tilldela licenser till dina användare så att de kan komma åt och använda Bookings.

## <a name="access-bookings-for-the-first-time"></a>Få åtkomst till Bookings för första gången

Om DIN IT-administratör har beviljat dig åtkomst till Bookings kan du komma åt appen via Office online.

Första gången du använder [Bookings](https://outlook.office.com/bookings/onboarding) skapar du en kalender och affärsverksamhet.

1. Välj **Skaffa nu och** välj Lägg till en **bokningskalender**. Om det inte är första gången du gör det på den här sidan kan du välja en annan bokningskalender eller göra en sökning efter en annan bokningskalender.

:::image type="content" source="media/bookings - get it now.png" alt-text="Skärmbild: Sidan Skaffa nu bookings. Välj Skaffa nu för att börja använda Bookings":::

2. Ange företagets namn och affärstyp och välj **Fortsätt**.

Nu kan du konfigurera Bookings för din organisation. Du kan gå till sidan Bookings med den [här länken](https://outlook.office.com/bookings/onboarding), från startprogrammet eller från office.com. Använd följande avsnitt för att fortsätta att konfigurera bookings-funktioner.

- [Schemalägga verksamhetsslut, ledighet och semesterdagar](schedule-closures-time-off-vacation.md)
- [Ange dina principer för schemaläggning](set-scheduling-policies.md)
- [Ange arbetstider för anställda](employee-hours.md)
- [Anpassa och publicera din bokningssida](customize-booking-page.md)
