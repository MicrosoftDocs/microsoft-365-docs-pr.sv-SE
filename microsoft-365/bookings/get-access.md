# <a name="get-access-to-microsoft-bookings"></a>Få tillgång till Microsoft Bookings

I det här avsnittet beskrivs hur du får till gång till bokningar för första gången. Du får också information om hur du aktiverar eller inaktiverar bokningar i din organisation. Dessutom förklaras hur du kan tilldela licenser till användarna så att de kan komma åt och använda bokningar.

## <a name="access-bookings-for-the-first-time"></a>Åtkomst till bokningar för första gången

Om din IT-administratör har tilldelat dig åtkomst till bokningarna kan du komma åt appen via Office Online.

Första gången du använder [bokningar](https://outlook.office.com/bookings/onboarding) skapar du en kalender och konfigurerar ditt företag.

1. Välj **Hämta den nu** och välj, **Lägg till en boknings kalender**. Om det inte är första gången på den här sidan kan du välja en annan boknings kalender eller göra en sökning för en annan boknings kalender.

:::image type="content" source="media/bookings - get it now.png" alt-text="Skärm bild: sidan för att hämta fönstret nu. Välj Hämta det nu för att börja använda bokningar":::

2. Ange ditt företags namn och din företags typ och välj **Fortsätt**.

Nu är du redo att konfigurera pärmar för din organisation. Du kan gå till sidan bokningar med den här [länken](https://outlook.office.com/bookings/onboarding)från Start programmet eller från Office.com. Använd följande avsnitt för att fortsätta att ställa in dina funktioner.
