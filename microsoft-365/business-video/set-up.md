# <a name="set-up-microsoft-365-business-premium"></a>Konfigurera Microsoft 365 Business Premium

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE471FJ?autoplay=false]

## <a name="try-it"></a>Prova!

1. Logga in på administrationscentret för Microsoft 365 och välj **Gå till installation.** Installationsguiden startar.
2.  På sidan **Installera Office-appar** kan du välja att installera Office-appar på datorn eller välja **Fortsätt.**
3.  På din **domän är ditt företags identitetssida,** ange domännamnet för ditt företag och välj sedan Använd den här **domänen.** Om du använder en vanlig registrator som WordPress eller GoDaddy läggs dina poster till automatiskt på den här sidan.
4. På sidan  **Verifiera din domän**  väljer du Verifiera  **för** att verifiera att du äger domänen och loggar sedan in på ditt domänregistratorkonto. Om du vill låta Microsoft lägga till en TXT-post för kontoverifiering väljer **du Bekräfta.**
5. På sidan  **Lägg till användare och**  tilldela licenser anger du användarens förnamn, efternamn och användarnamn. Välj sedan **Lägg till användare och tilldela licenser.**
6. Ange alternativa **e-postadresser för användarna på** sidan Dela inloggningsinformation och välj Skicka e-post och **fortsätt.**
7.  Klicka på **Fortsätt på** sidan Aktivera **poster.**
8.  På sidan  **Lägg till DNS-poster**  rekommenderar vi att du låter tjänsterna vara påslagna för e-post, Teams och säkerhet på mobila enheter. Välj  **Konfigurera** och sedan Anslut  **för att** bekräfta ändringarna med din domänregistrator.
9.  På sidan Öka skyddet mot avancerade **cyberhot** ska du låta **Office** avancerat skydd mot hot vara på, granska vad som kommer att genomsökas och sedan välja Skapa **avancerade****skyddsprinciper.**   
10.  Om du vill kan du installera Outlook på dina mobila enheter. Välj **Fortsätt.**
11. På sidan **Hjälp personer använder Microsoft Teams** väljer du om du vill skicka e-post till användare i organisationen om Teams och välj **Fortsätt.**
12. På sidan **Skydda data i Office** för mobil, låt hantering av mobilappar vara på, expandera inställningarna och granska dem och välj sedan Skapa princip för hantering av **mobilappar.**
13. Återgå till Administrationscenter för Microsoft när konfigurationen är klar. I administrationscentret kan du fortsätta att konfigurera funktioner som Windows 10-principer, DLP osv. på **konfigurationssidan.**

Konfigurationen är klar.
