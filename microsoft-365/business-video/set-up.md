# <a name="set-up-microsoft-365-business-premium"></a>Konfigurera Microsoft 365 Business Premium

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE471FJ?autoplay=false]

## <a name="try-it"></a>Prova!

1. Logga in på administrationscentret för Microsoft 365 och välj  **Gå till inställningar**. Installationsguiden startar.
2.  På sidan  **Installera Office-appar**  kan du välja att installera Office-appar på datorn eller välja  **Fortsätt**.
3.  På din  **domän är ditt företags identitetssida,**  ange domännamnet för ditt företag och välj sedan Använd den här  **domänen**. Om du använder en vanlig registrator som WordPress eller GoDaddy läggs dina poster till automatiskt på den här sidan.
4. På sidan  **Verifiera din domän**  väljer du Verifiera för  **att** verifiera att du äger domänen och loggar sedan in på ditt domänregistratorkonto. Om du vill att Microsoft ska lägga till en TXT-post för kontoverifiering väljer  **du Bekräfta**.
5. På sidan  **Lägg till användare och**  tilldela licenser anger du användarens förnamn, efternamn och användarnamn. Välj sedan **Lägg till användare och tilldela licenser.**
6. På sidan Dela inloggningsinformation anger du alternativa e-postadresser för användarna och väljer Skicka **e-post** **och fortsätt.**
7.  På sidan **Aktivera poster** klickar du på **Fortsätt.**
8.  På sidan  **Lägg till DNS-poster**  rekommenderar vi att du lämnar tjänsterna påslagna för e-post, Team och säkerhet på mobila enheter. Välj  **Konfigurera** och välj sedan  **Anslut för att** bekräfta ändringarna med din domänregistrator.
9.  På sidan Öka skyddet mot avancerade cyberhot ska du lämna **Office avancerat** skydd för hot på, granska vad som **söks** igenom och sedan välja Skapa principer för **avancerat skydd för****hot.**   
10.  Om du vill kan du installera Outlook på dina mobila enheter. Välj **Fortsätt.**
11. På sidan **Hjälp personer att använda Microsoft Teams** väljer du om du vill skicka e-post till användare i din organisation om Teams och väljer **Fortsätt.**
12. På sidan **Skydda data i Office** för mobil kan du låta hantering av mobilappar vara på, expandera inställningarna och granska dem och sedan välja Skapa policy för hantering av **mobilappar.**
13. Gå tillbaka till Microsofts administrationscenter när konfigurationen är klar. I administrationscentret kan du fortsätta att konfigurera funktioner som Windows 10-principer, DLP osv. på **sidan Inställningar.**

Konfigurationen är klar.
