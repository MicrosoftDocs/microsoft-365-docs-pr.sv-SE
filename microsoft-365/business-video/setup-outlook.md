# <a name="set-up-outlook-for-microsoft-365-for-business-email"></a>Konfigurera Outlook för Microsoft 365 för företag-e-post 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/9fe86884-8a83-42cc-bca9-61a12e6dad31?autoplay=false]

## <a name="try-it"></a>Prova!

När du har installerat Office-apparna kan du&#39;du vill att Outlook ska börja använda e-post, kalender och kontakter. Här&#39;s how.

1. Öppna Start-menyn. Sök efter Outlook och välj det.
2. Ange din Microsoft 365-e-postadress och välj  **Anslut**.
3. Ange ytterligare e-postadresser som du vill använda, till exempel din gamla eller personliga e-postadress. Välj  **Nästa**.
4. Ange ett lösen ord om du uppmanas till det och välj sedan  **Logga in**.
5. När du har lagt till alla dina konton kan du välja om du vill konfigurera Outlook Mobile eller vänta till senare.
6. Välj  **klar**. Det kan ta flera minuter för Outlook att hämta din e-post och andra data.

Nu kan du Visa e-post för de konton som du har lagt till. Du kan också visa kalendern, kontakter och uppgifter.
