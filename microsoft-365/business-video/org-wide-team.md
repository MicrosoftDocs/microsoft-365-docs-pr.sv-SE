# <a name="create-an-org-wide-team"></a>Skapa ett organisationsteam

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2GCG9?autoplay=false]

## <a name="try-it"></a>Prova själv!

Skapa ett organisationsomfattande team i Microsoft Teams som alla på företaget kan delta i. När du lägger till fler Microsoft 365 läggs de automatiskt till i teamet.

1. I Microsoft Teams väljer du **Teams** och sedan **Skapa team.**
2. Välj **Skapa ett team från grunden.**
3. Välj **Skapa ett team för hela organisationen.**

> [!NOTE]
> Du måste vara administratör Microsoft 365 kunna visa det här alternativet.

1. Ge teamet ett namn, lägg till en beskrivning och välj sedan  **Skapa**.

I teamet finns kanaler. Kanalen Allmänt skapas automatiskt. Du kan lägga till fler kanaler, separata konversationer och filer för olika avdelningar, händelser eller projekt.

Prova något av följande:

- Använd fliken  **Inlägg** för konversationer och meddelanden som gäller hela organisationen, till exempel ett välkomstmeddelande till alla.
- På fliken **Filer** kan du skapa nya dokument som alla kan bidra till, till exempel ett Microsoft Word dokument. Du kan också dra och släppa befintliga dokument på  **fliken** Filer.
- Använd  **fliken Wiki** för att göra snabbanteckningar. Och välj plustecknet **+** () för att lägga till appar eller filer som ytterligare flikar.
- Med flikarna till vänster kan du visa den senaste aktiviteten, chatta med andra teammedlemmar, visa de team du tillhör och din kalender. Du kan ringa röst- och videosamtal till andra teamanvändare och visa filer som du nyligen arbetat med.
