# <a name="overview-of-microsoft-bookings"></a>Översikt över Microsoft-bokningar

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE26B1q?autoplay=false]

Microsoft-bokningar är ett online-och mobil program för småföretag som tillhandahåller tjänster till kunder med avtalad tid. Exempel på verksamheter är bland annat hår Salons, tand kontor, Spas, juridik företag, operatörer för finansiell tjänst, konsulter och automatiska butiker.

## <a name="get-started-with-bookings"></a>Komma igång med bokningar

1. För att komma igång loggar du in i Microsoft 365 och väljer **bokningar** från Start programmet.
1. Välj **Skaffa det nu** på sidan **bokningar** .
1. Lägg till din företags information så är du redo att börja använda bokningar.

    Om du vill hantera dina bokningar på din mobila enhet kan du hämta dem från App Store.

## <a name="use-bookings"></a>Använda bokningar

- Kontrol lera din företags adress, dina arbets tider och annan företags information **på panelen instrument** paneler och uppdatera om det behövs.
- Du och din personal kan lägga till eller uppdatera avtalade tider på sidan **kalender** . Du kan visa kalendern per dag, vecka eller månad.
- På sidan **kunder** kan nya kunder läggas till eller importeras från andra konton, till exempel Gmail eller en enhet.
- På sidan **personal** kan du lägga till och hantera personalen. Det är viktigt att lägga till deras e-postadresser så att de kan få omedelbara meddelanden om nya avtalade tider.
- Med sidan **tjänster** kan du lägga till och hantera tjänster som kunder väljer när de bokar en avtalad tid.
- Slutligen finns det många inställningar som schemaläggnings policy, e-postaviseringar, tids zons inställningar och mer på sidan **bokningar** .

När du har sparat och publicerat kalender kalendern kan den bäddas in på företagets webbplats eller Facebook-sida.
