# <a name="install-outlook-and-office-apps-on-ios-devices"></a>Installera Outlook- och Office-appar på iOS-enheter

När du&#39;installerat Microsoft 365 Business Premium kan du installera Office-appar på din iPhone. När du har installerat programmen uppmanas&#39;att skapa en PIN-kod för att skydda dina Office-data.

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWee2n?autoplay=false]

## <a name="try-it"></a>Prova!

1. Öppna App Store på din iPhone.
2. Välj  **Sök,** skriv  **Outlook** i sökfältet och välj det sedan.
3. Välj **Hämta**   >   **installation.**
4. Ange ditt Apple ID-lösenord och välj **Logga in**  >   **Öppna.**
5. Ange din e-postadress till arbetet.
6. Välj **Lägg till** konto, ange ditt lösenord och välj sedan Logga **in.**
7. (Valfritt), Välj **Låt oss göra det ** om du vill lägga till fler konton, eller välj  **Kanske senare för**  att hoppa över det här steget.
8. Välj  **OK** för att starta om programmet så att Microsoft 365 Business kan skydda dina Outlook-data.
9. Öppna Outlook igen så visas ett bekräftelsemeddelande. Välj **OK.**
10. Skapa en PIN-kod för Outlook och andra Office-program och bekräfta den. Från och med nu används den här PIN-koden för att skydda alla data i Office-appen.
11. Välj  **Aktivera för att aktivera**  e-postaviseringar.
12. Välj  **Tillåt för** att aktivera push-meddelanden.

Nu kan du använda Outlook på din iPhone för att kontrollera din e-post, kalender och kontakter.

Du kan installera resten av Office-apparna på samma sätt, och data i de apparna skyddas av Microsoft 365 Business Premium.
