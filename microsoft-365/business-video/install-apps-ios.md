# <a name="install-outlook-and-office-apps-on-ios-devices"></a>Installera Outlook och Office-appar på iOS-enheter

När du&#39;aktivera Microsoft 365 Business Premium kan du installera Office-appar på din iPhone. När du har installerat programmen uppmanas du&#39;lla att skapa en PIN-kod för att skydda dina Office-data.

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWee2n?autoplay=false]

## <a name="try-it"></a>Prova!

1. Öppna App Store på din iPhone.
2. Välj  **Sök** , Skriv  **Outlook** i Sök fältet och välj sedan det.
3. Välj **Skaffa**   >   **installation**.
4. Ange ditt lösen ord för Apple-ID, Välj **Logga in**  >   **Open**.
5. Ange din e-postadress för arbetet.
6. Välj  **Lägg till konto** , ange ditt lösen ord och välj sedan  **Logga in**.
7. (Valfritt), välj * * Låt oss göra det * * för att lägga till fler konton eller Välj  **kanske senare**  för att hoppa över det här steget.
8. Välj  **OK** för att starta om programmet så att Microsoft 365 Business kan skydda dina Outlook-data.
9. Öppna Outlook igen och ett bekräftelse meddelande visas. Välj  **OK**.
10. Skapa en PIN-kod för Outlook och andra Office-program och bekräfta det. Från och med nu används denna PIN-kod för att skydda alla Office-AppData.
11. Välj  **Aktivera**  för att aktivera e-postaviseringar.
12. Välj  **Tillåt** för att aktivera push-meddelanden.

Du kan nu använda Outlook på din iPhone för att kontrol lera e-post, kalender och kontakter.

Du kan installera resten av Office-programmen på samma sätt och data i dessa appar skyddas av Microsoft 365 Business Premium.
