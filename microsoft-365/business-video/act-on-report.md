# <a name="review-usage-reports-in-microsoft-365"></a>Granska användningsrapporter i Microsoft 365

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1SOFC?autoplay=false]

Ta reda på hur anställda använder Microsoft 365 tjänster som e-OneDrive post och vidta åtgärder.

I [administrationscentret](https://admin.microsoft.com)kan du se alla tillgängliga rapporter.

## <a name="try-it"></a>Prova själv!

1. Gå till **Rapporter** och sedan **Användning.** Du hittar information om en rapport genom att välja en rapport. I rapporten Aktiva användare visas vilka tjänster som används mest av personerna i företaget.
1. Som exempel ska vi titta på den nya OneDrive rapporten. Här kan du se hur många som har ett konto, du kan också visa det totala antalet och aktiva filer samt lagringsutrymmet som används. Om du vill se alla andra rapporter går du till den nedrullningsmenyn.
1. Här kan vi titta på rapporten om användning av e-postapp och se vilka metoder dina anställda använder mest för att komma åt sin e-post. Du kan också visa information som är specifik för en enskild person, men informationen är dold som standard. Du visar användarinformation genom att gå till **Inställningar** och sedan **till Tjänster & tillägg.**
1. På rapportpanelen kan vi aktivera den anonyma identifieraren. Nu stänger vi panelen och går tillbaka till **Rapporter,** Användning **och** **E-postaktivitet.** Här kan du se specifik information för en enskild person.

    Kom ihåg att om du någon gång fastnar kan du alltid gå tillbaka till Hjälpen som ger dig information om den specifika rapport som du visar.
