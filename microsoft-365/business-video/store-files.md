# <a name="where-to-store-files"></a>Platser för fillagring

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FTHX?autoplay=false]

I Microsoft 365 kan du lagra ditt arbete antingen i OneDrive för företag eller på SharePoint-webbplatser. Varje person i organisationen har sitt eget bibliotek för OneDrive för företag för att lagra personliga arbetsfiler. När du skapar och sparar filer i en OneDrive-mapp synkroniseras de till molnet så att du kan komma åt dem var du än är, en webbläsare, dator eller en mobil enhet. Använd en SharePoint-webbplats för delad fil lagring och grupp samarbete. SharePoint-webbplatser synkroniseras också med molnet för enkel åtkomst var som helst.

## <a name="try-it"></a>Prova!

### <a name="store-or-share-files-with-onedrive"></a>Lagra eller dela filer med OneDrive

1. Öppna OneDrive i Utforskaren på datorn. Eller från [Office 365](https://www.office.com/)öppnar du  **OneDrive**  från Start programmet.
2. Visa och ladda upp personliga filer eller dela dokument eller mappar genom att klicka på  **dela**  och sedan antingen bjuda in andra att visa dokumenten eller skicka en länk till dem. [Mer information](https://support.microsoft.com/office/9fcc2f7d-de0c-4cec-93b0-a82024800c07#os_type=onedrive_-_business).

### <a name="collaborate-with-a-sharepoint-site"></a>Samar beta med en SharePoint-webbplats

1. Från [Office 365](https://www.office.com/)öppnar du  **SharePoint**  från Start programmet och väljer grupp webbplatsen.
2. Lägg till filer till standard platsen för dokument eller skapa fler dokument platser, så kallade bibliotek.

Du kan anpassa din SharePoint-webbplats genom att skapa under webbplatser för specifika projekt, team eller kunder. Du kan också lägga till företagets namn, logo typ och utseende, så att webbplatsen hamnar i ett företags intranät. [Mer information](https://support.microsoft.com/office/06bbadc3-6b04-4a60-9d14-894f6a170818).
