# <a name="usage-insights"></a>Användningsinsikter
Den här vyn innehåller användningsmått för dina Microsoft Hanterat skrivbord enheter. För dessa ändamål anses en app vara "i bruk" om den körs och är i fokus.

Om du vill visa användningsdata väljer **du fliken** Användning.

![Fönstret Användning. Diagram över enhetsanvändning (procentandelsanvändning kontra tid) i övre vänstra hörnet. Tabellen Programanvändning uppe till höger. Enhetslisttabell längst ned med kolumnnamn, modell, serienummer, visningsnamn, användarnamn, aktuellt tillstånd (aktivt, lågt eller inaktivt), total enhetsanvändning i timmar och antal aktiva dagar.](../../media/insights_usage.png)

## <a name="device-usage"></a>Enhetsanvändning

I området **Enhetsanvändning** visar vi användningsnivån för rapporteringsenheter över tid samt hur många registrerade enheter som rapporterar data. För alla data som visas på den här sidan kan du ändra intervallet för att visa rapporterade data under antingen de senaste 14 eller 28 dagarna med hjälp av den nedrullningsmenyn i det övre högra hörnet.

Vi kategoriserar användningsnivån med följande villkor:

- **Aktiv:** minst 55 timmars användning
- **Låg:** mellan 8 och 55 timmars användning
- **Inaktiv:** mindre än 8 timmars användning




## <a name="application-usage"></a>Programanvändning

I **området Programanvändning** finns information om programmen efter användning och deras respektive timmars användning för dina hanterade enheter. Dessutom visas det totala antalet hanterade enheter med ett visst program. Välj **Sök** för att söka efter ett visst program om det inte redan finns i listan.


## <a name="device-details"></a>Enhetsinformation
I informationsområdet finns information om specifika enheter, inklusive totalt antal enhetstimmar och aktiva dagar under tidsperioden. Använd filtren för att begränsa vyn till de enheter med användningsmönstret "Aktiv", "Inaktiv" eller "Låg" eller de som inte rapporterar.
