# <a name="usage-insights"></a>Användningsinsikter
Den här vyn innehåller användnings mått för Microsoft Managed Station ära datorer. I de här syftena betraktas en app som "används" om den körs och är i fokus.

Om du vill visa användnings data väljer du fliken **användning** .

## <a name="device-usage"></a>Användning av enheter

I området för **användning av enheter** visar vi användnings nivån för rapporterings enheter över tiden, samt hur många registrerade enheter som rapporterar data. För alla data som visas på den här sidan kan du ändra intervallet för att Visa rapporterade data under de senaste 14 eller 28 dagarna med hjälp av den nedrullningsbara menyn högst upp till höger.

Vi kategoriserar användnings nivån med följande villkor:

- **Aktivt:** minst 55 timmars användning
- **Låg:** mellan 8 och 55 timmars användning
- **Inaktiv:** mindre än 8 timmars användning




## <a name="application-usage"></a>Program användning

**Program användnings** området visar vilka program som är avsedda för användning och deras respektive timmars användning för dina hanterade enheter. Dessutom visas det totala antalet hanterade enheter som använder ett givet program. Välj **Sök** för att hitta ett specifikt program om det inte redan visas.


## <a name="device-details"></a>Enhets uppgifter
Detalj området innehåller information om specifika enheter, inklusive totalt antal enheter och antalet aktiva dagar under tids perioden. Använd filtren för att begränsa vyn till de enheter som har en "aktiv", "inaktiv" eller "Low"-mönster eller de som är "rapporterar".
