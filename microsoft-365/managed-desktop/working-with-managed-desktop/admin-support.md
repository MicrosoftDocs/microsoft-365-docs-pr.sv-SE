# <a name="admin-support-for-microsoft-managed-desktop"></a>Admin-support för Microsoft Hanterat skrivbord

Du kan skicka supportärenden eller feedbackförfrågningar till Microsoft via Microsoft Administrationsportalen för skrivbordet. Supportbegäranden prioriteras alltid framför feedback som skickas in. Supportbegäranden är triangelde och hanteras enligt allvarlighetsgraden som anges i tabellen [för allvarlighetsdefinition.](#sev) Feedback granskas och ett svar tillhandahålls där så begärs. 

>[!IMPORTANT]
>Kontrollera att du har [ställt in en administratörskontakt](../get-started/add-admin-contacts.md) för appförpackningar, enheter, säkerhet och annat. Du kan inte skicka in en supportbegäran i något av dessa områden om en administratörskontakt inte har konfigurerats.

**Så här skickar du in en supportbegäran**
1. Logga in på [Microsoft Endpoint Manager och](https://endpoint.microsoft.com/) gå till menyn Felsökning + **support.**
2. Titta efter avsnittet Microsoft Hanterat skrivbord, välj **Tjänstförfrågan.**
3. Välj **+ Nytt supportärenden** **om du har supportbegäranden.**
4. Välj den **typ av supportbegäran** som matchar den hjälp du behöver. I tabellen nedan visas en översikt över alternativen. 
5. Välj **allvarlighetsnivån.** Mer information finns i definitioner [av allvarlighetsgrad för supportbegäran.](#sev) 

Typ av supportbegäran | När ska jag använda?
--- | ---
Incident | Du kräver att Microsoft Managed Desktop Operations-teamet undersöker, t.ex. påverkas av en ändring eller ett avbrott i tjänsten.
Begäran om information | Du planerar en ändring för nätverk, proxykonfiguration, VPN-system, förfallodatum för certifikat eller bara behöver information om tjänsten. Du bör få ett svar från Microsoft Managed Desktop Operations Team när du kommunicerar en ändring inom organisationen.
Ändra begäran | Du kräver att Microsoft Managed Desktop Operations-teamet gör en ändring, till exempel att flytta enheter mellan uppdateringsgrupper.

## <a name="support-request-severity-definitions"></a>Definitioner av allvarlighetsgrad för supportbegäran

Den ursprungliga svarstiden är den tidsperiod från då du skickar din supportbegäran till när en tekniker från Microsoft Hanterad dator kontaktar dig och börjar arbeta med din supportbegäran. Den ursprungliga svarstiden varierar beroende på begärans affärs påverkan, beroende på begärans allvarlighetsgrad.

Mer information:
- **Supportspråk** – allt stöd finns på engelska.
- **Ändringar av allvarlighetsnivå** – Microsoft kan nedgradera allvarlighetsnivån om kunden inte kan tillhandahålla lämpliga resurser eller svar för att Microsoft ska kunna fortsätta med problemlösningen. 
- **Öppettider –** I de flesta länder är arbetstiden från 9:00 till 17:00, Pacific Standard Time.
- **Programkompatibilitet** – För att ett kompatibilitetsproblem ska kunna beaktas måste det finnas ett reproducerbart fel, av samma version av programmet, mellan den föregående och aktuella versionen av Windows eller Office. Microsoft kräver att du använder en kontaktkontakt hos en kund för att lösa kompatibilitetsproblem med programmet. Personen måste arbeta direkt med vårt Fast Track-team för att undersöka och lösa problemet.
- **Svarstid för kunder** Om en kund inte kan uppfylla de förväntade svarskraven nedgraderar Microsoft begäran med en allvarlighetsnivå, till ett minimum av allvarlighetsgrad C. Om en kund inte svarar på en begäran om åtgärd minimerar och stänger Microsoft supportbegäran inom 48 timmar från den senaste begäran.

## <a name="provide-feedback"></a>Ge feedback

Vi uppskattar din feedback och använder den för att förbättra administratörssupporten.

När en biljett är i ett **minimerat eller** **löst** tillstånd kan du dela din feedback om din upplevelse av det specifika problemet. Om du vill dela feedback går du **till sidan Tjänstförfrågningar** på **menyn Felsökning + support** på MEM-portalen. Välj den specifika biljetten. Information om biljetten visas i infället till höger, väljer **fliken Feedback** och ger den information som efterfrågas. Se till att du inte tar med någon personlig information i feedbackformuläret. Mer information om sekretess finns i [Microsofts sekretesspolicy.](https://privacy.microsoft.com/privacystatement)
