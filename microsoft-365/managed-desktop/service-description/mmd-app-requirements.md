# <a name="microsoft-managed-desktop-app-requirements"></a>Microsofts appkrav för hanterade skrivbord

<!--This topic is the target for aka.ms/app-req. This is aka link is used from EA agreement for MMD. do not delete.-->

<!--Application addendum -->
 
Microsoft Managed Desktop kräver att vi hanterar enheter med en specifik metod för att garantera enheters prestanda, tillförlitlighet och service. Om du är säker på att den metod som Används av Microsoft Managed Desktop för områdena nedan inte fungerar för dig, kan du begära ett [undantag från serviceplanen](customizing.md).


|Ledningsområde  |Microsoft-metod för hanterade skrivbord  |
|---------|---------|
|Enhetskonfiguration eller principhantering     |  Microsoft Intune       |
|Hantering av program     | Microsoft Intune och företagsportal        |
|Drivrutinsdistribution     |  Drivrutiner som ingår i enheten, Windows Update eller Intune       |
|Enhetssäkerhet     | Se [Enhetssäkerhet](security.md#device-security)      |
|Identitets- och åtkomsthantering     | Se [Hantering av identitet och åtkomst](security.md#identity-and-access-management)        |
|Nätverkssäkerhet     | Se [Nätverkssäkerhet](security.md#network-security)        |
|Informationssäkerhet     |  Se [Informationssäkerhet](security.md#information-security)       |
|Dataräddning     | OneDrive för företag        |
|Kärnproduktivitet     | Microsoft 365-applikationer för företag    |
|Webbläsare     | Microsoft Edge        |




Microsoft Managed Desktop kan övervaka annan programvara som körs på hanterade enheter. Om det påverkar systemets säkerhet, prestanda eller tillförlitlighet negativt kan du behöva begära ett undantag från serviceplanen.
