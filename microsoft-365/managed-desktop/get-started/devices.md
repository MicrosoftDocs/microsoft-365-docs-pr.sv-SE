# <a name="order-microsoft-managed-desktop-devices"></a>Beställ Microsoft Hanterat skrivbord enheter

Vi rekommenderar att du arbetar med en av våra godkända enhetspartner. Du kan arbeta med din Microsoft-kontokontakt för att få mer hjälp med att konfigurera en enhetskontakt.

Microsoft Hanterat skrivbord kräver inte längre att enhetsmodeller finns från listan över [godkända enheter.](../service-description/device-list.md) Den 3 maj 2021 bör alla enheter från en godkänd tillverkare uppfylla våra publicerade maskin- och programvarukrav. Du kan fortsätta att använda enheter i listan på ett enkelt sätt, men du hittar fler enheter som Microsoft Hanterat skrivbord på [webbplatsen Handla Windows 10 Pro företagsenheter.](https://www.microsoft.com/windowsforbusiness/view-all-devices) På webbplatsen kan du visa de rekommenderade enheterna genom att **expandera Funktioner** i **området Filtrera efter** och sedan välja **Microsoft Hanterat skrivbord**. När du planerar att registrera en viss enhetsmodell i tjänsten för första gången bör du testa ett exempel för att säkerställa att den ger den användarupplevelse du förväntar dig. Mer information finns i [Validera nya enheter](../get-started/validate-device.md).
