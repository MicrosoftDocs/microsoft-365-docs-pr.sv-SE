# <a name="enable-user-support-features"></a>Aktivera användarsupportfunktioner

Oavsett om du ger din egen support eller arbetar med en partner för att tillhandahålla det, följer du dessa steg för att ge supportleverantören möjlighet att begära utökad enhetsåtkomst eller eskalera problem till Microsoft Hanterat skrivbord, om det behövs.

1. Om de inte redan har ett konto behöver användarna ett konto i samma Azure Active Directory (AAD) som Microsoft Hanterat skrivbord enheter.
2. Lägg till användarkontona från steg 1 till den **moderna arbetsplatsen Roles-Support partnersäkerhetsgruppen** i AAD.
