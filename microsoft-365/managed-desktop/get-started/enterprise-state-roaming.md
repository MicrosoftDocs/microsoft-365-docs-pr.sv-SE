# <a name="enable-enterprise-state-roaming"></a>Aktivera roaming mellan företagstillstånd

För bästa upplevelse med Microsoft Managed Desktop aktiverar du [Företagstillståndsroaming](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-overview), som gör att användarna på ett säkert sätt kan synkronisera användar- och programinställningsdata till molnet. Detta innebär att de kommer att ha samma upplevelse oavsett vilken Windows-enhet de loggar in på. Om du till exempel ersätter en av deras Microsoft Managed Desktop-enheter med en ny, kommer den att se ut och bete sig exakt samma som den förra.

Om du vill aktivera företagscertifikatroaming följer du stegen i [Aktivera företagstillståndsroaming i Azure Active Directory](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-enable)och återgå sedan till den här dokumentationen.

Om du har några problem med Företagsstatens roaming kontaktar du [administratörssupport](../working-with-managed-desktop/admin-support.md).
