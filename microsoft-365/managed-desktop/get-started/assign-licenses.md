# <a name="assign-licenses"></a>Tilldela licenser

Som en del av förberedelserna för att registrera dig på Microsoft Managed Desktop måste du vara säker på att du har fått de nödvändiga licenserna. Om du inte redan har gjort detta läser du [Mer om licenser](../get-ready/prerequisites.md#more-about-licenses) för mer information om exakt vilka licenser du behöver.


Om din licens är uppradade är det dags nu att tilldela dem till användarna. För att göra detta rekommenderar vi att du drar nytta av den [gruppbaserade licensieringsfunktionen](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-licensing-whatis-azure-portal) i Azure Active Directory.

Om du har några problem med licenstilldelning kontaktar du [administratörssupport](../working-with-managed-desktop/admin-support.md).
