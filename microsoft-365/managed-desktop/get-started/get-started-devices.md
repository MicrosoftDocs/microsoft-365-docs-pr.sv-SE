# <a name="get-your-users-ready-to-use-devices"></a>Gör användarna redo att använda enheter

När en Microsoft Managed Desktop-enhet är i händerna på slutanvändaren är det snabbt och enkelt att komma igång med den. Enheter är förkonfigurerade med den aktuella versionen av Windows och konfigurationer och appar installeras från molnet när användaren slutför installationen. 
 
För att göra detta ännu enklare erbjuder vi en guide som går igenom den första installationen och ger hjälpresurser både för installationen och för användning senare, om det behövs. Du kan anpassa den här guiden så att den innehåller vissa detaljer som är specifika för din organisation. Du distribuerar sedan guiden direkt till användarna tillsammans med deras enhet. Det finns två versioner, en för organisationer som använder Microsofts supportsystem (Appen Få hjälp), en för dem som tillhandahåller alternativt stöd för sina användare.

## <a name="to-prepare-the-guide"></a>Så här förbereder du guiden

1. Ladda ned den version som är lämplig för din organisation:
- [Microsoft Managed Desktop – Kom igång med enheten (Get Help version)](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-help-custom-v3.pdf) – använd den här versionen om slutanvändarna använder Microsofts supportsystem.
- [Microsoft Managed Desktop – Kom igång med enheten](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-custom-v2.pdf) – använd den här versionen om slutanvändarna *inte* använder Microsofts supportsystem.
2. Använd alla appar som kan öppna PDF-filer för att fylla i information som är relevant för din organisation:
    - Namnet på nätverket som användarna bör ansluta till för att fortsätta installationen (steg 3 i guiden)
    - Namnet på organisationens Azure-klientkonto (steg 4 i guiden)
    - Kontaktinformation för organisationens interna IT-stöd (överst på andra sidan)
3. Spara den redigerade PDF-filen och distribuera sedan till slutanvändarna. 

## <a name="ready-to-use-guide"></a>"Guide klar att använda"

Vi tillhandahåller också en mer generisk version av guiden för de organisationer som inte behöver anpassa den. Det finns två versioner, en för organisationer som använder Microsofts supportsystem (Appen Få hjälp), en för dem som tillhandahåller alternativt stöd för sina användare. 

- [Microsoft Managed Desktop – Kom igång med enheten (Get Help-version – klar att användas)](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-help-v3.pdf) – använder den här versionen om slutanvändarna använder Microsofts supportsystem.
- [Microsoft Managed Desktop – Kom igång med enheten (klar att användas)](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-v2.pdf) – använd den här versionen om slutanvändarna *inte* använder Microsofts supportsystem.
