# <a name="set-up-microsoft-managed-desktop-devices"></a>Konfigurera Microsoft Managed Desktop-enheter

Du kan använda både nya och befintliga enheter i Microsoft Managed Desktop.

## <a name="to-obtain-new-devices"></a>Så här hämtar du nya enheter

Vi rekommenderar att du arbetar med en av våra godkända enhetspartner. Du kan arbeta med din Microsoft-kontokontakt för mer hjälp med att konfigurera ett enhetspartnerskap. I huvudsak är dock processen så här:

1. Granska listan över [enheter som för närvarande är godkända](../service-description/device-list.md).
2. Beställ enheterna och arbeta med en godkänd enhetspartner.
3. När de har anlänt gör du något av följande:
    - [Registrera nya enheter själv](register-devices-self.md)
    - Arbeta med en partner för att registrera enheterna
4. [Gör användarna redo](get-started-devices.md) att använda Microsoft Managed Desktop-enheter

## <a name="to-re-use-existing-devices"></a>Så här återanvänder du befintliga enheter

1. Gå igenom listan över [enheter som för närvarande är godkända](../service-description/device-list.md) för att identifiera vilka av dina enheter som är godkända för användning med Microsoft Managed Desktop. Om du återanvänder en befintlig enhet måste du avbilda om den.
2. Gör något av följande:
    - [Registrera befintliga enheter själv](register-reused-devices-self.md)
    - Arbeta med en partner för att registrera enheterna
3. [Gör användarna redo](get-started-devices.md) att använda Microsoft Managed Desktop-enheter

4. [Distribuera Intune-företagsportal](company-portal.md)
5. [Aktivera roaming i företagstillstånd](enterprise-state-roaming.md)
6. Konfigurera enheter (det här avsnittet)
7. [Gör användarna redo att använda enheter](get-started-devices.md)
8. [Distribuera appar](deploy-apps.md)
