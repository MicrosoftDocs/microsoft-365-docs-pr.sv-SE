# <a name="set-up-microsoft-managed-desktop-devices"></a>Konfigurera Microsoft Hanterat skrivbord enheter

Du kan använda både nya och befintliga enheter i Microsoft Hanterat skrivbord.

## <a name="to-obtain-new-devices"></a>Skaffa nya enheter

Vi rekommenderar att du arbetar med en av våra godkända enhetspartner. Du kan arbeta med din Microsoft-kontokontakt för att få mer hjälp med att konfigurera en enhetskontakt. I grund och botten är processen dock så här:

1. Granska listan med rekommenderade enheter genom att filtrera efter Microsoft Hanterat skrivbord på [webbplatsen för handla Windows 10 Pro företagsenheter.](https://www.microsoft.com/windowsforbusiness/view-all-devices)
2. Beställ ett eller några exempel på enheter som du vill använda med en kompatibel bild. Beställningen kan kräva [specifika ordningssteg.](../service-description/device-images.md)
3. [Verifiera](validate-device.md) exempelenheterna.
5. När verifieringen är lyckad beställer du enheterna och samarbetar med en godkänd enhetspartner.
6. När de har anlänt gör du något av följande:
    - [Registrera nya enheter själv](register-devices-self.md)
    - Arbeta med en partner för att registrera enheterna
7. [Förbereda användarna för](get-started-devices.md) användning Microsoft Hanterat skrivbord enheter

## <a name="to-reuse-existing-devices"></a>Återanvända befintliga enheter

1. Kontrollera att dina befintliga enheter uppfyller våra [enhetskrav.](../service-description/device-requirements.md) Du kan också använda den hämtningsbara [beredskapsutvärderingskontrollen för](../get-ready/readiness-assessment-downloadable.md) att verifiera att en viss enhet uppfyller de nödvändiga kraven. 
2. Om du återanvänder en befintlig enhet kan du behöva animera den igen. Mer information om bildalternativ finns [i Enhetsbilder](../service-description/device-images.md).
3. Välj ett eller några exempel på enheter som du vill återanvända och verifiera [dem sedan](validate-device.md).
4. När verifieringen är lyckad gör du något av följande:
    - [Registrera befintliga enheter själv](register-reused-devices-self.md)
    - Arbeta med en partner för att registrera enheterna
5. [Förbereda användarna för](get-started-devices.md) användning Microsoft Hanterat skrivbord enheter
