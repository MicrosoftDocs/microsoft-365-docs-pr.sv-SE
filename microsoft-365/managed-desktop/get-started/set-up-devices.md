# <a name="set-up-microsoft-managed-desktop-devices"></a>Konfigurera Microsoft Hanterat skrivbord enheter

Du kan använda både nya och befintliga enheter på Microsoft Managed Desktop.

## <a name="to-obtain-new-devices"></a>Skaffa nya enheter

Vi rekommenderar att du arbetar med en av våra godkända enhets partners. Du kan arbeta med din Microsoft-kontoinformation för mer hjälp med att konfigurera en enhets koppling. I själva verket är det emellertid så här:

1. Granska listan över [godkända enheter](../service-description/device-list.md).
2. Beställ enheterna och arbeta med en godkänd enhets partner.
3. När de har kommit gör du något av följande:
    - [Registrera nya enheter själv](register-devices-self.md)
    - Arbeta med en partner för att registrera enheterna
4. Förbered [användarna](get-started-devices.md) för att använda Microsoft Managed Station ära enheter

## <a name="to-reuse-existing-devices"></a>Återanvända befintliga enheter

1. Kontrol lera listan över [godkända enheter](../service-description/device-list.md) för att identifiera vilka av dina enheter som är godkända för användning med Microsoft Managed Desktop. Om du återanvändar en befintlig enhet måste du göra det igen.
2. Gör något av följande:
    - [Registrera befintliga enheter själv](register-reused-devices-self.md)
    - Arbeta med en partner för att registrera enheterna
3. Förbered [användarna](get-started-devices.md) för att använda Microsoft Managed Station ära enheter

4. [Distribuera Intune-företagsportalen](company-portal.md)
5. [Aktivera Enterprise State Roaming](enterprise-state-roaming.md)

7. [Gör användarna redo att använda enheter](get-started-devices.md)
8. [Distribuera appar](deploy-apps.md)
