# <a name="ediscovery-solutions-in-microsoft-365"></a>eDiscovery-lösningar i Microsoft 365

E-dataidentifiering är processen med att identifiera och leverera elektronisk information som kan användas som bevis i rättsfall. Du kan använda eDiscovery-verktyg i Microsoft 365 för att söka efter innehåll i Exchange Online-postlådor, Microsoft 365-grupper, Microsoft Teams, SharePoint Online och OneDrive för företag-webbplatser och Skype för företag-konversationer och Yammer-team. Du kan söka i postlådor och webbplatser i samma eDiscovery-sökning med hjälp av verktyget Innehållssökning. Och du kan använda grundläggande eDiscovery-ärenden för att identifiera, vänta och exportera innehåll från postlådor och webbplatser. Om din organisation har en Office 365 E5- eller Microsoft 365 E5-prenumeration (eller relaterade E5-tilläggsprenumerationer) kan du ytterligare hantera dem och analysera innehåll med hjälp av Advanced eDiscovery-lösningen i Microsoft 365.
  
Microsoft 365 innehåller följande eDiscovery-verktyg:
  
- [Innehållssökning](#content-search)

- [Core eDiscovery](#core-ediscovery)

- [Advanced eDiscovery](#advanced-ediscovery)

## <a name="advanced-ediscovery"></a>Advanced eDiscovery

The Advanced eDiscovery solution in Microsoft 365 (also called *Advanced eDiscovery v2.0*) builds on the existing eDiscovery and analytics capabilities in Microsoft 365. Den här eDiscovery-lösningen ger ett end-to-end-arbetsflöde för att bevara, samla in, granska, analysera och exportera innehåll som svarar på organisationens interna och externa undersökningar. Det gör det också möjligt för juridiska grupper att hantera dokumentelag och hela arbetsflödet för aviseringar om juridiskt håll för att kommunicera med dokumente uppslagna personer som är inblandade i ett ärende.
