# <a name="privileged-access-management-in-microsoft-365"></a>Hantering av behörighet i Microsoft 365

Att vissa användare har stående åtkomst till känslig information eller viktiga nätverkskonfigurationsinställningar i Microsoft Exchange Online är en möjlig väg för komprometterade konton eller interna hotaktiviteter. Hantering av privilegierad åtkomst hjälper till att skydda organisationen från överträdelser och hjälper dig att uppfylla metodtips för efterlevnad genom att begränsa stående åtkomst till känsliga data eller åtkomst till viktiga konfigurationsinställningar. I stället för att administratörer har konstant åtkomst implementeras direktåtkomstregler för aktiviteter som behöver förhöjda behörigheter. Genom att aktivera hantering av privilegierad åtkomst för Exchange Online i Microsoft 365 kan organisationen arbeta utan några stående behörigheter och tillhandahålla ett skyddslager mot säkerhetsproblem av administrativ åtkomst.

1. Läs mer om [hantering av privilegierad](privileged-access-management-overview.md) åtkomst i Microsoft 365
2. Skapa en [godkännaresgrupp](privileged-access-management-configuration.md#step-1-create-an-approvers-group)
